Logaritmos: Definición y Aplicación de las Propiedades

Definición · Condiciones de existencia · Propiedades operativas · Cambio de base · Logaritmo decimal y natural

Explicación completa

1. Definición de logaritmo

El logaritmo es la operación inversa de la potenciación. Responde a la pregunta: "¿a qué exponente debo elevar la base b para obtener N?"
logb N = L    ⇔    bL = N
Donde:
  • b = base (b > 0, b ≠ 1)
  • N = argumento o logaritmando (N > 0)
  • L = logaritmo (puede ser cualquier número real)
📐 Ejemplos de conversión forma logarítmica ↔ forma exponencial:
log2 8 = 3  ⇔  23 = 8
log3 81 = 4  ⇔  34 = 81
log5 (1/25) = −2  ⇔  5−2 = 1/25
log4 1 = 0  ⇔  40 = 1
⚠️ Condiciones de existencia del logaritmo:
• La base: b > 0 y b ≠ 1
• El argumento: N > 0
Nunca existe logb(0) ni logb(número negativo), y nunca puede usarse b = 1 como base (pues 1 elevado a cualquier potencia siempre da 1, lo que haría al logaritmo indefinido).

2. Gráfica de la función logarítmica

La función logarítmica f(x) = logb x siempre pasa por el punto (1, 0) y tiene como asíntota vertical el eje Y (x = 0).

3. Propiedades fundamentales de los logaritmos

Estas propiedades son la herramienta central para simplificar y resolver expresiones logarítmicas.

3.1 Propiedades básicas (directas de la definición)

logb b = 1      logb 1 = 0      blogb N = N      logb bn = n
📐 log5 5 = 1  |  log7 1 = 0  |  3log3 9 = 9  |  log2 210 = 10

3.2 Propiedad del producto

logb (M · N) = logb M + logb N
📐 log2(4 · 8) = log2 4 + log2 8 = 2 + 3 = 5    (verificación: log2 32 = 5 ✓)

3.3 Propiedad del cociente

logb (M/N) = logb M − logb N
📐 log3(81/9) = log3 81 − log3 9 = 4 − 2 = 2    (verificación: log3 9 = 2 ✓)

3.4 Propiedad de la potencia

logb Mn = n · logb M
📐 log2 84 = 4 · log2 8 = 4 · 3 = 12   |   log5 √25 = log5 251/2 = (1/2) · 2 = 1

3.5 Propiedad de la raíz (caso especial de potencia)

logb n√M = (1/n) · logb M
⚠️ Errores frecuentes en admisión:
• log(A + B) ≠ log A + log B    (¡no existe esta propiedad!)
• log(A · B) ≠ log A · log B    (el producto adentro se convierte en suma afuera, no en producto)
• [log A]n ≠ n · log A    (la potencia debe estar en el argumento, no fuera del log)

4. Cambio de base

Cuando necesitamos calcular o comparar logaritmos de distintas bases, usamos la fórmula de cambio de base:
logb N = logc N / logc b    (c cualquier base válida)
La forma más usada en exámenes (pasando a base 10 o base natural):
logb N = log N / log b = ln N / ln b
📐 Calcular log4 32:
log4 32 = log2 32 / log2 4 = 5 / 2 = 2,5
Verificación: 42,5 = 45/2 = (22)5/2 = 25 = 32 ✓
Consecuencia directa muy útil:
logb a · loga b = 1      y      logb a = 1 / loga b

5. Logaritmo decimal y logaritmo natural

5.1 Logaritmo decimal (base 10)

Se escribe simplemente log N (sin escribir la base). Es el logaritmo más usado en cálculos científicos y financieros.
log N = log10 N
📐 log 100 = 2  |  log 1000 = 3  |  log(0,01) = −2  |  log 1 = 0

5.2 Logaritmo natural (base e)

Se escribe ln N. La base es el número de Euler: e ≈ 2,71828... Es el logaritmo fundamental en cálculo y ciencias naturales.
ln N = loge N      ln e = 1      ln 1 = 0      eln N = N

Formulario

logb N = L ⇔ bL = N
Condiciones: b > 0, b ≠ 1  |  N > 0

PROPIEDADES OPERATIVAS

logb b = 1    logb 1 = 0    blogbN = N
Producto: logb(M·N) = logbM + logbN
Cociente: logb(M/N) = logbM − logbN
Potencia: logbMn = n · logbM

Problemas tipo admisión

NIVEL BÁSICO
1. ¿Cuánto vale log3 27?
Ver resolución
Respuesta: a)
Pregunta: ¿a qué exponente elevo 3 para obtener 27?
3? = 27 = 33 → log3 27 = 3.
2. ¿Cuánto vale log5 1?
Ver resolución
Respuesta: a)
Por propiedad: logb 1 = 0 para cualquier base válida. 50 = 1. Entonces log5 1 = 0.
3. Convierte a forma exponencial: log2 64 = 6.
Ver resolución
Respuesta: a)
La forma es: logb N = L ⇔ bL = N.
log2 64 = 6 ⇔ 26 = 64. Verificación: 26 = 64 ✓
4. ¿Cuánto vale log10 10 000?
Ver resolución
Respuesta: a)
10 000 = 104. Por propiedad: log10 104 = 4.
5. ¿Cuál es el valor de 3log3 15?
Ver resolución
Respuesta: a)
Por propiedad de exponenciación inversa: blogbN = N.
3log315 = 15.
NIVEL INTERMEDIO
6. Simplifica: log2 4 + log2 8.
Ver resolución
Respuesta: a)
log2 4 = 2  |  log2 8 = 3
Suma = 2 + 3 = 5.
Alternativa: log2(4·8) = log2 32 = 5 ✓ (propiedad del producto)
7. Simplifica: log3 243 − log3 9.
Ver resolución
Respuesta: a)
log3 243 = 5 (35=243)  |  log3 9 = 2 (32=9)
5 − 2 = 3.
Alternativa: log3(243/9) = log3 27 = 3 ✓
8. Calcula: log2 45.
Ver resolución
Respuesta: a)
Por propiedad de potencia: log2 45 = 5 · log2 4 = 5 · 2 = 10.
9. Si log 2 = 0,301, ¿cuánto vale log 8?
Ver resolución
Respuesta: a)
log 8 = log 23 = 3 · log 2 = 3 · 0,301 = 0,903.
10. Calcula log4 8 usando cambio de base a base 2.
Ver resolución
Respuesta: a)
log4 8 = log2 8 / log2 4 = 3 / 2 = 3/2.
Verificación: 43/2 = (22)3/2 = 23 = 8 ✓
NIVEL AVANZADO
11. Si log 2 = a y log 3 = b, expresa log 72 en términos de a y b.
Ver resolución
Respuesta: a)
72 = 8 · 9 = 23 · 32
log 72 = log 23 + log 32 = 3 log 2 + 2 log 3 = 3a + 2b.
12. Simplifica: logb b3 · blogb 4.
Ver resolución
Respuesta: a)
Primero: logb b3 = 3.
Luego: blogb4 = 4.
El producto es 3 · 4 = 12.
13. Si log 2 = 0,301 y log 3 = 0,477, ¿cuánto vale log 1,5?
Ver resolución
Respuesta: a)
1,5 = 3/2 → log(3/2) = log 3 − log 2 = 0,477 − 0,301 = 0,176.
14. Calcula: log9 27.
Ver resolución
Respuesta: a)
Cambio de base a base 3: log9 27 = log3 27 / log3 9 = 3/2 = 3/2.
Verificación: 93/2 = (32)3/2 = 33 = 27 ✓
15. Si loga 2 = 0,4, ¿cuánto vale loga √2?
Ver resolución
Respuesta: a)
loga √2 = loga 21/2 = (1/2) · loga 2 = (1/2) · 0,4 = 0,2.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Si log 2 = a, log 3 = b, expresa log √(12/5) sabiendo que log 5 = 1 − a.
Ver resolución
Respuesta: b)
log √(12/5) = (1/2) log(12/5) = (1/2)[log 12 − log 5]
log 12 = log(4·3) = log 22 + log 3 = 2a + b
log 5 = 1 − a
= (1/2)[(2a + b) − (1 − a)] = (1/2)(2a + b − 1 + a) = (1/2)(3a + b − 1)
Resultado = (3a + b − 1)/2 (la opción b está correctamente simplificada con log5=1-a ya aplicado).
17. Calcula: log2 3 · log3 4 · log4 8.
Ver resolución
Respuesta: a)
Usamos cambio de base a logaritmo natural (o cualquier base común):
= (ln3/ln2) · (ln4/ln3) · (ln8/ln4)
Los términos intermedios se cancelan (telescopio):
= ln8 / ln2 = log2 8 = 3.
18. Si loga b = 3 y logb c = 2, ¿cuánto vale loga c?
Ver resolución
Respuesta: a)
Usando cambio de base a base a:
logb c = loga c / loga b → 2 = loga c / 3
loga c = 2 · 3 = 6.
19. Simplifica completamente: logb(b2 · √b).
Ver resolución
Respuesta: a)
logb(b2 · √b) = logb b2 + logb b1/2 = 2 + 1/2 = 5/2.
20. Si log 2 = 0,301, ¿cuántos dígitos tiene 250?
Ver resolución
Respuesta: a)
El número de dígitos de un entero positivo N es ⌊log N⌋ + 1.
log(250) = 50 · log 2 = 50 · 0,301 = 15,05
Número de dígitos = ⌊15,05⌋ + 1 = 15 + 1 = 16 dígitos.
