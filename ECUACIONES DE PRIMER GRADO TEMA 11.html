ÁLGEBRA · TEMA 11

Ecuaciones de Primer Grado

Concepto · Despeje · Tipos de solución · Ecuaciones fraccionarias · Planteo de problemas

Explicación completa

1. ¿Qué es una ecuación de primer grado?

Una ecuación es una igualdad que solo es cierta para ciertos valores de la incógnita. Resolverla significa hallar ese valor (la solución o raíz). Es de primer grado (o lineal) cuando la incógnita aparece elevada solo a la potencia 1. Su forma general es:
ax + b = 0   (con a ≠ 0)
⚠️ Ecuación vs. identidad: una ecuación se cumple solo para algunos valores (x + 2 = 5 solo con x = 3). Una identidad se cumple para todos (como los productos notables del Tema 6). ¡No las confundas!

2. El despeje: la regla de oro

La idea base es mantener el equilibrio: lo que hago a un lado de la igualdad, lo hago al otro. En la práctica, "lo que está sumando pasa restando, lo que está multiplicando pasa dividiendo".

Pasos para resolver:

  1. Si hay denominadores, multiplica todo por el MCM para eliminarlos.
  2. Quita paréntesis (propiedad distributiva).
  3. Agrupa las x a un lado y los números al otro.
  4. Reduce términos semejantes.
  5. Despeja x dividiendo entre su coeficiente.
📐 Ejemplo: 3x − 5 = x + 7
• Agrupo: 3x − x = 7 + 5
• Reduzco: 2x = 12
• Despejo: x = 12/2 = 6

3. Solución de ax + b = 0

Despejando la forma general se obtiene la fórmula directa:
x = −b / a
📐 Ejemplo: en 5x − 20 = 0, tenemos a = 5, b = −20 → x = −(−20)/5 = 20/5 = 4.

4. Tipos de solución (¡muy preguntado!)

No toda ecuación lineal tiene una única solución. Al reducir, pueden pasar tres cosas:
Al reducir queda...Tipo de ecuaciónSolución
ax = b   (con a ≠ 0)Compatible determinadaUna sola: x = b/a
0 = 0   (algo siempre cierto)Compatible indeterminadaInfinitas soluciones
0 = k   (con k ≠ 0)Incompatible (absurda)No tiene solución
⚠️ Truco de examen: en una ecuación tipo (m)x + ... = ..., si te piden que tenga infinitas soluciones, busca que ambos lados queden idénticos; si te piden que no tenga solución, busca que el coeficiente de x se anule pero los números no coincidan.

5. Ecuaciones fraccionarias

Cuando la incógnita aparece en algún denominador. Se resuelven multiplicando por el MCM de los denominadores, pero cuidado: hay que verificar que la solución no anule ningún denominador original.
📐 Ejemplo: x/2 + x/3 = 5. MCM(2,3) = 6. Multiplico todo por 6: 3x + 2x = 30 → 5x = 30 → x = 6.
⚠️ Soluciones extrañas: si al resolver una ecuación fraccionaria obtienes un valor que hace cero un denominador, ese valor se descarta (no es solución válida).

Formulario

FORMA GENERAL Y SOLUCIÓN

ax + b = 0   (a ≠ 0)
x = −b / a

TIPOS DE SOLUCIÓN

ax = b (a ≠ 0) → una solución (determinada)
0 = 0 → infinitas soluciones (indeterminada)
0 = k (k ≠ 0) → sin solución (incompatible)

ESTRATEGIA DE RESOLUCIÓN

1) Quitar denominadores (× MCM)
2) Quitar paréntesis (distributiva)
3) Agrupar x a un lado, números al otro
4) Reducir y despejar

Problemas tipo admisión

NIVEL BÁSICO
1. Resuelve: x + 7 = 12
Ver resolución
Respuesta: a)
El 7 está sumando, pasa restando: x = 12 − 7 = 5.
2. Resuelve: 3x = 21
Ver resolución
Respuesta: c)
El 3 está multiplicando, pasa dividiendo: x = 21/3 = 7.
3. Resuelve: 2x − 8 = 0
Ver resolución
Respuesta: a)
2x = 8 → x = 8/2 = 4.
4. Resuelve: 5x + 3 = 2x + 18
Ver resolución
Respuesta: b)
Agrupo: 5x − 2x = 18 − 3 → 3x = 15 → x = 5.
5. En la ecuación ax + b = 0, ¿cuál es el valor de x?
Ver resolución
Respuesta: b)
ax + b = 0 → ax = −b → x = −b/a.
NIVEL INTERMEDIO
6. Resuelve: 2(x − 3) = x + 4
Ver resolución
Respuesta: a)
Quito paréntesis: 2x − 6 = x + 4. Agrupo: 2x − x = 4 + 6 → x = 10.
7. Resuelve: x/2 + x/3 = 5
Ver resolución
Respuesta: b)
Multiplico todo por MCM(2,3)=6: 3x + 2x = 30 → 5x = 30 → x = 6.
8. "El doble de un número, disminuido en 3, es igual a 15." Halla el número.
Ver resolución
Respuesta: b)
Traduzco: 2x − 3 = 15 → 2x = 18 → x = 9.
9. Resuelve: 4x − 7 = 6x + 5
Ver resolución
Respuesta: b)
Agrupo: 4x − 6x = 5 + 7 → −2x = 12 → x = 12/(−2) = −6.
10. "La suma de tres números consecutivos es 48." Halla el menor.
Ver resolución
Respuesta: b)
Sean x, x+1, x+2. Suma: 3x + 3 = 48 → 3x = 45 → x = 15 (el menor). Los números son 15, 16 y 17.
NIVEL AVANZADO
11. Resuelve: (x + 1)/3 − (x − 2)/4 = 1
Ver resolución
Respuesta: a)
Multiplico todo por MCM(3,4)=12: 4(x + 1) − 3(x − 2) = 12.
4x + 4 − 3x + 6 = 12 → x + 10 = 12 → x = 2.
12. ¿Qué valor debe tener "m" para que la ecuación (m − 3)x = 7 NO tenga solución?
Ver resolución
Respuesta: a)
No tiene solución cuando el coeficiente de x se anula pero el otro lado no es 0. (m − 3) = 0 → m = 3. Quedaría 0 = 7 (absurdo). Por tanto m = 3.
13. "La edad de Ana es el triple de la de Beto. Si dentro de 10 años Ana tendrá el doble que Beto, ¿qué edad tiene Beto hoy?"
Ver resolución
Respuesta: a)
Beto hoy: x ; Ana hoy: 3x. Dentro de 10 años: 3x + 10 = 2(x + 10).
3x + 10 = 2x + 20 → x = 10. (Beto 10, Ana 30; en 10 años: 40 y 20 ✓.)
14. ¿Para qué valor de "k" la ecuación 2x + k = 2x + 5 tiene infinitas soluciones?
Ver resolución
Respuesta: b)
Para infinitas soluciones, ambos lados deben ser idénticos. Como 2x = 2x, basta que k = 5. Quedaría 0 = 0, cierto para todo x. Por tanto k = 5.
15. Resuelve: 6/(x − 1) = 2   (x ≠ 1)
Ver resolución
Respuesta: b)
Multiplico por (x − 1): 6 = 2(x − 1) → 6 = 2x − 2 → 2x = 8 → x = 4 (válido, no anula el denominador).
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Resuelve: x/(x − 2) = 3/(x − 2) + 1   (x ≠ 2)
Ver resolución
Respuesta: a)
Multiplico por (x − 2): x = 3 + (x − 2) → x = x + 1 → 0 = 1 (absurdo).
Aunque despejaras, la única candidata sería x = 2, que anula el denominador y se descarta. Por eso no tiene solución.
17. "Un padre tiene 40 años y su hijo 10. ¿Dentro de cuántos años la edad del padre será el triple de la del hijo?"
Ver resolución
Respuesta: a)
Dentro de x años: 40 + x = 3(10 + x) → 40 + x = 30 + 3x → 10 = 2x → x = 5. (En 5 años: padre 45, hijo 15; 45 = 3·15 ✓.)
18. Si x = 2 es solución de la ecuación 3x − a = x + 6, halla el valor de "a".
Ver resolución
Respuesta: a)
Si x = 2 es solución, debe cumplirse la igualdad. Reemplazo: 3(2) − a = 2 + 6 → 6 − a = 8 → −a = 2 → a = −4.
