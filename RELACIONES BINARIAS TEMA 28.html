ÁLGEBRA · TEMA 28

Relaciones Binarias I: Definición y Tipo de Relaciones

Par ordenado · Producto cartesiano · Definición de relación binaria · Dominio y rango · Propiedades: reflexiva, simétrica, antisimétrica, transitiva · Relación de equivalencia

Explicación completa

1. Par ordenado y producto cartesiano

Un par ordenado (a, b) es una pareja de elementos en la que el orden importa: (a, b) ≠ (b, a) en general. Dos pares ordenados son iguales si y solo si sus primeras componentes son iguales y sus segundas componentes son iguales:
(a, b) = (c, d)  ⇔  a = c   y   b = d
El producto cartesiano de dos conjuntos A y B, denotado A × B, es el conjunto de todos los pares ordenados posibles con el primer elemento en A y el segundo en B:
A × B = { (a, b) / a ∈ A   y   b ∈ B }
📐 Si A = {1, 2} y B = {a, b, c}:
A × B = {(1,a), (1,b), (1,c), (2,a), (2,b), (2,c)}
|A × B| = |A| · |B| = 2 · 3 = 6 pares ordenados
B × A = {(a,1), (a,2), (b,1), (b,2), (c,1), (c,2)} — ¡distinto de A × B!
⚠️ Cardinalidad del producto cartesiano:
|A × B| = |A| · |B|
En general A × B ≠ B × A (a menos que A = B)

2. Definición de relación binaria

Una relación binaria R de A en B (o sobre A si A = B) es cualquier subconjunto del producto cartesiano A × B:
R ⊆ A × B
Si (a, b) ∈ R se dice que "a está relacionado con b" y se escribe a R b. El número de relaciones posibles de A en B es 2|A×B| = 2|A|·|B|.
📐 A = {1, 2, 3}, B = {2, 4, 6}. Define R = {(a, b) / b = 2a}:
(1,2): 2=2(1) ✓   (2,4): 4=2(2) ✓   (3,6): 6=2(3) ✓   (1,4): 4≠2 ✗ ...
R = {(1,2), (2,4), (3,6)}

3. Dominio, rango y campo de una relación

📐 R = {(1,3), (2,3), (4,5), (6,7)}

4. Relación inversa

La relación inversa R−1 se obtiene intercambiando los elementos de cada par:
R−1 = { (b, a) / (a, b) ∈ R }
📐 Si R = {(1,4), (2,5), (3,6)}, entonces R−1 = {(4,1), (5,2), (6,3)}
Dom(R−1) = Ran(R) = {4,5,6}    Ran(R−1) = Dom(R) = {1,2,3}

5. Propiedades de las relaciones en un conjunto A

Cuando la relación es de A en sí mismo (R ⊆ A × A), pueden analizarse cuatro propiedades fundamentales:

5.1 Propiedad Reflexiva

R es reflexiva ⇔ a R a para todo a ∈ A    (todo elemento se relaciona consigo mismo)
📐 A = {1,2,3}. R reflexiva debe contener (1,1), (2,2) y (3,3).
Ejemplo reflexivo: R = {(1,1), (2,2), (3,3), (1,2)}
NO reflexivo: R = {(1,1), (2,2)} — falta (3,3)

5.2 Propiedad Simétrica

R es simétrica ⇔ si a R b entonces b R a    (las relaciones van "en ambas direcciones")
📐 R = {(1,2), (2,1), (3,3)} es simétrica: (1,2) implica (2,1) ✓
R = {(1,2), (3,3)} NO es simétrica: (1,2) existe pero (2,1) no

5.3 Propiedad Antisimétrica

R es antisimétrica ⇔ si a R b y b R a entonces a = b    (solo se permiten pares simétricos si son iguales)
📐 R = {(1,1), (1,2), (2,3)} es antisimétrica: no hay (b,a) con a≠b que sea par de otro (a,b)
R = {(1,2), (2,1)} NO es antisimétrica: 1R2 y 2R1 pero 1≠2

5.4 Propiedad Transitiva

R es transitiva ⇔ si a R b y b R c entonces a R c    (se puede "encadenar")
📐 R = {(1,2), (2,3), (1,3)} es transitiva: (1,2) y (2,3) → debe estar (1,3) ✓
R = {(1,2), (2,3)} NO es transitiva: (1,2) y (2,3) existen pero falta (1,3)

6. Relaciones especiales: equivalencia y orden

Relación de equivalencia

Una relación es de equivalencia si cumple las tres propiedades: reflexiva, simétrica y transitiva (RST). Las clases de equivalencia particionan el conjunto A en grupos de elementos "equivalentes".
R es de equivalencia ⇔ R es Reflexiva + Simétrica + Transitiva
📐 A = Z (enteros). R definida por: a R b ↔ a − b es par.
Reflexiva: a−a=0 par ✓   Simétrica: si a−b par entonces b−a par ✓   Transitiva: si a−b y b−c pares, a−c = (a−b)+(b−c) par ✓
Es de equivalencia. Clases: pares {... ,−2, 0, 2, 4,...} e impares {...,−1, 1, 3, 5,...}

Relación de orden (parcial)

Una relación es de orden si es reflexiva, antisimétrica y transitiva (RAT).
R es de orden ⇔ R es Reflexiva + Antisimétrica + Transitiva
📐 La relación ≤ sobre R: 2≤2 ✓ (reflexiva); si a≤b y b≤a entonces a=b ✓ (antisimétrica); si a≤b y b≤c entonces a≤c ✓ (transitiva) → es de orden.

Formulario

PRODUCTO CARTESIANO Y CARDINALIDAD

A × B = {(a,b) / a∈A, b∈B}
|A × B| = |A| · |B|
N° de relaciones posibles = 2|A|·|B|

DOMINIO, RANGO E INVERSA

Dom(R) = primeras componentes
Ran(R) = segundas componentes
R−1 = {(b,a) / (a,b)∈R}  → Dom(R−1) = Ran(R)

PROPIEDADES — DEFINICIONES RÁPIDAS

Reflexiva: a R a ∀a∈A
Simétrica: aRb → bRa
Antisimétrica: aRb y bRa → a=b
Transitiva: aRb y bRc → aRc
Equivalencia = R + S + T  |  Orden = R + AS + T

Problemas tipo admisión

NIVEL BÁSICO
1. Si A = {1, 2} y B = {3, 4, 5}, ¿cuántos pares ordenados tiene A × B?
Ver resolución
Respuesta: a)
|A × B| = |A| · |B| = 2 · 3 = 6.
A × B = {(1,3),(1,4),(1,5),(2,3),(2,4),(2,5)}.
2. ¿Cuál es el dominio de R = {(2,5), (3,7), (4,5), (6,9)}?
Ver resolución
Respuesta: a)
El dominio es el conjunto de las primeras componentes de cada par. Dom(R) = {2, 3, 4, 6}.
3. ¿Cuál es la relación inversa de R = {(1,4), (2,5), (3,6)}?
Ver resolución
Respuesta: a)
R−1 se obtiene intercambiando cada par: (1,4)→(4,1); (2,5)→(5,2); (3,6)→(6,3).
R−1 = {(4,1), (5,2), (6,3)}.
4. La relación R = {(1,1), (2,2), (3,3)} sobre A = {1,2,3}, ¿es reflexiva?
Ver resolución
Respuesta: a)
Contiene (1,1), (2,2) y (3,3): todos los pares (a,a) con a∈A. Es reflexiva. (Esta es la relación de igualdad, la mínima relación reflexiva.)
5. R = {(1,2), (2,1), (3,3)} sobre A = {1,2,3}. ¿Es simétrica?
Ver resolución
Respuesta: a)
Verificamos: (1,2)∈R → ¿(2,1)∈R? Sí ✓. (2,1)∈R → ¿(1,2)∈R? Sí ✓. (3,3)∈R → ¿(3,3)∈R? Sí ✓.
La relación es simétrica.
NIVEL INTERMEDIO
6. Si A = {a, b, c} y B = {1, 2}, ¿cuántas relaciones binarias distintas existen de A en B?
Ver resolución
Respuesta: a)
|A × B| = 3 × 2 = 6. Cada relación es un subconjunto de A×B, y hay 26 = 64 subconjuntos posibles.
7. Sea R sobre A = {1,2,3,4} definida por: a R b ↔ a divide a b. ¿Es R transitiva?
Ver resolución
Respuesta: a)
Si a|b existe k tal que b=ak; si b|c existe m tal que c=bm. Entonces c=bm=(ak)m=a(km), luego a|c.
La divisibilidad sí es transitiva.
8. R = {(1,2), (2,3), (1,3), (1,1), (2,2), (3,3)} sobre A={1,2,3}. ¿Qué propiedades cumple?
Ver resolución
Respuesta: a)
Reflexiva: (1,1),(2,2),(3,3) ✓
Simétrica: (1,2)∈R pero (2,1)∉R → NO simétrica
Antisimétrica: no hay par (a,b) y (b,a) con a≠b ✓
Transitiva: (1,2),(2,3)→(1,3) ✓
Cumple R, AS, T → es relación de orden.
9. Sea R sobre Z: a R b ↔ a ≡ b (mod 3) (misma resta es múltiplo de 3). ¿Qué tipo de relación es?
Ver resolución
Respuesta: a)
Reflexiva: a−a=0=3·0 ✓
Simétrica: si 3|(a−b) entonces 3|(b−a) ✓
Transitiva: si 3|(a−b) y 3|(b−c), entonces 3|((a−b)+(b−c))=a−c ✓
Cumple R, S, T → relación de equivalencia. Las clases son los residuos mod 3: {0,3,6,...}, {1,4,7,...}, {2,5,8,...}.
10. Si R es una relación sobre A = {1,2,3} y R = {(1,2), (2,3)}, ¿qué par debe agregarse para que R sea transitiva?
Ver resolución
Respuesta: a)
(1,2)∈R y (2,3)∈R → por transitividad debe estar (1,3) en R.
Sin ese par, la propiedad transitiva no se cumple.
NIVEL AVANZADO
11. R sobre A = {1,2,3,4}. R = {(a,b) / a + b es par}. Halla R y determina su tipo.
Ver resolución
Respuesta: a)
R = {(1,1),(1,3),(2,2),(2,4),(3,1),(3,3),(4,2),(4,4)} (pares donde a+b es par)
Reflexiva: 1+1=2✓, 2+2=4✓, 3+3=6✓, 4+4=8✓ → Reflexiva ✓
Simétrica: si a+b par → b+a par ✓
Transitiva: si a+b par y b+c par → a y b misma paridad; b y c misma paridad → a y c misma paridad → a+c par ✓
Es relación de equivalencia. Clases: {1,3} (impares) y {2,4} (pares).
12. Sea A = {1,2,3}. ¿Cuántos pares hay en la relación de igualdad (la más pequeña relación reflexiva)?
Ver resolución
Respuesta: a)
La relación de igualdad es Δ = {(a,a) / a∈A} = {(1,1),(2,2),(3,3)}. Tiene |A| = 3 pares.
Es la mínima relación reflexiva (solo los pares de la diagonal).
13. ¿Puede una relación ser a la vez simétrica y antisimétrica? Si sí, ¿cuándo?
Ver resolución
Respuesta: a)
Sí es posible. Si R solo contiene pares (a,a) con a∈A:
Simétrica: (a,a)∈R → (a,a)∈R ✓ (el simétrico de (a,a) es él mismo)
Antisimétrica: si (a,b) y (b,a) ∈ R, como solo hay (a,a), esto implica a=b ✓
La relación de identidad cumple ambas propiedades a la vez.
14. Sea R sobre N (naturales): a R b ↔ a ≤ b. Determina las propiedades que cumple.
Ver resolución
Respuesta: a)
Reflexiva: a≤a ✓
Simétrica: 2≤5 pero 5≤2 ✗ → NO simétrica
Antisimétrica: si a≤b y b≤a entonces a=b ✓
Transitiva: si a≤b y b≤c entonces a≤c ✓
Cumple R, AS, T → es una relación de orden (parcial/total).
15. R sobre A={1,2,3,4,5,6}: a R b ↔ a y b tienen el mismo resto al dividir entre 3. ¿Cuántas clases de equivalencia tiene?
Ver resolución
Respuesta: a)
Los restos posibles al dividir entre 3 son 0, 1 y 2:
Resto 0: {3, 6}   Resto 1: {1, 4}   Resto 2: {2, 5}
Hay 3 clases de equivalencia. (Partición de {1,2,3,4,5,6} en 3 grupos.)
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. A = {1,2,3,4}. ¿Cuántas relaciones de equivalencia distintas existen sobre A?
Ver resolución
Respuesta: a)
Las relaciones de equivalencia corresponden a las particiones del conjunto. El número de particiones de un conjunto de n elementos es el número de Bell B(n).
B(4) = 15. (Las particiones de {1,2,3,4} son: {1|2|3|4}, {12|3|4}, ..., {1234}.)
17. R sobre R×R: (a,b) R (c,d) ↔ a²+b² = c²+d². ¿Qué tipo de relación es y qué describen sus clases?
Ver resolución
Respuesta: a)
Reflexiva: a²+b²=a²+b² ✓   Simétrica: si a²+b²=c²+d² entonces c²+d²=a²+b² ✓
Transitiva: si a²+b²=c²+d² y c²+d²=e²+f², entonces a²+b²=e²+f² ✓
Es de equivalencia. Cada clase agrupa todos los puntos del plano a la misma distancia del origen: son circunferencias concéntricas centradas en (0,0).
18. Si R y S son relaciones de equivalencia sobre A, ¿R ∩ S también lo es?
Ver resolución
Respuesta: a)
Si R y S son de equivalencia, R∩S también lo es:
Reflexiva: (a,a)∈R y (a,a)∈S → (a,a)∈R∩S ✓
Simétrica: si (a,b)∈R∩S → (b,a)∈R y (b,a)∈S → (b,a)∈R∩S ✓
Transitiva: si (a,b),(b,c)∈R∩S → (a,c)∈R y (a,c)∈S → (a,c)∈R∩S ✓
Sí, R∩S es siempre de equivalencia.
19. R sobre A={1,2,3,4}: R = {(1,2),(2,1),(1,3),(3,1),(2,3),(3,2),(1,1),(2,2),(3,3),(4,4)}. ¿Qué tipo de relación es?
Ver resolución
Respuesta: a)
Reflexiva: (1,1),(2,2),(3,3),(4,4) ✓
Simétrica: (1,2)↔(2,1)✓; (1,3)↔(3,1)✓; (2,3)↔(3,2)✓; (4,4)↔(4,4)✓
Transitiva: verificar casos como (1,2),(2,3)→(1,3)✓ — todos se cumplen ✓
Es de equivalencia. Clases: {1,2,3} (todos relacionados entre sí) y {4} (solo con él mismo).
20. Sea R la relación sobre P(A) (partes de A) definida por: X R Y ↔ |X| = |Y| (misma cardinalidad). Si A = {a,b,c}, ¿cuántas clases de equivalencia tiene R?
Ver resolución
Respuesta: a)
P(A) tiene 2³=8 subconjuntos. Las cardinalidades posibles son 0, 1, 2 y 3:
|X|=0: {∅} — 1 elemento
|X|=1: {a},{b},{c} — 3 elementos
|X|=2: {a,b},{a,c},{b,c} — 3 elementos
|X|=3: {a,b,c} — 1 elemento
Hay 4 clases de equivalencia (una por cada posible cardinalidad: 0, 1, 2 y 3).
