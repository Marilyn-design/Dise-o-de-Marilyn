Funciones II: Funciones Especiales

Función par e impar · Función creciente y decreciente · Función definida por tramos · Función valor absoluto · Función mayor entero

Explicación completa

1. Función par e función impar

Dada una función f con dominio simétrico respecto al origen (si x pertenece al dominio, −x también), se clasifica así:
TipoCondiciónSimetría gráficaEjemplo
Parf(−x) = f(x) para todo xRespecto al eje Yf(x) = x2, f(x) = |x|, f(x) = x4 + 3
Imparf(−x) = −f(x) para todo xRespecto al origen (0,0)f(x) = x3, f(x) = x, f(x) = x5 − x
NingunaNo cumple ningunaSin simetría especialf(x) = x2 + x, f(x) = x + 1
📐 Ejemplo: ¿f(x) = x4 − 3x2 + 1 es par, impar o ninguna?
f(−x) = (−x)4 − 3(−x)2 + 1 = x4 − 3x2 + 1 = f(x) ✓ → es par
📐 Ejemplo: ¿f(x) = x3 − 5x es par, impar o ninguna?
f(−x) = (−x)3 − 5(−x) = −x3 + 5x = −(x3 − 5x) = −f(x) ✓ → es impar
⚠️ Truco rápido: si todos los exponentes de x son pares (y el término independiente puede estar), la función es par. Si todos los exponentes son impares y no hay término independiente, es impar. Si mezcla ambos tipos, no es ni par ni impar.

2. Función creciente y decreciente

Una función describe cómo cambia y cuando x aumenta:
  • Creciente en un intervalo: si x1 < x2 entonces f(x1) < f(x2). La gráfica sube de izquierda a derecha.
  • Decreciente en un intervalo: si x1 < x2 entonces f(x1) > f(x2). La gráfica baja de izquierda a derecha.
  • Constante en un intervalo: f(x1) = f(x2) para cualquier x1, x2 del intervalo.

3. Función definida por tramos

Una función por tramos (o a trozos) usa distintas reglas según el intervalo en que se encuentre x. Para evaluarla, primero se identifica a qué tramo pertenece el valor de x y luego se aplica la fórmula correspondiente.
f(x) = { 2x + 1   si   x < 0
         x2        si   x ≥ 0
📐 Con la función anterior: f(−3) = 2(−3) + 1 = −5 (usa el tramo x < 0)
f(4) = 42 = 16 (usa el tramo x ≥ 0)  |  f(0) = 02 = 0 (usa x ≥ 0 porque incluye el 0)
⚠️ Puntos abiertos y cerrados: en la gráfica de funciones por tramos, un círculo vacío ○ indica que ese punto no está incluido (desigualdad estricta), y un punto sólido ● indica que sí está incluido.

4. Función valor absoluto: f(x) = |x|

El valor absoluto de un número es su distancia al origen; siempre es no negativo. La función valor absoluto es en realidad una función por tramos:
|x| = { x    si   x ≥ 0
        {−x   si   x < 0
Su gráfica tiene forma de V con vértice en el origen. Propiedades clave:
  • Dom = R  |  Ran = [0 ; +∞)
  • Es una función par: |−x| = |x|
  • |ab| = |a| · |b|    y    |a/b| = |a|/|b| (b ≠ 0)
  • |a + b| ≤ |a| + |b|    (desigualdad triangular)
Cuando la función tiene la forma f(x) = |ax + b| + c, el vértice se halla haciendo ax + b = 0:
📐 f(x) = |2x − 4| + 3: vértice cuando 2x − 4 = 0 → x = 2. Vértice = (2, 3). Rango = [3 ; +∞)

5. Función parte entera (mayor entero): f(x) = ⌊x⌋

La función parte entera, también escrita como f(x) = [x] o ⌊x⌋, devuelve el mayor entero menor o igual a x; es decir, redondea hacia abajo siempre.
⌊x⌋ = n   si y solo si   n ≤ x < n + 1   (con n entero)
📐 Ejemplos de evaluación:
⌊3,7⌋ = 3  |  ⌊5⌋ = 5  |  ⌊−1,2⌋ = −2 (OJO: se va al entero más a la izquierda)  |  ⌊−3⌋ = −3
⚠️ Cuidado con los negativos: ⌊−1,2⌋ = −2, no −1. El entero más cercano hacia abajo de −1,2 es −2. Este es el error más frecuente en admisión.
Su gráfica es una escalera: segmentos horizontales con puntos cerrados a la izquierda y abiertos a la derecha.
Propiedades importantes de la parte entera:
  • Si x es entero: ⌊x⌋ = x
  • ⌊x⌋ ≤ x < ⌊x⌋ + 1
  • ⌊x + n⌋ = ⌊x⌋ + n   (n entero)
  • Dom = R  |  Ran = Z (todos los enteros)

Formulario

FUNCIÓN PAR E IMPAR

Par: f(−x) = f(x) → simetría eje Y
Impar: f(−x) = −f(x) → simetría en origen
Truco: exponentes pares = par  |  exponentes impares, sin término ind. = impar

FUNCIÓN VALOR ABSOLUTO

|x| = x si x ≥ 0  |  |x| = −x si x < 0
Vértice de |ax + b| + c: ax + b = 0
Ran = [c ; +∞)  |  Es función par

FUNCIÓN PARTE ENTERA

⌊x⌋ = n  ⇔  n ≤ x < n + 1
Negativos: ⌊−1,3⌋ = −2 (no −1)
Dom = R  |  Ran = Z

FUNCIÓN POR TRAMOS

Identificar el intervalo de x → aplicar la fórmula correcta
Punto cerrado ● = incluido  |  Punto abierto ○ = excluido
Verificar continuidad en los puntos de empalme

Problemas tipo admisión

NIVEL BÁSICO
1. ¿Cuánto vale |−8|?
Ver resolución
Respuesta: a)
El valor absoluto siempre es no negativo. |−8| = 8.
2. ¿Cuánto vale ⌊4,9⌋?
Ver resolución
Respuesta: a)
La parte entera es el mayor entero ≤ x. Como 4 ≤ 4,9 < 5, tenemos ⌊4,9⌋ = 4.
3. ¿La función f(x) = x2 es par, impar o ninguna?
Ver resolución
Respuesta: a)
f(−x) = (−x)2 = x2 = f(x) ✓. Es par (exponente par, sin término impar).
4. Dada f(x) =
   { 3    si x < 0
   { x + 1 si x ≥ 0
¿Cuánto vale f(2)?
Ver resolución
Respuesta: a) (valor 3)
x = 2 ≥ 0, así que se usa f(x) = x + 1. f(2) = 2 + 1 = 3.
5. ¿Cuál es el rango de f(x) = |x|?
Ver resolución
Respuesta: a)
|x| nunca es negativo; puede ser 0 (cuando x = 0) o cualquier positivo. Rango = [0 ; +∞).
NIVEL INTERMEDIO
6. ¿Cuánto vale ⌊−2,3⌋?
Ver resolución
Respuesta: a)
Necesitamos el mayor entero ≤ −2,3. Entre −3 y −2, el mayor que sigue siendo ≤ −2,3 es −3. ⌊−2,3⌋ = −3.
Error común: elegir −2, pero −2 > −2,3, así que no cumple la condición.
7. ¿La función f(x) = x3 + x es par, impar o ninguna?
Ver resolución
Respuesta: a)
f(−x) = (−x)3 + (−x) = −x3 − x = −(x3 + x) = −f(x) ✓
Todos los exponentes son impares (3 y 1) y no hay término independiente. Es impar.
8. ¿Cuál es el vértice de f(x) = |x − 5| + 2?
Ver resolución
Respuesta: a)
El vértice ocurre cuando el argumento del valor absoluto es cero: x − 5 = 0 → x = 5. En ese punto f(5) = |0| + 2 = 2. Vértice = (5, 2). Rango = [2 ; +∞).
9. Si f(x) = ⌊2x⌋, ¿cuánto vale f(1,5)?
Ver resolución
Respuesta: a)
Primero 2x = 2(1,5) = 3. Luego ⌊3⌋ = 3 (3 es entero, la parte entera de un entero es él mismo).
10. ¿La función f(x) = x2 + x es par, impar o ninguna?
Ver resolución
Respuesta: a)
f(−x) = (−x)2 + (−x) = x2 − x.
¿Igual a f(x) = x2 + x? No. ¿Igual a −f(x) = −x2 − x? No. Mezcla de exponente par e impar: es ninguna.
NIVEL AVANZADO
11. Dada f(x) = { x + 3 si x ≤ 1  |  x2 + 1 si x > 1. ¿Cuánto vale f(1) + f(2)?
Ver resolución
Respuesta: a)
f(1): x = 1 ≤ 1 → usa primer tramo: f(1) = 1 + 3 = 4
f(2): x = 2 > 1 → usa segundo tramo: f(2) = 22 + 1 = 5
f(1) + f(2) = 4 + 5 = 9
12. ¿Cuánto vale ⌊3,2⌋ + ⌊−1,7⌋?
Ver resolución
Respuesta: a)
⌊3,2⌋ = 3 (el mayor entero ≤ 3,2)
⌊−1,7⌋ = −2 (el mayor entero ≤ −1,7; ojo: −2 < −1,7 < −1)
Suma: 3 + (−2) = 1
13. ¿Cuál es el rango de f(x) = |3x − 6| − 1?
Ver resolución
Respuesta: a)
El valor absoluto satisface |3x − 6| ≥ 0 siempre. Por tanto |3x − 6| − 1 ≥ −1.
El mínimo se alcanza cuando 3x − 6 = 0, es decir x = 2: f(2) = 0 − 1 = −1.
Rango = [−1 ; +∞). Vértice: (2, −1).
14. Si f es función par y f(3) = 7, ¿cuánto vale f(−3)?
Ver resolución
Respuesta: a)
Por definición de función par: f(−x) = f(x) para todo x. Entonces f(−3) = f(3) = 7.
15. Si f es función impar y f(2) = −5, ¿cuánto vale f(−2)?
Ver resolución
Respuesta: a)
Por definición de función impar: f(−x) = −f(x). Entonces f(−2) = −f(2) = −(−5) = 5.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Halla todos los valores de x que satisfacen |2x − 3| = 7.
Ver resolución
Respuesta: a)
|2x − 3| = 7 equivale a dos casos:
Caso 1: 2x − 3 = 7 → 2x = 10 → x = 5
Caso 2: 2x − 3 = −7 → 2x = −4 → x = −2
Verificación: |2(5)−3| = |7| = 7 ✓  |  |2(−2)−3| = |−7| = 7 ✓
17. Halla los valores enteros de x tales que ⌊x/2⌋ = 3.
Ver resolución
Respuesta: a)
⌊x/2⌋ = 3 significa 3 ≤ x/2 < 4, es decir 6 ≤ x < 8.
Los enteros en ese intervalo son x = 6 y x = 7.
18. ¿Para qué valores de x se cumple |x + 1| ≤ 3?
Ver resolución
Respuesta: a)
|x + 1| ≤ 3 equivale a: −3 ≤ x + 1 ≤ 3
Restando 1 en todos los miembros: −4 ≤ x ≤ 2
Solución: [−4 ; 2]
19. Dada f(x) = { |x| si x < 2  |  ⌊x⌋ − 1 si x ≥ 2. Calcula f(−3) + f(2,7).
Ver resolución
Respuesta: a)
f(−3): x = −3 < 2 → usa |x|: f(−3) = |−3| = 3
f(2,7): x = 2,7 ≥ 2 → usa ⌊x⌋ − 1: ⌊2,7⌋ − 1 = 2 − 1 = 1
f(−3) + f(2,7) = 3 + 1 = 4
20. Si f(x) es función impar, f(0) puede ser:
Ver resolución
Respuesta: a)
Si f es impar: f(−x) = −f(x) para todo x en el dominio (incluido x = 0 si pertenece).
Con x = 0: f(0) = −f(0) → 2f(0) = 0 → f(0) = 0.
Si 0 está en el dominio, la función impar obligatoriamente pasa por el origen.
