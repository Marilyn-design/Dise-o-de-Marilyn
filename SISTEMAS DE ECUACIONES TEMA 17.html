ÁLGEBRA · TEMA 17

Sistemas de Ecuaciones

Métodos de resolución · Clasificación · Sistemas 3x3 · Cramer · Sistemas no lineales

Explicación completa

1. ¿Qué es un sistema de ecuaciones?

Es un conjunto de dos o más ecuaciones que comparten las mismas incógnitas. Resolverlo es hallar los valores que satisfacen todas las ecuaciones a la vez. Geométricamente, la solución de un sistema lineal 2x2 es el punto donde se cruzan las dos rectas.
a1x + b1y = c1  ;  a2x + b2y = c2

2. Métodos de resolución

MétodoIdea central
SustituciónDespejar una incógnita en una ecuación y reemplazarla en la otra.
IgualaciónDespejar la misma incógnita en ambas e igualar los resultados.
Reducción (eliminación)Sumar o restar las ecuaciones para que una incógnita se cancele.
📐 Por reducción: x + y = 5 y x − y = 1. Sumo las dos: 2x = 6 → x = 3. Reemplazo: 3 + y = 5 → y = 2. Solución: (3, 2).

3. Clasificación de un sistema lineal

TipoSolucionesRectas
Compatible determinadoUna únicaSe cortan en un punto
Compatible indeterminadoInfinitasSon la misma recta
IncompatibleNingunaParalelas
⚠️ Atajo con los coeficientes: comparando a1/a2, b1/b2 y c1/c2: si la de las x difiere de la de las y → determinado; si las tres son iguales → infinitas; si las de x e y coinciden pero la de c no → sin solución.

4. Determinantes y regla de Cramer (2x2)

Para el sistema 2x2, se define el determinante del sistema:
D = a1b2 − a2b1
Y las soluciones son x = Dx/D, y = Dy/D, donde Dx y Dy se forman reemplazando la columna respectiva por los términos independientes.
📐 2x + 3y = 8 ; x + y = 3. D = 2·1 − 3·1 = −1. Dx = 8·1 − 3·3 = −1 → x = 1. Dy = 2·3 − 8·1 = −2 → y = 2.

5. Sistemas 3x3

Con tres incógnitas se usa el mismo método de reducción: combinar las ecuaciones de a dos para ir eliminando incógnitas hasta quedar con una sola.

6. Sistemas no lineales

Cuando una ecuación es de grado mayor (por ejemplo, una recta y una cuadrática), se resuelve por sustitución: se despeja de la lineal y se reemplaza en la otra.
📐 x + y = 5 y xy = 6. Como x e y suman 5 y multiplican 6, son las raíces de t2 − 5t + 6 = 0 → t = 2 y t = 3. Solución: (2, 3) o (3, 2).

Problemas tipo admisión

NIVEL BÁSICO
1. Resuelve: x + y = 5 ; x − y = 1
Ver resolución
Respuesta: a)
Sumo ambas: 2x = 6 → x = 3. Luego 3 + y = 5 → y = 2. Solución: x = 3, y = 2.
2. ¿Cuántas soluciones tiene un sistema compatible determinado?
Ver resolución
Respuesta: b)
Compatible determinado significa que tiene una única solución.
3. Resuelve: y = 2x ; x + y = 6
Ver resolución
Respuesta: a)
Sustituyo y = 2x: x + 2x = 6 → 3x = 6 → x = 2, y = 4. Solución: x = 2, y = 4.
4. En el sistema x + y = 10 ; x − y = 4, halla x.
Ver resolución
Respuesta: a)
Sumo ambas: 2x = 14 → x = 7.
5. Un sistema que no tiene solución se llama:
Ver resolución
Respuesta: c)
Sin solución → sistema incompatible (rectas paralelas).
NIVEL INTERMEDIO
6. Resuelve: 2x + y = 7 ; x − y = 2
Ver resolución
Respuesta: a)
Sumo: 3x = 9 → x = 3. Luego 3 − y = 2 → y = 1. Solución: x = 3, y = 1.
7. Resuelve: x + 2y = 8 ; 3x − 2y = 0
Ver resolución
Respuesta: a)
Sumo ambas (se cancela 2y): 4x = 8 → x = 2. Luego 2 + 2y = 8 → y = 3. Solución: x = 2, y = 3.
8. Resuelve por sustitución: y = x + 1 ; 2x + y = 7
Ver resolución
Respuesta: a)
Reemplazo y = x + 1: 2x + (x + 1) = 7 → 3x = 6 → x = 2, y = 3. Solución: x = 2, y = 3.
9. Si 2x + 3y = 12 y x = 3, halla y.
Ver resolución
Respuesta: a)
2(3) + 3y = 12 → 6 + 3y = 12 → 3y = 6 → y = 2.
10. El sistema x + y = 4 ; 2x + 2y = 8 es:
Ver resolución
Respuesta: a)
La segunda ecuación es el doble de la primera (la misma recta): tiene infinitas soluciones.
NIVEL AVANZADO
11. Resuelve: 3x + 2y = 12 ; 5x − 2y = 4
Ver resolución
Respuesta: a)
Sumo (se cancela 2y): 8x = 16 → x = 2. Luego 6 + 2y = 12 → y = 3. Solución: x = 2, y = 3.
12. Resuelve: 4x − 3y = 5 ; 2x + y = 5
Ver resolución
Respuesta: a)
De la 2.ª: y = 5 − 2x. Reemplazo: 4x − 3(5 − 2x) = 5 → 4x − 15 + 6x = 5 → 10x = 20 → x = 2, y = 1. Solución: x = 2, y = 1.
13. ¿Para qué valor de k el sistema x + y = 3 ; 2x + 2y = k tiene infinitas soluciones?
Ver resolución
Respuesta: a)
La 2.ª debe ser el doble de la 1.ª: 2(x + y) = 2(3) = 6. Así k = 6 (misma recta → infinitas soluciones).
14. Resuelve el sistema no lineal: x + y = 5 ; xy = 6
Ver resolución
Respuesta: a)
x e y suman 5 y multiplican 6 → son raíces de t2 − 5t + 6 = 0 = (t − 2)(t − 3). Solución: (2, 3) o (3, 2).
15. Resuelve (enteros positivos): x2 + y2 = 25 ; x + y = 7
Ver resolución
Respuesta: a)
De x + y = 7: y = 7 − x. Reemplazo: x2 + (7 − x)2 = 25 → 2x2 − 14x + 24 = 0 → x2 − 7x + 12 = 0 = (x − 3)(x − 4). Solución: 3 y 4.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Resuelve el sistema 3x3 y halla x + z:   x + y + z = 6 ; x + y − z = 0 ; x − y + z = 2
Ver resolución
Respuesta: a)
Ec1 − Ec2: 2z = 6 → z = 3. Ec1: x + y = 3. Ec3: x − y = −1. Sumo: 2x = 2 → x = 1, y = 2. Entonces x + z = 1 + 3 = 4.
17. Usando determinantes (Cramer), resuelve: 2x + 3y = 8 ; x + y = 3
Ver resolución
Respuesta: a)
D = 2·1 − 3·1 = −1. Dx = 8·1 − 3·3 = −1 → x = 1. Dy = 2·3 − 8·1 = −2 → y = 2. Solución: x = 1, y = 2.
18. ¿Para qué valor de "a" el sistema ax + y = 4 ; 2x + y = 4 tiene infinitas soluciones?
Ver resolución
Respuesta: a)
Para que sean la misma recta, los coeficientes deben coincidir: a = 2 (las dos ecuaciones quedan idénticas: 2x + y = 4). Por tanto a = 2.
19. Resuelve el sistema no lineal: x − y = 1 ; x2 − y2 = 5
Ver resolución
Respuesta: a)
x2 − y2 = (x − y)(x + y) = 1·(x + y) = 5 → x + y = 5. Con x − y = 1: sumo → 2x = 6 → x = 3, y = 2. Solución: x = 3, y = 2.
20. La suma de dos números es 20 y su diferencia es 6. Halla el número mayor.
Ver resolución
Respuesta: a)
x + y = 20 ; x − y = 6. Sumo: 2x = 26 → x = 13 (el mayor); y = 7. El mayor es 13.
