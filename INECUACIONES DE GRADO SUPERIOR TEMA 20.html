ÁLGEBRA · TEMA 20

Inecuaciones de Grado Superior

Puntos críticos · Método de la curva de signos · Multiplicidad de las raíces

Explicación completa

1. ¿Qué son?

Son inecuaciones con polinomios de grado 3 o mayor, del tipo P(x) > 0 (o <, ≥, ≤). La idea es la misma que en las cuadráticas, pero ahora hay más puntos críticos, así que conviene un método ordenado.

2. Método de la curva de signos (puntos críticos)

Es la herramienta clave de todo el bloque de inecuaciones. Pasos:
  1. Llevar todo a un lado, comparado con cero, y factorizar completamente.
  2. Hallar los puntos críticos (las raíces) y marcarlos en la recta numérica, en orden.
  3. Si el coeficiente principal es positivo, en la zona más a la derecha el signo es (+).
  4. Los signos van alternando (+, −, +, −, ...) de derecha a izquierda al cruzar cada punto crítico simple.
  5. Se eligen las zonas que cumplen la desigualdad pedida.
📐 (x − 1)(x − 2)(x − 3) > 0. Críticos: 1, 2, 3. Signos de derecha a izquierda: (+) para x > 3, (−) entre 2 y 3, (+) entre 1 y 2, (−) para x < 1. Tomo las (+): ⟨1 ; 2⟩ ∪ ⟨3 ; +∞⟩.

3. Multiplicidad de las raíces (¡clave!)

Cuando una raíz aparece repetida, el comportamiento del signo cambia:
Multiplicidad del factor¿El signo cambia al cruzar?
Impar (1, 3, 5, ...)Sí, el signo se invierte (como siempre)
Par (2, 4, ...)No, el signo se mantiene (la curva "rebota")
⚠️ Por ejemplo, en (x − 2)2(x − 5), la raíz 2 tiene multiplicidad par: al pasar por x = 2 el signo NO cambia. Solo cambia en x = 5.

4. Coeficiente principal negativo

Si el coeficiente principal del polinomio es negativo, conviene multiplicar toda la inecuación por (−1) invirtiendo el sentido, para volver al caso de signo (+) a la derecha.

5. Extremos abiertos o cerrados

Si la desigualdad es estricta (> o <), los puntos críticos no se incluyen (paréntesis). Si es ≥ o ≤, las raíces del polinomio sí se incluyen (corchetes).

Formulario

MÉTODO DE LA CURVA DE SIGNOS

1) Comparar con 0 y factorizar
2) Marcar puntos críticos en orden
3) Coef (+) → signo (+) a la derecha
4) Alternar signos hacia la izquierda

MULTIPLICIDAD

Impar → el signo cambia
Par → el signo NO cambia (rebota)

EXTREMOS

> , < → abiertos (paréntesis)
≥ , ≤ → raíces incluidas (corchetes)

Problemas tipo admisión

NIVEL BÁSICO
1. ¿Cuáles son los puntos críticos de (x − 1)(x − 2)(x − 3) = 0?
Ver resolución
Respuesta: a)
Cada factor igualado a cero da 1, 2 y 3.
2. ¿Cuántos puntos críticos tiene (x + 1)(x − 2)(x − 4)?
Ver resolución
Respuesta: a)
Tres factores → tres raíces: −1, 2 y 4. Son 3 puntos críticos.
3. Con coeficiente principal positivo y factores simples, el signo del extremo derecho de la recta es:
Ver resolución
Respuesta: a)
Con coeficiente principal positivo, la zona más a la derecha siempre empieza en (+).
4. Resuelve: (x − 1)(x − 2)(x − 3) > 0
Ver resolución
Respuesta: a)
Signos: (+) si x > 3, (−) en ⟨2;3⟩, (+) en ⟨1;2⟩, (−) si x < 1. Tomo las (+): ⟨1 ; 2⟩ ∪ ⟨3 ; +∞⟩.
5. Resuelve: (x − 1)(x − 2)(x − 3) < 0
Ver resolución
Respuesta: a)
Tomo las zonas (−): x < 1 y entre 2 y 3: ⟨−∞ ; 1⟩ ∪ ⟨2 ; 3⟩.
NIVEL INTERMEDIO
6. Resuelve: x(x − 1)(x − 2) > 0
Ver resolución
Respuesta: a)
Críticos 0, 1, 2. Signos: (+) x>2, (−) 1<x<2, (+) 0<x<1, (−) x<0. Tomo (+): ⟨0 ; 1⟩ ∪ ⟨2 ; +∞⟩.
7. Resuelve: (x + 2)(x − 1)(x − 3) < 0
Ver resolución
Respuesta: a)
Críticos −2, 1, 3. Signos: (+) x>3, (−) 1<x<3, (+) −2<x<1, (−) x<−2. Tomo (−): ⟨−∞ ; −2⟩ ∪ ⟨1 ; 3⟩.
8. Resuelve: x3 − x > 0
Ver resolución
Respuesta: a)
x(x − 1)(x + 1) > 0. Críticos −1, 0, 1. Signos: (+) x>1, (−) 0<x<1, (+) −1<x<0, (−) x<−1. Tomo (+): ⟨−1 ; 0⟩ ∪ ⟨1 ; +∞⟩.
9. Resuelve: (x − 2)2(x − 5) > 0
Ver resolución
Respuesta: a)
La raíz 2 es de multiplicidad par (no cambia el signo). (x − 2)2 ≥ 0 siempre, así que el signo lo da (x − 5): positivo solo si x > 5. Solución: ⟨5 ; +∞⟩.
10. Resuelve: x3 ≤ 0
Ver resolución
Respuesta: a)
x3 tiene el mismo signo que x. x3 ≤ 0 cuando x ≤ 0: ⟨−∞ ; 0].
NIVEL AVANZADO
11. Resuelve: (x − 1)(x − 2)(x − 3)(x − 4) > 0
Ver resolución
Respuesta: a)
Críticos 1,2,3,4. Desde la derecha: (+) x>4, (−) 3-4, (+) 2-3, (−) 1-2, (+) x<1. Tomo (+): ⟨−∞ ; 1⟩ ∪ ⟨2 ; 3⟩ ∪ ⟨4 ; +∞⟩.
12. Resuelve: (x + 1)2(x − 3) < 0
Ver resolución
Respuesta: a)
(x + 1)2 ≥ 0 (no cambia signo en −1). El signo lo da (x − 3): negativo si x < 3, pero hay que excluir x = −1 (ahí el producto es 0, no < 0). Solución: ⟨−∞ ; −1⟩ ∪ ⟨−1 ; 3⟩.
13. Resuelve: x3 − 4x ≤ 0
Ver resolución
Respuesta: a)
x(x − 2)(x + 2) ≤ 0. Críticos −2, 0, 2. Signos: (+) x>2, (−) 0-2, (+) −2-0, (−) x<−2. Tomo (−) con extremos incluidos: ⟨−∞ ; −2] ∪ [0 ; 2].
14. Resuelve: x3 − 3x2 + 2x > 0
Ver resolución
Respuesta: a)
x(x − 1)(x − 2) > 0. Críticos 0, 1, 2. Tomo las (+): ⟨0 ; 1⟩ ∪ ⟨2 ; +∞⟩.
15. Resuelve: (x − 1)3(x + 2) > 0
Ver resolución
Respuesta: a)
(x − 1)3 tiene multiplicidad impar → se comporta igual que (x − 1). Entonces es como (x − 1)(x + 2) > 0, con críticos −2 y 1, por fuera: ⟨−∞ ; −2⟩ ∪ ⟨1 ; +∞⟩.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Resuelve: (x − 2)2(x + 1) ≥ 0
Ver resolución
Respuesta: a)
(x − 2)2 ≥ 0 siempre; el signo lo da (x + 1). El producto es ≥ 0 cuando x + 1 ≥ 0, es decir x ≥ −1 (en x = 2 vale 0, ya incluido). Solución: [−1 ; +∞⟩.
17. Resuelve: x4 − 5x2 + 4 < 0
Ver resolución
Respuesta: a)
(x2 − 1)(x2 − 4) = (x − 1)(x + 1)(x − 2)(x + 2). Críticos −2, −1, 1, 2. Tomo las zonas (−): ⟨−2 ; −1⟩ ∪ ⟨1 ; 2⟩.
18. Resuelve: (x − 3)(x − 1)2(x + 2) ≤ 0
Ver resolución
Respuesta: a)
(x − 1)2 ≥ 0 (no cambia signo en 1). El signo lo dan (x − 3)(x + 2): negativo entre −2 y 3. Con ≤ incluyo extremos (y x = 1 da 0): [−2 ; 3].
19. ¿Cuántos enteros satisfacen (x − 1)(x − 2)(x − 3)(x − 4) < 0?
Ver resolución
Respuesta: a)
Las zonas (−) son ⟨1 ; 2⟩ y ⟨3 ; 4⟩ (abiertas). No hay enteros estrictamente dentro de esos intervalos → 0 enteros.
20. Resuelve: x3 + x2 − 2x ≥ 0
Ver resolución
Respuesta: a)
x(x2 + x − 2) = x(x + 2)(x − 1). Críticos −2, 0, 1. Signos: (+) x>1, (−) 0-1, (+) −2-0, (−) x<−2. Tomo (+) con extremos incluidos: [−2 ; 0] ∪ [1 ; +∞⟩.
