Ecuaciones e Inecuaciones Logarítmicas

Explicación completa

1. Ecuaciones exponenciales

Una ecuación exponencial tiene la incógnita en el exponente. La estrategia principal es igualar bases o, cuando no es posible, aplicar logaritmos a ambos miembros.

Método 1: Igualación de bases (misma base)

Si bf(x) = bg(x), como la función exponencial es inyectiva, se concluye que f(x) = g(x).
bf(x) = bg(x)  ⇔  f(x) = g(x)    (con b > 0, b ≠ 1)
📐 Resuelve: 23x−1 = 2x+5
Misma base (2), igualamos exponentes: 3x − 1 = x + 5 → 2x = 6 → x = 3
Verificación: 28 = 28 ✓
📐 Resuelve: 9x = 27
Expresar en base 3: (32)x = 33 → 32x = 33 → 2x = 3 → x = 3/2

Método 2: Aplicar logaritmos (bases distintas)

Cuando no podemos igualar bases directamente, aplicamos log (o ln) a ambos miembros y usamos la propiedad de la potencia.
ax = b  ⇒  x = loga b = log b / log a = ln b / ln a
📐 Resuelve: 3x = 15
Aplicamos log: log(3x) = log 15 → x · log 3 = log 15 → x = log 15 / log 3 = log3 15 ≈ 2,46
⚠️ Truco para reducir bases: Siempre intenta expresar todos los términos con la misma base antes de aplicar logaritmos. Bases frecuentes en admisión: 2, 3, 4, 8, 9, 16, 27, 32, 81, 125, 243.

2. Ecuaciones logarítmicas

Una ecuación logarítmica tiene la incógnita dentro del argumento del logaritmo. Se resuelven convirtiendo a forma exponencial o usando propiedades para unificar logaritmos.

Tipo 1: logb f(x) = k → convertir a exponencial

logb f(x) = k  ⇔  f(x) = bk
📐 Resuelve: log2(x − 3) = 4
x − 3 = 24 = 16 → x = 19
Verificación: log2(19 − 3) = log2 16 = 4 ✓    Además 19 − 3 = 16 > 0 ✓

Tipo 2: logb f(x) = logb g(x) → igualar argumentos

Como el logaritmo es inyectivo (con la misma base), argumentos iguales implican que los logaritmos son iguales:
logb f(x) = logb g(x)  ⇔  f(x) = g(x)   (con f(x) > 0 y g(x) > 0)
📐 Resuelve: log3(2x + 1) = log3(x + 7)
Igualamos argumentos: 2x + 1 = x + 7 → x = 6
Verificación: argumentos positivos: 2(6)+1=13>0 ✓   6+7=13>0 ✓

Tipo 3: Ecuaciones con propiedades (unir logaritmos)

Cuando aparecen varios logaritmos, los unimos usando las propiedades del producto, cociente o potencia, y luego aplicamos uno de los tipos anteriores.
📐 Resuelve: log x + log(x − 3) = 1
Propiedad del producto: log[x(x−3)] = 1
Convirtiendo: x(x−3) = 101 = 10
x2 − 3x − 10 = 0 → (x−5)(x+2) = 0 → x = 5 o x = −2
Verificación: x=5: log5+log2=log10=1 ✓; x=−2: log(−2) no existe ✗
Solución: x = 5
⚠️ ¡VERIFICAR SIEMPRE! Al resolver ecuaciones logarítmicas pueden aparecer soluciones extrañas (extraneous solutions): valores de x que satisfacen la ecuación algebraica pero hacen que algún argumento sea ≤ 0. Esos valores se descartan. Es el error más penalizado en admisión.

3. Inecuaciones exponenciales

Las inecuaciones exponenciales se resuelven igualando bases y luego comparando exponentes, con especial atención al signo de la desigualdad según la base.
Si b > 1:   bf(x) > bg(x)  ⇔  f(x) > g(x)   (se conserva la desigualdad)
Si 0 < b < 1:   bf(x) > bg(x)  ⇔  f(x) < g(x)   (se invierte la desigualdad)
📐 Base > 1: Resuelve 32x−1 > 3x+2
Base 3 > 1 → conserva: 2x − 1 > x + 2 → x > 3. Solución: (3 ; +∞)
📐 Base entre 0 y 1: Resuelve (1/2)x > (1/2)3
Base 1/2 < 1 → invierte: x < 3. Solución: (−∞ ; 3)

4. Inecuaciones logarítmicas

Las inecuaciones logarítmicas también requieren atención al signo de la desigualdad según la base, y es obligatorio verificar el dominio (que el argumento sea positivo).
Si b > 1:   logb f(x) > logb g(x)  ⇔  f(x) > g(x)
Si 0 < b < 1:   logb f(x) > logb g(x)  ⇔  f(x) < g(x)
Procedimiento completo para inecuaciones logarítmicas:
  1. Establecer las condiciones de dominio: todos los argumentos > 0.
  2. Reunir los logaritmos en un solo miembro si es necesario.
  3. Quitar los logaritmos según la base (conservar o invertir la desigualdad).
  4. Resolver la inecuación algebraica resultante.
  5. Intersectar la solución con el dominio.
📐 Resuelve: log2(x − 1) > 3
Dominio: x − 1 > 0 → x > 1
Base 2 > 1 → conserva: x − 1 > 23 = 8 → x > 9
Intersección con dominio (x>1): Solución = (9 ; +∞)
📐 Resuelve: log1/3(2x − 1) ≥ log1/3(x + 2)
Dominio: 2x − 1 > 0 → x > 1/2  y  x + 2 > 0 → x > −2. Dom = (1/2 ; +∞)
Base 1/3 < 1 → invierte: 2x − 1 ≤ x + 2 → x ≤ 3
Intersección con dominio: Solución = (1/2 ; 3]

5. Estrategia unificada de resolución

TipoEstrategia principalRegla crítica
Ec. exponencial (misma base)Igualar exponentesbf=bg ⇒ f=g

Ec. logarítmica (log = k)Convertir a exponencialf(x) = bk
Ec. logarítmica (log = log)Igualar argumentos + verificar dominiof(x) = g(x), f(x)>0 y g(x)>0

Formulario

ECUACIONES EXPONENCIALES

Misma base: bf(x)=bg(x) ⇒ f(x)=g(x)
Base distinta: ax=b ⇒ x = log b / log a
Truco: reducir a la misma base potenciando (22=4, 23=8, 33=27...)

ECUACIONES LOGARÍTMICAS

logb f(x) = k ⇒ f(x) = bk
logb f(x) = logb g(x) ⇒ f(x) = g(x)
¡Siempre verificar! Los argumentos deben ser > 0

INECUACIONES: REGLA DEL SIGNO

b > 1: desigualdad se conserva
0 < b < 1: desigualdad se invierte
Logarítmica: establecer dominio → resolver → intersectar

13. Resuelve la inecuación: log2(x + 3) ≥ log2(2x − 1).
Ver resolución
Respuesta: a)

Base 2>1 → conserva: x+3 ≥ 2x−1 → 4 ≥ x → x ≤ 4
Intersección: (1/2;+∞) ∩ (−∞;4] = (1/2 ; 4].
14. Resuelve: log2 x + log2(x − 2) = 3.
Ver resolución
Respuesta: a)
log2[x(x−2)] = 3 → x(x−2) = 23 = 8

Verificación: x=4: log24+log22=2+1=3 ✓ ; x=−2: argumentos negativos ✗

15. Resuelve: 22x+1 ≤ 4x−1.
Ver resolución
Respuesta: a)
4x−1 = (22)x−1 = 22x−2.
22x+1 ≤ 22x−2. Base 2 > 1 → conserva: 2x+1 ≤ 2x−2 → 1 ≤ −2.
Esto es siempre falso, así que... revisemos: 2x+1 ≤ 2x−2 no tiene solución.
Corrección del enunciado interpretado: 22x+1 ≤ 22(x−1) = 22x−2 → 2x+1 ≤ 2x−2 → sin sol.
Si el enunciado es 2x+1 ≤ 4x+3: (2)x+1 ≤ 22x+6 → x+1 ≤ 2x+6 → −5 ≤ x → x ≥ −5.

Ver resolución
Respuesta: a)

u=1: log x=1 → x=10 ; u=2: log x=2 → x=100
Ambos positivos ✓. Solución: x = 10 y x = 100.

Ver resolución
Respuesta: a)

Intersección: (2;+∞) ∩ [−1;4] = (2 ; 4].

u2 − 4u + 4 = 0 → (u−2)2 = 0 → u = 2.
2x = 2 = 21 → x = 1. Solución única (raíz doble).
