ÁLGEBRA · TEMA 21

Inecuaciones Fraccionarias

Valores prohibidos · Puntos críticos · Método de signos del cociente · El error de multiplicar en cruz

Explicación completa

1. ¿Qué es una inecuación fraccionaria?

Es una desigualdad en la que la incógnita aparece en el denominador, del tipo:
P(x) / Q(x) > 0   (o con <, ≥, ≤)
La clave es analizar el signo del cociente, no el valor numérico.

2. El error más grave: multiplicar en cruz

⚠️ NUNCA multipliques ambos lados por el denominador para "quitarlo". Como el denominador tiene una incógnita, no sabes si es positivo o negativo, así que no sabes si debes invertir o no el sentido. En su lugar, lleva todo a un lado, deja 0 al otro y analiza signos.

3. Valores prohibidos

El denominador nunca puede valer cero. Los valores que lo anulan están prohibidos y se excluyen siempre de la solución (van con paréntesis, aunque la desigualdad sea ≥ o ≤).
📐 En (x − 1)/(x − 2), el valor x = 2 está prohibido porque anula el denominador.

4. Método de los puntos críticos

Funciona igual que en las inecuaciones polinómicas, con un detalle extra:
  1. Llevar todo a un lado comparado con 0 y dejar un solo cociente factorizado.
  2. Hallar los puntos críticos: las raíces del numerador y del denominador.
  3. Marcarlos en la recta y analizar el signo del cociente por zonas (igual que el método de la curva de signos).
  4. Los críticos del denominador SIEMPRE van abiertos (paréntesis). Los del numerador se incluyen solo si la desigualdad es ≥ o ≤.
📐 (x − 1)/(x − 2) > 0. Críticos 1 y 2. El cociente es positivo "por fuera": x < 1 o x > 2 → ⟨−∞ ; 1⟩ ∪ ⟨2 ; +∞⟩.

5. Cuando no está comparado con cero

Si la inecuación es del tipo a/(x − b) < c, primero pasa todo a un lado para que quede comparado con 0 y reúne en una sola fracción. Recién ahí aplica el método de signos.
📐 2/(x − 1) < 1 → 2/(x − 1) − 1 < 0 → (3 − x)/(x − 1) < 0. Críticos 1 y 3 → x < 1 o x > 3.

Formulario

REGLAS DE ORO

NUNCA multiplicar en cruz por el denominador
Denominador ≠ 0 (valores prohibidos)
Comparar siempre con 0 y reunir en un cociente

PUNTOS CRÍTICOS

Críticos = ceros del numerador y del denominador
Analizar signo del cociente por zonas

EXTREMOS

Denominador → SIEMPRE abierto ( )
Numerador → cerrado [ ] solo si es ≥ o ≤

Problemas tipo admisión

NIVEL BÁSICO
1. En la expresión (x − 1)/(x − 2), ¿qué valor de x está prohibido?
Ver resolución
Respuesta: b)
El denominador no puede ser 0: x − 2 = 0 → x = 2 está prohibido.
2. Resuelve: 1/x > 0
Ver resolución
Respuesta: a)
1/x es positivo solo cuando x es positivo: x > 0.
3. Resuelve: 1/x < 0
Ver resolución
Respuesta: a)
1/x es negativo solo cuando x es negativo: x < 0.
4. ¿Cuáles son los puntos críticos de (x − 1)/(x − 3)?
Ver resolución
Respuesta: a)
Críticos = ceros del numerador (x = 1) y del denominador (x = 3): 1 y 3.
5. ¿Qué valor NO puede tomar x en 5/(x + 2)?
Ver resolución
Respuesta: a)
x + 2 = 0 → x = −2 anula el denominador, está prohibido.
NIVEL INTERMEDIO
6. Resuelve: (x − 1)/(x − 2) > 0
Ver resolución
Respuesta: a)
Críticos 1 y 2. El cociente es positivo "por fuera": ⟨−∞ ; 1⟩ ∪ ⟨2 ; +∞⟩.
7. Resuelve: (x − 1)/(x − 2) < 0
Ver resolución
Respuesta: a)
El cociente es negativo "por dentro" de los críticos 1 y 2: ⟨1 ; 2⟩.
8. Resuelve: (x + 2)/(x − 3) ≥ 0
Ver resolución
Respuesta: a)
Críticos −2 (num, se incluye) y 3 (denom, abierto). Positivo por fuera: ⟨−∞ ; −2] ∪ ⟨3 ; +∞⟩.
9. Resuelve: x/(x − 1) > 0
Ver resolución
Respuesta: a)
Críticos 0 y 1. Positivo por fuera: ⟨−∞ ; 0⟩ ∪ ⟨1 ; +∞⟩.
10. Resuelve: (x − 2)/(x + 1) ≤ 0
Ver resolución
Respuesta: a)
Críticos −1 (denom, abierto) y 2 (num, incluido). Negativo por dentro: ⟨−1 ; 2].
NIVEL AVANZADO
11. Resuelve: (x − 1)(x − 3)/(x − 2) > 0
Ver resolución
Respuesta: a)
Críticos 1, 2, 3. Signos de derecha a izquierda: (+) x>3, (−) 2<x<3, (+) 1<x<2, (−) x<1. Tomo (+): ⟨1 ; 2⟩ ∪ ⟨3 ; +∞⟩.
12. Resuelve: 2/(x − 1) < 1
Ver resolución
Respuesta: a)
Paso a un lado: 2/(x−1) − 1 < 0 → (2 − (x−1))/(x−1) < 0 → (3 − x)/(x − 1) < 0. Críticos 1 y 3. El cociente es negativo: ⟨−∞ ; 1⟩ ∪ ⟨3 ; +∞⟩.
13. Resuelve: (x + 1)/(x − 2) ≤ 0
Ver resolución
Respuesta: a)
Críticos −1 (num, incluido por ≤) y 2 (denom, abierto). Negativo por dentro: [−1 ; 2⟩.
14. Resuelve: x/(x2 − 4) > 0
Ver resolución
Respuesta: a)
x/[(x − 2)(x + 2)] > 0. Críticos −2, 0, 2. Signos: (+) x>2, (−) 0<x<2, (+) −2<x<0, (−) x<−2. Tomo (+): ⟨−2 ; 0⟩ ∪ ⟨2 ; +∞⟩.
15. Resuelve: (x − 3)/(x + 2) ≥ 1
Ver resolución
Respuesta: a)
Paso a un lado: (x−3)/(x+2) − 1 ≥ 0 → ((x−3) − (x+2))/(x+2) ≥ 0 → (−5)/(x + 2) ≥ 0. Como el numerador es −5 (negativo), el cociente es positivo solo si el denominador es negativo: x + 2 < 0 → ⟨−∞ ; −2⟩.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Resuelve: (x − 1)2/(x − 3) > 0
Ver resolución
Respuesta: a)
(x − 1)2 ≥ 0 siempre (y es 0 en x = 1, donde el cociente no es > 0). El signo lo da el denominador: positivo solo si x − 3 > 0 → ⟨3 ; +∞⟩.
17. Resuelve: (x − 2)/[(x − 1)(x − 4)] < 0
Ver resolución
Respuesta: a)
Críticos 1, 2, 4. Signos de derecha a izquierda: (+) x>4, (−) 2<x<4, (+) 1<x<2, (−) x<1. Tomo (−): ⟨−∞ ; 1⟩ ∪ ⟨2 ; 4⟩.
18. ¿Cuántos enteros satisfacen (x − 1)/(x − 5) < 0?
Ver resolución
Respuesta: a)
El cociente es negativo entre los críticos: 1 < x < 5. Los enteros son 2, 3 y 4 → 3 enteros.
19. Resuelve: (x2 − 4)/(x − 1) ≥ 0
Ver resolución
Respuesta: a)
(x − 2)(x + 2)/(x − 1) ≥ 0. Críticos −2, 1, 2. Signos: (+) x>2, (−) 1<x<2, (+) −2<x<1, (−) x<−2. Tomo (+) incluyendo −2 y 2 (num), excluyendo 1 (denom): [−2 ; 1⟩ ∪ [2 ; +∞⟩.
20. Resuelve: 3/(x − 2) > 2
Ver resolución
Respuesta: a)
3/(x−2) − 2 > 0 → (3 − 2(x−2))/(x−2) > 0 → (7 − 2x)/(x − 2) > 0. Críticos 2 y 7/2. El cociente es positivo entre ellos: ⟨2 ; 7/2⟩.
