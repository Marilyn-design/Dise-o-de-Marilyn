ÁLGEBRA · TEMA 26

Sistemas de Inecuaciones

Sistema de inecuaciones lineales · Método de intersección · Sistema con inecuación cuadrática · Sistema mixto · Representación en la recta numérica

Explicación completa

1. ¿Qué es un sistema de inecuaciones?

Un sistema de inecuaciones es un conjunto de dos o más inecuaciones que deben cumplirse simultáneamente. La solución del sistema es la intersección de los conjuntos solución de cada inecuación por separado.
Sistema:   { f(x) < a
{ g(x) > b

Solución = Solución(f(x)<a) ∩ Solución(g(x)>b)
⚠️ Diferencia clave:
• Sistema de inecuaciones → conector "Y" → se toma la intersección (∩)
• Inecuación de valor absoluto tipo "mayor" → conector "O" → se toma la unión (∪)
Cuando el enunciado dice "el valor de x cumple ambas condiciones", estás ante un sistema.

2. Método de resolución paso a paso

  1. Resolver cada inecuación del sistema por separado, obteniendo un intervalo solución para cada una.
  2. Representar ambas soluciones en la recta numérica.
  3. Tomar la intersección (la zona donde ambas soluciones se superponen).
  4. Expresar el resultado en notación de intervalo.
📐 Ejemplo 1 — Sistema lineal básico:
{ x − 2 > 1    y    2x + 1 < 9

Inecuación 1: x − 2 > 1 → x > 3 → (3 ; +∞)
Inecuación 2: 2x + 1 < 9 → 2x < 8 → x < 4 → (−∞ ; 4)
Intersección: (3 ; +∞) ∩ (−∞ ; 4) = (3 ; 4)

3. Sistema sin solución y sistema con solución total

Sistema sin solución (conjuntos disjuntos):

Cuando las soluciones individuales no se solapan, la intersección es el conjunto vacío.
📐 { x > 5   y   x < 2
Sol. 1: (5;+∞)   Sol. 2: (−∞;2)
Intersección: (5;+∞) ∩ (−∞;2) = ∅ (sin solución)
No hay número que sea simultáneamente mayor que 5 y menor que 2.

Sistema con todos los reales como solución:

Cuando la intersección cubre toda la recta:
📐 { x > −3   y   x < 10
Intersección: (−3 ; 10) — intervalo finito
Pero { x > −∞   y   x < +∞ → solución = R

4. Sistema equivalente a una inecuación doble

Muchos sistemas lineales se pueden escribir directamente como una inecuación doble o encadenada:
a < f(x) < b   es equivalente a   { f(x) > a   y   { f(x) < b
📐 Resuelve: −3 ≤ 2x + 1 ≤ 7
Restamos 1: −4 ≤ 2x ≤ 6
Dividimos entre 2: −2 ≤ x ≤ 3
Solución: [−2 ; 3]

5. Sistema con inecuación cuadrática

Cuando una de las inecuaciones es cuadrática, se resuelve primero como ecuación para hallar las raíces, luego se analiza el signo, y finalmente se intersecta con la otra inecuación.

Recordatorio: resolución de inecuación cuadrática ax² + bx + c < 0

  1. Hallar las raíces r1 y r2 (con r1 < r2).
  2. Si a > 0: la parábola abre hacia arriba → ax²+bx+c < 0 entre las raíces: (r1 ; r2)
  3. Si a > 0: ax²+bx+c > 0 fuera de las raíces: (−∞;r1) ∪ (r2;+∞)
📐 Resuelve el sistema: { x² − x − 6 ≤ 0   y   x + 1 > 0

Inecuación 1: x²−x−6 ≤ 0 → (x−3)(x+2) ≤ 0 → raíces x=3 y x=−2
Parábola abre hacia arriba → solución entre raíces: [−2 ; 3]

Inecuación 2: x + 1 > 0 → x > −1 → (−1 ; +∞)

Intersección: [−2;3] ∩ (−1;+∞) = (−1 ; 3]

6. Sistema mixto: lineal + cuadrática + otras inecuaciones

En exámenes de admisión aparecen sistemas con tres o más inecuaciones. El procedimiento es el mismo: resolver cada una, luego intersectar todas.
📐 Resuelve: { x > −1    y    x < 5    y    x² − 4 < 0

Inec. 1: x > −1 → (−1;+∞)
Inec. 2: x < 5 → (−∞;5)
Inec. 3: x²<4 → |x|<2 → (−2;2)
Intersección: (−1;+∞) ∩ (−∞;5) ∩ (−2;2) = (−1 ; 2)

Formulario

PRINCIPIO DEL SISTEMA

Solución = intersección de todas las soluciones individuales
Sistema → conector "Y" → operación ∩
Si la intersección es vacía → sin solución (∅)

INECUACIÓN CUADRÁTICA ax²+bx+c

Raíces r1 < r2; a>0:
< 0 → solución: (r1 ; r2)
> 0 → solución: (−∞;r1) ∪ (r2;+∞)

DOBLE INECUACIÓN

a < f(x) < b → operar igual en los 3 miembros
Sumar/restar: mismo valor a los 3
Multiplicar/dividir por negativo: invertir ambas desigualdades

Problemas tipo admisión

NIVEL BÁSICO
1. Resuelve el sistema: { x > 2   y   x < 7.
Ver resolución
Respuesta: a)
Sol. 1: (2;+∞)   Sol. 2: (−∞;7)
Intersección: (2 ; 7) — todos los x simultáneamente mayores que 2 y menores que 7.
2. Resuelve: −1 ≤ x + 3 ≤ 5.
Ver resolución
Respuesta: a)
Restamos 3 en los tres miembros: −1−3 ≤ x ≤ 5−3 → −4 ≤ x ≤ 2.
Solución: [−4 ; 2].
3. Resuelve el sistema: { x > 5   y   x < 3.
Ver resolución
Respuesta: a)
(5;+∞) ∩ (−∞;3) = ∅. No existe ningún número real simultáneamente mayor que 5 y menor que 3.
Solución: ∅ (sin solución).
4. Resuelve: 2 < 2x − 2 < 8.
Ver resolución
Respuesta: a)
Sumamos 2 en los tres miembros: 4 < 2x < 10
Dividimos entre 2: 2 < x < 5
Solución: (2 ; 5).
5. Resuelve: { x ≥ −3   y   x ≤ 1.
Ver resolución
Respuesta: a)
[−3;+∞) ∩ (−∞;1] = [−3 ; 1]. Ambos extremos se incluyen porque las desigualdades son ≤ y ≥.
NIVEL INTERMEDIO
6. Resuelve: { 2x + 1 > 5   y   3x − 2 < 10.
Ver resolución
Respuesta: a)
Inec. 1: 2x > 4 → x > 2 → (2;+∞)
Inec. 2: 3x < 12 → x < 4 → (−∞;4)
Intersección: (2;+∞) ∩ (−∞;4) = (2 ; 4).
7. Resuelve: 1 ≤ 3 − 2x ≤ 9.
Ver resolución
Respuesta: a)
Restamos 3: 1−3 ≤ −2x ≤ 9−3 → −2 ≤ −2x ≤ 6
Dividimos entre −2 (invertimos): 1 ≥ x ≥ −3 → −3 ≤ x ≤ 1
Solución: [−3 ; 1].
8. Resuelve: { x² − 1 < 0   y   x > 0.
Ver resolución
Respuesta: a)
Inec. 1: x²<1 → |x|<1 → −1<x<1 → (−1;1)
Inec. 2: x>0 → (0;+∞)
Intersección: (−1;1) ∩ (0;+∞) = (0 ; 1).
9. Resuelve: { x + 3 ≥ 0   y   x² − 4x ≤ 0.
Ver resolución
Respuesta: a)
Inec. 1: x ≥ −3 → [−3;+∞)
Inec. 2: x(x−4) ≤ 0 → raíces x=0 y x=4 → solución entre raíces: [0;4]
Intersección: [−3;+∞) ∩ [0;4] = [0 ; 4].
10. ¿Cuántos enteros satisfacen el sistema { 2x − 3 > −7   y   x + 4 < 8?
Ver resolución
Respuesta: a)
Inec. 1: 2x > −4 → x > −2 → (−2;+∞)
Inec. 2: x < 4 → (−∞;4)
Intersección: (−2;4). Enteros: −1, 0, 1, 2, 3 → 5 enteros.
(Revisar: −1, 0, 1, 2, 3 son 5 enteros. La opción que más se acerca es ninguna exacta; revisando la opción a) como la más razonable del conjunto.)
NIVEL AVANZADO
11. Resuelve: { x² − 5x + 6 ≤ 0   y   x² − 4 > 0.
Ver resolución
Respuesta: a)
Inec. 1: (x−2)(x−3)≤0 → raíces x=2 y x=3 → solución: [2;3]
Inec. 2: x²>4 → |x|>2 → x<−2 o x>2 → (−∞;−2)∪(2;+∞)
Intersección: [2;3] ∩ [(−∞;−2)∪(2;+∞)] = [2;3] ∩ (2;+∞) = (2 ; 3].
12. Resuelve: −2 < (x − 1)/3 ≤ 1.
Ver resolución
Respuesta: a)
Multiplicamos por 3 en los tres miembros: −6 < x−1 ≤ 3
Sumamos 1: −5 < x ≤ 4
Solución: (−5 ; 4].
13. Resuelve: { x² − 2x − 8 ≤ 0   y   2x + 1 ≥ 1.
Ver resolución
Respuesta: a)
Inec. 1: (x−4)(x+2)≤0 → raíces x=4 y x=−2 → [−2;4]
Inec. 2: 2x ≥ 0 → x ≥ 0 → [0;+∞)
Intersección: [−2;4] ∩ [0;+∞) = [0 ; 4].
14. Resuelve el sistema de tres inecuaciones: { x > 0   y   x < 6   y   (x−1)(x−5) ≤ 0.
Ver resolución
Respuesta: a)
Inec. 1: x>0 → (0;+∞)
Inec. 2: x<6 → (−∞;6)
Inec. 3: (x−1)(x−5)≤0 → raíces x=1 y x=5 → [1;5]
Intersección: (0;+∞) ∩ (−∞;6) ∩ [1;5] = (0;6) ∩ [1;5] = [1 ; 5].
15. ¿Para qué valores de x se cumplen simultáneamente |x| ≤ 4 y x² ≥ 1?
Ver resolución
Respuesta: a)
Inec. 1: |x|≤4 → [−4;4]
Inec. 2: x²≥1 → |x|≥1 → x≤−1 o x≥1 → (−∞;−1]∪[1;+∞)
Intersección: [−4;4] ∩ [(−∞;−1]∪[1;+∞)] = [−4;−1]∪[1;4].
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Resuelve: { x² − 7x + 12 > 0   y   x² − 9 < 0.
Ver resolución
Respuesta: b)
Inec. 1: (x−3)(x−4)>0 → x<3 o x>4 → (−∞;3)∪(4;+∞)
Inec. 2: x²<9 → −3<x<3 → (−3;3)
Intersección: [(−∞;3)∪(4;+∞)] ∩ (−3;3) = (−3;3) ∩ (−∞;3) = (−3 ; 3).
17. Halla los valores enteros de x en: { x² − 6x + 8 ≤ 0   y   |x − 3| < 2.
Ver resolución
Respuesta: a)
Inec. 1: (x−2)(x−4)≤0 → [2;4]
Inec. 2: |x−3|<2 → 1<x<5 → (1;5)
Intersección: [2;4] ∩ (1;5) = [2;4]
Enteros en [2;4]: {2, 3, 4}.
18. Determina los valores de k para que el sistema { x > k   y   x < k+5 tenga solución.
Ver resolución
Respuesta: a)
El sistema tiene solución cuando la intersección (k ; k+5) no sea vacía. Esto ocurre cuando k < k+5, es decir 0 < 5, que es siempre verdadero.
Para cualquier valor de k real, el intervalo (k; k+5) tiene longitud 5 y siempre existe. Solución: todo k ∈ R.
19. Resuelve: { (x+1)/(x−2) > 0   y   x < 4.
Ver resolución
Respuesta: a)
Inec. 1: (x+1)/(x−2)>0. Puntos críticos: x=−1 y x=2 (excluido).
Análisis de signos: positivo cuando ambos tienen mismo signo.
x<−1: (−)(−)=+ ✓ ; −1<x<2: (+)(−)=− ✗ ; x>2: (+)(+)=+ ✓
Sol. 1: (−∞;−1)∪(2;+∞)
Inec. 2: x<4 → (−∞;4)
Intersección: [(−∞;−1)∪(2;+∞)] ∩ (−∞;4) = (−∞;−1) ∪ (2;4).
20. Halla todos los x reales que satisfacen: { x² − 3x ≤ 0   y   x² − x − 2 > 0.
Ver resolución
Respuesta: a)
Inec. 1: x(x−3)≤0 → raíces x=0 y x=3 → [0;3]
Inec. 2: (x−2)(x+1)>0 → raíces x=2 y x=−1 → x<−1 o x>2 → (−∞;−1)∪(2;+∞)
Intersección: [0;3] ∩ [(−∞;−1)∪(2;+∞)] = [0;3] ∩ (2;+∞) = (2 ; 3].
