ÁLGEBRA · TEMA 23

Inecuaciones Exponenciales

Monotonía de ax · Igualar bases · Base mayor que 1 vs. base entre 0 y 1

Explicación completa

1. ¿Qué es una inecuación exponencial?

Es una desigualdad en la que la incógnita está en el exponente:
2x > 8  ,  3x−1 ≤ 9  ,  (1/2)x < 1/4
Se resuelven igualando las bases, igual que las ecuaciones exponenciales, pero cuidando una regla extra según el valor de la base.

2. La clave: la monotonía de ax

El comportamiento de una potencia depende de si su base es mayor o menor que 1:
BaseLa función ax es...Al comparar exponentes
a > 1crecienteel sentido se mantiene
0 < a < 1decrecienteel sentido se invierte
Si a > 1:   am > an ⇔ m > n
Si 0 < a < 1:   am > an ⇔ m < n

3. Método de resolución

  1. Escribir ambos lados como potencias de la misma base.
  2. Comparar los exponentes.
  3. Si la base es > 1, conservar el sentido; si está entre 0 y 1, invertirlo.
📐 2x > 8 → 2x > 23. Base 2 > 1 (creciente), conservo: x > 3.

4. ¡Ojo con bases entre 0 y 1!

Cuando la base es una fracción como 1/2, la función decrece, así que el sentido se voltea al pasar a los exponentes.
📐 (1/2)x < 1/4 → (1/2)x < (1/2)2. Base entre 0 y 1, invierto: x > 2.
⚠️ Es el error típico del tema: tratar la base 1/2 igual que la base 2. Recuerda: base menor que 1 → se invierte la desigualdad.

Formulario

BASE MAYOR QUE 1

ax es creciente
am > an ⇔ m > n  (se mantiene)

BASE ENTRE 0 Y 1

ax es decreciente
am > an ⇔ m < n  (se invierte)

ESTRATEGIA

1) Igualar las bases
2) Comparar exponentes
3) Cuidar el sentido según la base
