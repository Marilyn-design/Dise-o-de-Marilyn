ÁLGEBRA · TEMA 14

Binomio de Newton

Desarrollo de (a + b)n · Número de términos · Término general · Coeficientes y casos notables

Explicación completa

1. ¿Qué es el Binomio de Newton?

Es la fórmula que permite desarrollar cualquier potencia de un binomio (a + b)n sin tener que multiplicar término por término. Usa los números combinatorios del Tema 13 como coeficientes.
(a + b)n = C(n,0)an + C(n,1)an−1b + C(n,2)an−2b2 + ... + C(n,n)bn
📐 (a + b)3 = C(3,0)a3 + C(3,1)a2b + C(3,2)ab2 + C(3,3)b3 = a3 + 3a2b + 3ab2 + b3.

2. Número de términos

El desarrollo de (a + b)n siempre tiene un término más que el exponente.
N° de términos = n + 1
📐 (x + 2)5 tiene 5 + 1 = 6 términos.

3. Cómo se comportan los exponentes

  • El exponente de a empieza en n y va bajando hasta 0.
  • El exponente de b empieza en 0 y va subiendo hasta n.
  • En cada término, la suma de ambos exponentes es siempre n.

4. Término general (el más importante)

Para hallar un término cualquiera sin desarrollar todo, se usa el término general. El término que ocupa el lugar (k + 1) es:
tk+1 = C(n, k) · an−k · bk
⚠️ Cuidado con el lugar: el k del combinatorio es una unidad menos que el número de término. Para el 3.er término, k = 2; para el 4.°, k = 3; etc.
📐 Tercer término de (x + 2)4: k = 2 → t3 = C(4,2)·x2·22 = 6·x2·4 = 24x2.

5. Suma de coeficientes y casos notables

  • La suma de coeficientes se obtiene haciendo a = b = 1: para (a + b)n da 2n. En general, se reemplaza cada variable por 1 (es el P(1) del Tema 4).
  • El término independiente es el que no tiene la variable; se halla pidiendo que su exponente sea 0.
  • Los coeficientes son simétricos: igual el primero que el último, el segundo que el penúltimo, etc. (triángulo de Pascal).
📐 Suma de coeficientes de (x + 1)4 = 24 = 16.

Formulario

DESARROLLO Y TÉRMINOS

(a + b)n = Σ C(n,k) an−k bk
N° de términos = n + 1

TÉRMINO GENERAL

tk+1 = C(n, k) · an−k · bk
(para el término N.°, usar k = N − 1)

CASOS NOTABLES

Suma de coeficientes = valor en a = b = 1
(a + b)n con a = b = 1 → 2n
Término independiente: exponente de la variable = 0

Problemas tipo admisión

NIVEL BÁSICO
1. ¿Cuántos términos tiene el desarrollo de (x + 2)5?
Ver resolución
Respuesta: b)
N° de términos = n + 1 = 5 + 1 = 6.
2. Los coeficientes del desarrollo de (a + b)3 son:
Ver resolución
Respuesta: b)
Fila 3 del triángulo de Pascal: 1, 3, 3, 1.
3. ¿Cuál es el primer término del desarrollo de (x + y)4?
Ver resolución
Respuesta: a)
El primer término lleva todo el exponente en a: C(4,0)x4 = x4.
4. Halla la suma de coeficientes de (x + 1)4.
Ver resolución
Respuesta: b)
Reemplazo x = 1: (1 + 1)4 = 24 = 16.
5. Desarrolla: (x + 1)2
Ver resolución
Respuesta: a)
Coeficientes 1, 2, 1: x2 + 2x + 1 = x2 + 2x + 1.
NIVEL INTERMEDIO
6. Halla el tercer término del desarrollo de (x + 2)4.
Ver resolución
Respuesta: a)
Tercer término → k = 2: t3 = C(4,2)x2(2)2 = 6·x2·4 = 24x2.
7. ¿Cuántos términos tiene el desarrollo de (2x − 3)7?
Ver resolución
Respuesta: b)
N° de términos = n + 1 = 7 + 1 = 8.
8. Halla el coeficiente de x2 en el desarrollo de (x + 1)4.
Ver resolución
Respuesta: b)
El término con x2 tiene k = 2: C(4,2)x2(1)2 = 6x2. El coeficiente es 6.
9. Halla la suma de coeficientes de (2x − 1)5.
Ver resolución
Respuesta: a)
Reemplazo x = 1: (2·1 − 1)5 = 15 = 1.
10. Desarrolla: (x + 1)3
Ver resolución
Respuesta: a)
Coeficientes 1, 3, 3, 1: x3 + 3x2 + 3x + 1.
NIVEL AVANZADO
11. Halla el cuarto término del desarrollo de (x + 1)6.
Ver resolución
Respuesta: a)
Cuarto término → k = 3: t4 = C(6,3)x6−3(1)3 = 20x3 = 20x3.
12. Halla el coeficiente de x3 en el desarrollo de (x + 2)5.
Ver resolución
Respuesta: a)
Término general: C(5,k)x5−k2k. Para x3 necesito 5 − k = 3 → k = 2: C(5,2)·22 = 10·4 = 40.
13. Halla el término central del desarrollo de (x + 1)4.
Ver resolución
Respuesta: a)
(x + 1)4 tiene 5 términos; el central es el 3.° (k = 2): C(4,2)x2 = 6x2.
14. En el desarrollo de (x + a)4, el coeficiente de x3 es 8. Halla "a".
Ver resolución
Respuesta: a)
El término con x3 es C(4,1)x3a = 4a·x3. Su coeficiente es 4a = 8 → a = 2.
15. Halla la suma de coeficientes de (3x − 2)4.
Ver resolución
Respuesta: a)
Reemplazo x = 1: (3 − 2)4 = 14 = 1.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Halla el término que contiene x2 en el desarrollo de (x + 1/x)4.
Ver resolución
Respuesta: a)
Término general: C(4,k)x4−k(1/x)k = C(4,k)x4−2k. Para x2 necesito 4 − 2k = 2 → k = 1: C(4,1)x2 = 4x2.
17. Halla el término independiente del desarrollo de (x + 1/x)6.
Ver resolución
Respuesta: a)
Término general: C(6,k)x6−k(1/x)k = C(6,k)x6−2k. Independiente → 6 − 2k = 0 → k = 3: C(6,3) = 20.
18. Si la suma de coeficientes de (x + 1)n es 64, halla "n".
Ver resolución
Respuesta: a)
Suma de coeficientes = 2n = 64 = 26 → n = 6.
19. Halla el término central del desarrollo de (x + 2)6.
Ver resolución
Respuesta: a)
(x + 2)6 tiene 7 términos; el central es el 4.° (k = 3): C(6,3)x323 = 20·8·x3 = 160x3.
20. Halla el término independiente del desarrollo de (x2 + 1/x)6.
Ver resolución
Respuesta: a)
Término general: C(6,k)(x2)6−k(1/x)k = C(6,k)x12−3k. Independiente → 12 − 3k = 0 → k = 4: C(6,4) = 15.
