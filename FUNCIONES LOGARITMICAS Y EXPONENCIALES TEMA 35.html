Funciones Logarítmicas y Exponenciales

Función exponencial · Crecimiento y decrecimiento exponencial · Función logarítmica · Relación inversa · Traslaciones y transformaciones

Explicación completa

1. Función exponencial: f(x) = bx

La función exponencial tiene la variable en el exponente. La base b es una constante positiva distinta de 1. No confundir con las funciones potencia f(x) = xn, donde la variable está en la base.
f(x) = bx    con    b > 0,   b ≠ 1,   x ∈ R
Características fundamentales:
  • Dominio: R (todos los reales, el exponente puede ser cualquier número)
  • Rango: (0 ; +∞) — la exponencial nunca toca ni cruza el eje X
  • Punto fijo: siempre pasa por (0, 1) porque b0 = 1
  • Asíntota horizontal: y = 0 (el eje X)
  • La función es siempre positiva: bx > 0 para todo x real

2. Crecimiento y decrecimiento exponencial

Los modelos exponenciales aparecen en biología (poblaciones), física (desintegración) y finanzas (interés compuesto). La forma general con un factor inicial es:
Crecimiento:   f(t) = A · bt   con b > 1
Decrecimiento:   f(t) = A · bt   con 0 < b < 1
Donde A es el valor inicial (cuando t = 0): f(0) = A · b0 = A.
📐 Crecimiento: Una bacteria se duplica cada hora. Si al inicio hay 100 bacterias: P(t) = 100 · 2t.
P(3) = 100 · 23 = 100 · 8 = 800 bacterias a las 3 horas.
📐 Decrecimiento: Un medicamento pierde la mitad de su concentración cada hora. C(0) = 80 mg: C(t) = 80 · (1/2)t.
C(4) = 80 · (1/16) = 5 mg al cabo de 4 horas.

3. Función logarítmica: f(x) = logb x

La función logarítmica es la inversa de la función exponencial. Sus propiedades se derivan directamente de esa relación.
f(x) = logb x    con    b > 0,   b ≠ 1,   x > 0
Características fundamentales:
  • Dominio: (0 ; +∞) — el argumento debe ser positivo
  • Rango: R (puede tomar cualquier valor real)
  • Punto fijo: siempre pasa por (1, 0) porque logb 1 = 0
  • Asíntota vertical: x = 0 (el eje Y)
  • La función es creciente si b > 1 y decreciente si 0 < b < 1

4. Relación inversa entre exponencial y logarítmica

Las funciones f(x) = bx y g(x) = logb x son inversas entre sí. Sus gráficas son simétricas respecto a la recta y = x.
blogb x = x      logb(bx) = x

5. Transformaciones de la función exponencial y logarítmica

A partir de las formas base f(x) = bx y f(x) = logb x, podemos construir funciones trasladadas o reflejadas:

📐 f(x) = 2x−3 + 1:
• Asíntota horizontal: y = 1 (subió 1 unidad desde y=0)
• Punto base trasladado: (0,1) → (3, 2) (se movió 3 unidades a la derecha y 1 hacia arriba)
• Dom = R  |  Ran = (1 ; +∞)
📐 f(x) = log3(x + 2):
• Asíntota vertical: x = −2 (se movió 2 unidades a la izquierda desde x=0)
• Punto base trasladado: (1,0) → (−1, 0)
• Dom = (−2 ; +∞)  |  Ran = R
⚠️ Regla clave para el dominio del logaritmo trasladado:
En f(x) = logb(x − h), el dominio se obtiene imponiendo x − h > 0, es decir x > h.
En f(x) = logb(ax + b), el dominio es ax + b > 0 → despejar x normalmente, cuidando el signo de a.

6. La función exponencial natural: f(x) = ex

La base e ≈ 2,71828... (número de Euler) da origen a la exponencial natural, la más importante en matemáticas avanzadas y ciencias. Su inversa es el logaritmo natural ln x.
f(x) = ex      f−1(x) = ln x      eln x = x      ln(ex) = x
📐 Evalúa: eln 7 = 7  |  ln(e3) = 3  |  ln(e) = 1  |  ln(1) = 0

Problemas tipo admisión

NIVEL BÁSICO
1. ¿Cuál es el dominio de f(x) = 3x?
Ver resolución
Respuesta: a)
La función exponencial bx admite cualquier exponente real. Dom = R.
2. ¿Por qué punto pasa siempre la gráfica de f(x) = bx?
Ver resolución
Respuesta: a)
f(0) = b0 = 1 para cualquier base b válida. La curva siempre cruza el eje Y en (0, 1).
3. ¿Cuál es el rango de f(x) = log4 x?
Ver resolución
Respuesta: a)
La función logarítmica tiene Ran = R; puede tomar cualquier valor real (positivos, negativos y cero).
4. Si f(x) = 5x, ¿cuánto vale f(0)?
Ver resolución
Respuesta: a)
f(0) = 50 = 1. Todo número (distinto de cero) elevado a la potencia 0 es igual a 1.
5. ¿Cuál es la asíntota vertical de f(x) = log2 x?
Ver resolución
Respuesta: a)
La función logb x no está definida para x ≤ 0; la gráfica se acerca indefinidamente al eje Y. Asíntota: x = 0.
NIVEL INTERMEDIO
6. ¿Cuál es el dominio de f(x) = log3(x − 5)?
Ver resolución
Respuesta: a)
Condición: x − 5 > 0 → x > 5. Dom = (5 ; +∞). La asíntota vertical se desplazó a x = 5.
7. Una población de bacterias sigue P(t) = 200 · 3t. ¿Cuántas bacterias hay a las 3 horas?
Ver resolución
Respuesta: a)
P(3) = 200 · 33 = 200 · 27 = 5 400 bacterias.
8. ¿Cuál es la asíntota horizontal de f(x) = 2x + 4?
Ver resolución
Respuesta: a)
La función base 2x tiene asíntota y=0. Al sumarle 4, la asíntota se traslada a y = 4. El rango pasa a ser (4;+∞).
9. Evalúa: eln 9.
Ver resolución
Respuesta: a)
Por la propiedad de funciones inversas: eln x = x. Entonces eln 9 = 9.
10. ¿Cuál es el dominio de f(x) = log2(3x − 9)?
Ver resolución
Respuesta: a)
Condición: 3x − 9 > 0 → 3x > 9 → x > 3. Dom = (3 ; +∞).
NIVEL AVANZADO
11. Un capital de S/1 000 crece según C(t) = 1000 · (1,05)t. ¿Cuánto vale al cabo de 2 años?
Ver resolución
Respuesta: a)
C(2) = 1000 · (1,05)2 = 1000 · 1,1025 = S/ 1 102,50.
Nota: los 0,05 de cada año se acumulan sobre el total anterior (interés compuesto).
12. ¿Cuál es el dominio de f(x) = log5(x2 − 4)?
Ver resolución
Respuesta: a)
Condición: x2 − 4 > 0 → x2 > 4 → |x| > 2 → x < −2 o x > 2.
Dom = (−∞;−2) ∪ (2;+∞).
13. La función f(x) = 2x−1 − 3 tiene asíntota horizontal y = −3. ¿Cuál es su rango?
Ver resolución
Respuesta: a)
2x−1 > 0 siempre, entonces 2x−1 − 3 > −3 siempre. La función se acerca pero nunca alcanza y = −3.
Ran = (−3 ; +∞).
14. ¿Cuál es el punto de intersección con el eje X de f(x) = log2(x − 3)?
Ver resolución
Respuesta: a)
f(x) = 0 cuando log2(x − 3) = 0 → x − 3 = 20 = 1 → x = 4.
Punto de corte con eje X: (4, 0). (El punto base (1,0) se trasladó 3 unidades a la derecha.)
15. ¿Para qué valor de x se cumple 2x = 32?
Ver resolución
Respuesta: a)
32 = 25. Igualando bases: 2x = 25 → x = 5.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Halla el dominio de f(x) = log3((x − 1)/(x + 2)).
Ver resolución
Respuesta: a)
Necesitamos (x−1)/(x+2) > 0. Esto ocurre cuando numerador y denominador tienen el mismo signo.
Casos: ambos positivos: x>1 y x>−2 → x>1  |  ambos negativos: x<1 y x<−2 → x<−2
Dom = (−∞;−2) ∪ (1;+∞).
17. Si f(x) = 4x y g(x) = 2x+1, ¿para qué valor de x se tiene f(x) = g(x)?
Ver resolución
Respuesta: a)
4x = 2x+1 → (22)x = 2x+1 → 22x = 2x+1
Igualando exponentes: 2x = x + 1 → x = 1.
Verificación: 41=4 y 22=4 ✓
18. Una sustancia radiactiva se reduce según M(t) = 640·(1/2)t. ¿En cuántas horas quedan 10 gramos?
Ver resolución
Respuesta: a)
640 · (1/2)t = 10 → (1/2)t = 10/640 = 1/64 = (1/2)6
Igualando exponentes: t = 6 horas.
19. ¿Cuál es la inversa de f(x) = 3x − 2?
Ver resolución
Respuesta: a)
y = 3x − 2 → y + 2 = 3x → x = log3(y + 2)
f−1(x) = log3(x + 2). Dom(f−1) = (−2 ; +∞) = Ran(f).
20. Dada f(x) = logb(x) con b > 1. Si f(p) = 3 y f(q) = 5, ¿cuánto vale f(p · q)?
Ver resolución
Respuesta: a)
f(p·q) = logb(p·q) = logbp + logbq = f(p) + f(q) = 3 + 5 = 8.
(Aplicación directa de la propiedad del producto de logaritmos.)
