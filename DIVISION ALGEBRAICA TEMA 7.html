ÁLGEBRA · TEMA 7

División Algebraica

Identidad fundamental · Grados · Método de Horner · Método de Ruffini · Teorema del resto

Explicación completa

1. ¿Qué es dividir polinomios?

Dividir polinomios es lo mismo que dividir números, pero con letras. Cuando divides 17 entre 5 obtienes un cociente (3) y un residuo (2), y se cumple que 17 = 5·3 + 2. Con polinomios pasa exactamente igual. Toda división de polinomios tiene cuatro partes: el dividendo D(x) (lo que se reparte), el divisor d(x) (entre cuánto se reparte), el cociente q(x) (el resultado) y el residuo R(x) (lo que sobra).
D(x) = d(x) · q(x) + R(x)
Esta es la identidad fundamental de la división. Memorízala: casi todos los problemas de este tema salen de despejar algo de aquí.
⚠️ Antes de dividir, SIEMPRE: ordena el polinomio de mayor a menor exponente y complétalo: si falta algún grado, escribe ese término con coeficiente 0. Por ejemplo, x³ + 2 debe escribirse como x³ + 0x² + 0x + 2. Olvidar los ceros es el error número uno en Horner y Ruffini.

2. Grados del cociente y del residuo

Antes incluso de operar, ya puedes saber el grado del resultado:
  • Grado del cociente:   grado(q) = grado(D) − grado(d)
  • Grado máximo del residuo:   grado(R)máx = grado(d) − 1
📐 Ejemplo: si divides un polinomio de grado 7 entre uno de grado 3, el cociente será de grado 7 − 3 = 4, y el residuo tendrá como máximo grado 3 − 1 = 2.
⚠️ División exacta: cuando el residuo es cero (R = 0), la división es exacta y se dice que el divisor es un factor del dividendo. Esto conecta directamente con la factorización (Tema 9).

3. Método de Horner

Horner sirve para dividir entre divisores de cualquier grado mayor o igual a 2. Es un esquema de cajas donde solo trabajamos con los coeficientes.

Pasos:

  1. Ordena y completa dividendo y divisor.
  2. Coloca los coeficientes del dividendo en la fila superior.
  3. A la izquierda pon los coeficientes del divisor: el primero tal cual; a los demás les cambias el signo.
  4. Divides la primera columna entre el coeficiente principal del divisor para obtener el primer término del cociente, lo multiplicas por los coeficientes cambiados y colocas en diagonal. Repites.
  5. La línea que separa cociente y residuo se traza dejando a la derecha tantas columnas como el grado del divisor.
📐 Ejemplo resuelto: dividir   x4 + 2x3 − 3x2 + 5x − 1   entre   x2 + x − 2.
El divisor tiene coeficiente principal 1 (no hay que dividir). Los otros coeficientes (+1 y −2) cambian de signo: −1 y +2.

4. Método de Ruffini

Ruffini es un caso particular (más rápido) de Horner, que solo se usa cuando el divisor es de primer grado, de la forma (x ± a).

Pasos para dividir entre (x − a):

  1. Escribe los coeficientes del dividendo (ordenado y completo).
  2. A la izquierda coloca el valor de a (el que hace cero al divisor: si es x − 3, usas 3; si es x + 2, usas −2).
  3. Baja el primer coeficiente. Multiplícalo por a, súmalo al siguiente, y repite hasta el final.
  4. El último número es el residuo; los anteriores son el cociente (de un grado menos).
📐 Ejemplo: dividir 2x3 − 3x2 + 4x − 5 entre (x − 2). Usamos a = 2.
• Bajo el 2 → 2·2 = 4 ; −3 + 4 = 1
• 1·2 = 2 ; 4 + 2 = 6
• 6·2 = 12 ; −5 + 12 = 7 (residuo)
Cociente: 2x2 + x + 6   Residuo: 7.
⚠️ Divisor con coeficiente (ax ± b): si el divisor es, por ejemplo, (2x − 1), usas a = 1/2 en Ruffini y al final divides el cociente entre 2 (el coeficiente que acompaña a la x). El residuo no se divide.

5. Teorema del resto (de Descartes)

Es el atajo más poderoso del tema. Sirve para hallar solo el residuo sin hacer toda la división.
El resto de dividir P(x) entre (x − a) es igual a P(a)
Es decir: para hallar el residuo, simplemente reemplazas en el polinomio la x por el valor que anula al divisor.
📐 Ejemplo: resto de dividir P(x) = x3 − 2x2 + 3x − 4 entre (x − 1).
Reemplazo x = 1: P(1) = 1 − 2 + 3 − 4 = −2. Ese es el residuo, sin dividir nada.
⚠️ Para divisores tipo (ax − b): el residuo es P(b/a). Por ejemplo, el resto de dividir entre (2x − 3) es P(3/2).

6. ¿Qué método usar en cada caso?

SituaciónMejor método
Solo necesito el residuo y el divisor es de primer gradoTeorema del resto (reemplazar)
Necesito cociente y residuo, divisor de primer grado (x ± a)Ruffini
Divisor de grado 2 o mayorHorner
Divisor cómodo de factorizar o muy simpleDivisión clásica (distributiva)

Formulario

IDENTIDAD Y GRADOS

D(x) = d(x) · q(x) + R(x)
grado(q) = grado(D) − grado(d)
grado(R) máximo = grado(d) − 1

TEOREMA DEL RESTO

Resto de P(x) ÷ (x − a) = P(a)
Resto de P(x) ÷ (ax − b) = P(b/a)
Si P(a) = 0 → (x − a) es factor de P(x)

MÉTODOS

Ruffini: divisores de primer grado (x ± a)
Horner: divisores de grado ≥ 2
En ambos: cambiar el signo a los coeficientes del divisor (menos al principal) y ordenar/completar el dividendo.

Problemas tipo admisión

NIVEL BÁSICO
1. Al dividir un polinomio de grado 7 entre un divisor de grado 3, ¿cuál es el grado del cociente?
Ver resolución
Respuesta: b)
grado(q) = grado(D) − grado(d) = 7 − 3 = 4.
2. ¿Cuál es el grado máximo que puede tener el residuo al dividir entre un divisor de grado 4?
Ver resolución
Respuesta: c)
El residuo siempre tiene grado menor que el divisor, así que su grado máximo es grado(d) − 1 = 4 − 1 = 3.
3. Por el teorema del resto, ¿cuál es el residuo de dividir P(x) entre (x − 3)?
Ver resolución
Respuesta: c)
El residuo es el valor del polinomio cuando x toma el valor que anula al divisor. x − 3 = 0 → x = 3 → residuo = P(3).
4. Halla el residuo de dividir P(x) = x2 + 5x + 6 entre (x + 1).
Ver resolución
Respuesta: a)
x + 1 = 0 → x = −1. Reemplazo: P(−1) = (−1)2 + 5(−1) + 6 = 1 − 5 + 6 = 2.
5. En la identidad D = d·q + R, si la división es exacta, ¿cuánto vale R?
Ver resolución
Respuesta: c)
Una división es exacta cuando no sobra nada, es decir, el residuo es 0. En ese caso D = d·q y el divisor es factor del dividendo.
NIVEL INTERMEDIO
6. Por Ruffini, halla el cociente de dividir x3 − 6x2 + 11x − 6 entre (x − 1).
Ver resolución
Respuesta: a)
Ruffini con a = 1: bajo 1 ; 1·1=1, −6+1=−5 ; −5·1=−5, 11−5=6 ; 6·1=6, −6+6=0 (residuo 0, exacta).
Cociente: x2 − 5x + 6.
7. Halla el residuo de dividir P(x) = 2x4 − 3x2 + x − 5 entre (x − 2).
Ver resolución
Respuesta: a)
Teorema del resto, x = 2: P(2) = 2(16) − 3(4) + 2 − 5 = 32 − 12 + 2 − 5 = 17.
8. Si la división de P(x) entre (x − 2) deja residuo cero, entonces necesariamente:
Ver resolución
Respuesta: c)
Por el teorema del resto, residuo = P(2). Si el residuo es 0, entonces P(2) = 0, lo que significa que x = 2 es raíz y que (x − 2) es factor de P(x).
9. Halla el residuo de dividir P(x) = x3 + 2x2 − x + 4 entre (x + 2).
Ver resolución
Respuesta: a)
x + 2 = 0 → x = −2. P(−2) = (−8) + 2(4) − (−2) + 4 = −8 + 8 + 2 + 4 = 6.
10. Halla el cociente de dividir x2 + 7x + 12 entre (x + 3).
Ver resolución
Respuesta: b)
x2 + 7x + 12 = (x + 3)(x + 4) (Stevin: 3+4=7, 3·4=12). Al dividir entre (x + 3) queda x + 4, exacto.
NIVEL AVANZADO
11. Halla el valor de "m" para que la división (x3 + mx2 + 2x − 6) entre (x − 1) sea exacta.
Ver resolución
Respuesta: a)
Exacta → residuo 0 → P(1) = 0. P(1) = 1 + m + 2 − 6 = m − 3 = 0 → m = 3.
12. Halla el residuo de dividir x4 + 2x3 − x + 3 entre (x2 − 1).
Ver resolución
Respuesta: a)
Como el divisor es x2 − 1, en el residuo podemos usar que x2 equivale a 1. Entonces x4 = (x2)2 → 1, y 2x3 = 2x·x2 → 2x. La expresión se reduce a 1 + 2x − x + 3 = x + 4 (grado 1, menor que el divisor → es el residuo).
13. Un polinomio P(x) se divide entre (x + 1)(x − 2) dejando residuo R(x) = ax + b. Si P(−1) = 3 y P(2) = 9, halla R(x).
Ver resolución
Respuesta: a)
Como P(x) = (x+1)(x−2)·q(x) + R(x), al reemplazar las raíces del divisor el primer término se anula y queda P = R.
R(−1) = P(−1) = 3 → −a + b = 3
R(2) = P(2) = 9 → 2a + b = 9
Restando: 3a = 6 → a = 2 ; b = 5. Entonces R(x) = 2x + 5.
14. Halla "n" si el residuo de dividir (x3 − 2x2 + nx − 1) entre (x − 2) es 5.
Ver resolución
Respuesta: a)
Residuo = P(2) = 8 − 8 + 2n − 1 = 2n − 1. Igualamos a 5: 2n − 1 = 5 → 2n = 6 → n = 3.
15. Si P(x) deja resto 5 al dividirse entre (x − 3), ¿cuánto vale P(3)?
Ver resolución
Respuesta: b)
El teorema del resto dice que el resto de dividir entre (x − 3) es exactamente P(3). Si el resto es 5, entonces P(3) = 5.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Un polinomio P(x) deja resto 2 al dividirse entre (x − 1) y resto 4 al dividirse entre (x − 3). Halla el resto de dividir P(x) entre (x − 1)(x − 3).
Ver resolución
Respuesta: a)
El divisor es de grado 2, así que el resto es de la forma R(x) = ax + b. Por el teorema del resto: P(1) = 2 y P(3) = 4, que al reemplazar en R dan:
a + b = 2   ;   3a + b = 4. Restando: 2a = 2 → a = 1, b = 1. Entonces R(x) = x + 1.
17. El polinomio x4 − 3x2 + ax + b es divisible entre (x2 − x − 2). Halla a + b.
Ver resolución
Respuesta: a)
x2 − x − 2 = (x − 2)(x + 1). Para que sea divisible, P(2) = 0 y P(−1) = 0.
P(2) = 16 − 12 + 2a + b = 4 + 2a + b = 0
P(−1) = 1 − 3 − a + b = −2 − a + b = 0
De la segunda: b = a + 2. Sustituyendo: 4 + 2a + a + 2 = 0 → 3a = −6 → a = −2, b = 0. Por tanto a + b = −2.
18. Halla el residuo de dividir x100 entre (x + 1).
Ver resolución
Respuesta: a)
Teorema del resto: x + 1 = 0 → x = −1. Residuo = (−1)100. Como 100 es par, (−1)100 = 1.
