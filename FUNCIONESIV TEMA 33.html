ÁLGEBRA · TEMA 33

Funciones IV: Función Inversa y Modelos Funcionales

Condición de existencia · Cómo hallar la inversa · Gráfica y simetría · Modelos lineales, cuadráticos y racionales

Explicación completa

1. ¿Qué es la función inversa?

La función inversa de f, denotada f−1, es aquella que deshace lo que hace f. Si f lleva x a y, entonces f−1 lleva y de vuelta a x.
Si f(x) = y    entonces    f−1(y) = x
La relación clave entre una función y su inversa es:
(f ° f−1)(x) = x    y    (f−1 ° f)(x) = x
Esto significa que componer una función con su inversa siempre da como resultado la función identidad I(x) = x.
⚠️ Condición indispensable: solo las funciones biyectivas (inyectivas y sobreyectivas) tienen función inversa definida en todo su conjunto de llegada. Una función que repite imágenes (no inyectiva) no puede invertirse, porque la inversa no sabría a cuál x original volver.

2. ¿Cómo se halla la función inversa? (Método paso a paso)

Para obtener f−1(x) a partir de y = f(x), se siguen estos 3 pasos:
  1. Escribir y = f(x).
  2. Despejar x en función de y.
  3. Intercambiar las letras: donde dice y escribir x (y viceversa). La expresión resultante es f−1(x).
📐 Ejemplo 1: Hallar la inversa de f(x) = 3x − 6.
Paso 1: y = 3x − 6
Paso 2: y + 6 = 3x → x = (y + 6) / 3
Paso 3: f−1(x) = (x + 6) / 3
Verificación: f(f−1(x)) = 3 · (x+6)/3 − 6 = x + 6 − 6 = x ✓
📐 Ejemplo 2: Hallar la inversa de f(x) = x3 + 2.
Paso 1: y = x3 + 2
Paso 2: y − 2 = x3 → x = ∛(y − 2)
Paso 3: f−1(x) = ∛(x − 2)
📐 Ejemplo 3: Hallar la inversa de f(x) = (2x + 1) / (x − 3), x ≠ 3.
Paso 1: y = (2x + 1) / (x − 3)
Paso 2: y(x − 3) = 2x + 1 → yx − 3y = 2x + 1 → yx − 2x = 3y + 1 → x(y − 2) = 3y + 1 → x = (3y + 1)/(y − 2)
Paso 3: f−1(x) = (3x + 1) / (x − 2), x ≠ 2

3. Dominio y rango de la función inversa

Existe una relación muy elegante: el dominio y el rango se intercambian al tomar la inversa.
f(x)f−1(x)
DominioAB (= rango de f)
RangoBA (= dominio de f)
📐 Si f(x) = √(x − 1) tiene Dom = [1;+∞) y Ran = [0;+∞), entonces:
f−1(x) = x2 + 1, con Dom = [0;+∞) y Ran = [1;+∞)

4. Gráfica de la función inversa y la recta y = x

Las gráficas de f y f−1 son simétricas respecto a la recta y = x. Si un punto (a, b) pertenece a f, entonces el punto (b, a) pertenece a f−1: simplemente se intercambian las coordenadas.

5. Modelos funcionales

Los modelos funcionales son funciones que representan situaciones del mundo real. En los exámenes de admisión aparecen principalmente tres tipos:

5.1 Modelo lineal: f(x) = mx + b

Representa situaciones con variación constante (la variable crece o decrece a ritmo fijo). m es la tasa de cambio (pendiente) y b es el valor inicial.
📐 Un taxi cobra S/5 de bajada de bandera y S/2 por kilómetro. El costo es: C(x) = 2x + 5, donde x es la distancia en km.
C(8) = 2(8) + 5 = S/21

5.2 Modelo cuadrático: f(x) = ax2 + bx + c

Representa trayectorias parabólicas, áreas o fenómenos con máximo o mínimo. El vértice indica el punto óptimo.
📐 La altura de un proyectil es h(t) = −5t2 + 20t + 0 (metros, t en segundos).
Vértice: t = −20/[2(−5)] = 2 s. Altura máxima: h(2) = −5(4) + 40 = 20 m

5.3 Modelo racional: f(x) = k/x o f(x) = (ax+b)/(cx+d)

Representa situaciones de variación inversamente proporcional o con asíntotas. Muy frecuente en problemas de velocidad, precio y producción.
📐 Si 6 obreros construyen un muro en 10 días, ¿cuántos días tardan x obreros? D(x) = 60/x.
D(4) = 60/4 = 15 días

6. Estrategia para resolver problemas con modelos

  1. Identificar qué tipo de relación describe el problema (lineal, cuadrática, racional).
  2. Definir variables claramente (cuál es x, cuál es f(x)).
  3. Construir la función con los datos del enunciado.
  4. Evaluar o resolver según lo que se pide (hallar f(x), hallar x dado f(x), encontrar máximo, etc.).
⚠️ Error frecuente: confundir f(x) = y con f−1(x). Recuerda: si el enunciado te da el resultado y te pide el valor inicial, estás aplicando la función inversa, no la función directa.

Formulario

FUNCIÓN INVERSA — PASOS

1. Escribir y = f(x)
2. Despejar x en función de y
3. Intercambiar x ↔ y → resultado es f−1(x)

PROPIEDADES CLAVE

Existe f−1 solo si f es biyectiva
(f ° f−1)(x) = (f−1 ° f)(x) = x
Dom(f−1) = Ran(f)  |  Ran(f−1) = Dom(f)
Simetría gráfica: respecto a y = x

MODELOS FUNCIONALES

Lineal: f(x) = mx + b → variación constante
Cuadrático: f(x) = ax2+bx+c → vértice en t = −b/2a
Racional: f(x) = k/x → variación inversa

Problemas tipo admisión

NIVEL BÁSICO
1. Si f(x) = x + 7, ¿cuál es f−1(x)?
Ver resolución
Respuesta: a)
y = x + 7 → x = y − 7 → f−1(x) = x − 7.
Verificación: f(f−1(x)) = (x−7)+7 = x ✓
2. Si f(x) = 4x, ¿cuál es f−1(x)?
Ver resolución
Respuesta: a)
y = 4x → x = y/4 → f−1(x) = x/4.
3. ¿Respecto a qué recta son simétricas las gráficas de f y f−1?
Ver resolución
Respuesta: a)
Las gráficas de f y su inversa son simétricas respecto a la bisectriz del primer y tercer cuadrante: y = x.
4. Un electricista cobra S/30 de visita más S/50 por hora. ¿Qué modelo representa el costo C en función de las horas h?
Ver resolución
Respuesta: a)
El costo fijo (visita) es 30 y la tasa es 50 por hora. Modelo lineal: C(h) = 50h + 30.
5. Si f(x) = 2x − 3 y f−1(a) = 4, ¿cuánto vale a?
Ver resolución
Respuesta: a)
f−1(a) = 4 significa f(4) = a. f(4) = 2(4)−3 = 8−3 = 5.
NIVEL INTERMEDIO
6. Halla la inversa de f(x) = 3x + 9.
Ver resolución
Respuesta: a)
y = 3x + 9 → y − 9 = 3x → x = (y−9)/3
f−1(x) = (x − 9)/3. Verificación: f((x−9)/3) = 3·(x−9)/3 + 9 = x − 9 + 9 = x ✓
7. Si Dom(f) = [2 ; +∞) y Ran(f) = [0 ; +∞), ¿cuál es el dominio de f−1?
Ver resolución
Respuesta: a)
El dominio de f−1 es igual al rango de f. Como Ran(f) = [0;+∞), entonces Dom(f−1) = [0 ; +∞).
8. Un proyectil sigue h(t) = −5t2 + 30t. ¿En qué momento alcanza la altura máxima?
Ver resolución
Respuesta: a)
El vértice de h(t) = −5t2 + 30t ocurre en t = −b/(2a) = −30/(2·(−5)) = −30/(−10) = 3 s.
Altura máxima: h(3) = −5(9) + 90 = −45 + 90 = 45 m.
9. Si 4 grifos llenan un tanque en 12 horas, ¿qué función modela las horas H según el número n de grifos?
Ver resolución
Respuesta: a)
La cantidad de trabajo es constante: n · H = 4 · 12 = 48. Despejando: H(n) = 48/n.
Verificación: H(4) = 48/4 = 12 ✓. Modelo racional de variación inversa.
10. Halla la inversa de f(x) = x3 − 5.
Ver resolución
Respuesta: a)
y = x3 − 5 → y + 5 = x3 → x = ∛(y+5)
f−1(x) = ∛(x + 5).
NIVEL AVANZADO
11. Halla la inversa de f(x) = (x + 2) / (x − 1), con x ≠ 1.
Ver resolución
Respuesta: a)
y = (x+2)/(x−1) → y(x−1) = x+2 → yx−y = x+2 → yx−x = y+2 → x(y−1) = y+2 → x = (y+2)/(y−1)
f−1(x) = (x+2)/(x−1). ¡Esta función es su propia inversa! Se llama función involutiva.
12. Si f−1(x) = 2x + 4, ¿cuál es f(x)?
Ver resolución
Respuesta: a)
Si f−1(x) = 2x + 4, hallamos su inversa para obtener f(x).
y = 2x + 4 → y − 4 = 2x → x = (y−4)/2
f(x) = (x − 4)/2. Verificación: f(f−1(x)) = (2x+4−4)/2 = 2x/2 = x ✓
13. Una tienda vende juguetes a S/15 cada uno. El costo de producción es C(x) = 8x + 420, donde x es la cantidad producida. ¿Cuántas unidades se deben vender para cubrir exactamente los costos?
Ver resolución
Respuesta: a)
Ingreso = 15x. Costo = 8x + 420. Igualando: 15x = 8x + 420 → 7x = 420 → x = 60 unidades.
14. Si f(x) = √(x − 3) (x ≥ 3), ¿cuál es f−1(x) y su dominio?
Ver resolución
Respuesta: a)
y = √(x−3) → y2 = x − 3 → x = y2 + 3
f−1(x) = x2 + 3. Su dominio es el rango de f: Ran(f) = [0;+∞), entonces Dom(f−1) = [0 ; +∞).
15. El precio de un producto varía inversamente con la demanda. Si a 50 unidades el precio es S/80, ¿cuál es el precio cuando la demanda es 200 unidades?
Ver resolución
Respuesta: a)
Modelo racional: P(x) = k/x. Con P(50)=80: k = 80·50 = 4000.
P(200) = 4000/200 = S/20.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Si f(x) = (3x − 2)/(x + 4), halla f−1(x) y su dominio.
Ver resolución
Respuesta: a)
y = (3x−2)/(x+4) → y(x+4) = 3x−2 → yx+4y = 3x−2 → yx−3x = −2−4y → x(y−3) = −4y−2
x = (−4y−2)/(y−3) = (−4x−2)/(x−3), con Dom = R − {3}.
17. Si f(f−1(5)) + f−1(f(3)) = n, ¿cuánto vale n?
Ver resolución
Respuesta: a)
Por propiedad: (f ° f−1)(x) = x y (f−1 ° f)(x) = x.
f(f−1(5)) = 5    y    f−1(f(3)) = 3
n = 5 + 3 = 8.
18. Una pelota se lanza desde el suelo y su altura es h(t) = −4t2 + 16t. ¿Cuáles son los instantes en que la pelota toca el suelo?
Ver resolución
Respuesta: a)
h(t) = 0 → −4t2 + 16t = 0 → −4t(t − 4) = 0 → t = 0 o t = 4.
La pelota parte del suelo (t=0) y regresa al suelo en t=4 s.
19. Si f(x) = ax + b y f−1(x) = (1/3)x − 2, ¿cuánto valen a y b?
Ver resolución
Respuesta: a)
Hallamos la inversa de f−1(x) = (1/3)x − 2:
y = (1/3)x − 2 → y + 2 = x/3 → x = 3y + 6
f(x) = 3x + 6, entonces a = 3, b = 6.
20. Una empresa produce x unidades con costo C(x) = x2 − 80x + 2000. ¿Cuántas unidades minimiza el costo y cuál es ese costo mínimo?
Ver resolución
Respuesta: a)
C(x) = x2 − 80x + 2000, con a = 1 > 0 (parábola con mínimo).
Vértice en x = −(−80)/(2·1) = 80/2 = 40 unidades.
C(40) = 1600 − 3200 + 2000 = S/400 de costo mínimo.
