ÁLGEBRA · TEMA 30

Funciones I: Dominio y Rango

Definición de función · Criterio de la línea vertical · Dominio · Rango · Notación f(x)

Explicación completa

1. ¿Qué es una función?

Una función es una relación binaria especial: es una regla que asigna a cada elemento x del dominio exactamente un único elemento y del rango. En otras palabras, una función es una relación en la que ningún valor de x se repite con diferentes valores de y.
f : A → B   es función si para todo x ∈ A existe un único y ∈ B tal que (x, y) ∈ f
⚠️ Diferencia clave: toda función es una relación, pero no toda relación es una función. Lo que distingue a una función es la unicidad: a cada x le corresponde uno y solo un y.
📐 La relación y = 2x + 1 sí es función: a cada x le corresponde exactamente un y. En cambio, x2 + y2 = 25 no es función: para x = 0 se obtiene y = 5 y también y = −5 (dos valores distintos de y para el mismo x).

2. Criterio de la línea vertical (prueba gráfica)

Para determinar visualmente si una gráfica representa una función, se usa la prueba de la línea vertical: si cualquier recta vertical corta la gráfica en a lo sumo un punto, la relación es una función. Si existe alguna recta vertical que corta la gráfica en dos o más puntos, no es función.

3. Notación funcional f(x)

En lugar de escribir y, usamos la notación f(x) para nombrar la función. Se lee "f de x" y representa el valor que la función asigna a x. Esta notación permite evaluar la función en cualquier valor de manera clara.
Si f(x) = 3x − 2, entonces  f(4) = 3(4) − 2 = 10
📐 Si g(x) = x2 + 1, calcular g(0), g(3) y g(−2):
g(0) = 02 + 1 = 1  |  g(3) = 9 + 1 = 10  |  g(−2) = 4 + 1 = 5

4. Dominio de una función

El dominio (Dom f) es el conjunto de todos los valores de x para los que la función está definida (produce un resultado real). Para hallarlo, se eliminan los valores que causarían una operación imposible:
  • Denominadores: se excluyen los x que anulan el denominador.
  • Raíces pares: el radicando debe ser ≥ 0.
  • Logaritmos: el argumento debe ser > 0.
  • Si no hay restricciones (polinomios, rectas): dominio = R.
FunciónRestricciónDominio
f(x) = 3x − 5NingunaR = (−∞ ; +∞)
f(x) = 1 / (x − 2)x − 2 ≠ 0 → x ≠ 2R − {2}
f(x) = √(x + 3)x + 3 ≥ 0 → x ≥ −3[−3 ; +∞)
f(x) = √(4 − x2)4 − x2 ≥ 0 → −2 ≤ x ≤ 2[−2 ; 2]
f(x) = 1 / √(x − 1)x − 1 > 0 → x > 1(1 ; +∞)
📐 Ejemplo combinado: f(x) = √(x + 4) / (x − 1)
Condición 1 (raíz): x + 4 ≥ 0 → x ≥ −4
Condición 2 (denominador): x − 1 ≠ 0 → x ≠ 1
Dominio = [−4 ; +∞) − {1} = [−4 ; 1) ∪ (1 ; +∞)

5. Rango de una función

El rango (Ran f) es el conjunto de todos los valores que toma y = f(x) cuando x recorre el dominio. Es la "imagen" de la función. Para hallarlo se puede:
  1. Despejar x en función de y y analizar qué valores de y permiten que x exista en el dominio.
  2. Identificar la gráfica y proyectarla sobre el eje Y.
  3. Usar propiedades conocidas (un cuadrado nunca es negativo, etc.).
FunciónRangoRazonamiento
f(x) = 2x + 1RRecta: toma todos los valores reales
f(x) = x2[0 ; +∞)Un cuadrado nunca es negativo; mínimo en x = 0
f(x) = x2 + 3[3 ; +∞)x2 ≥ 0, entonces x2 + 3 ≥ 3
f(x) = √x[0 ; +∞)La raíz cuadrada siempre es ≥ 0
f(x) = −x2 + 4(−∞ ; 4]Parábola invertida; máximo en x = 0, y = 4

6. Dominio y rango desde la gráfica

Una estrategia visual muy útil en los exámenes de admisión:
  • Dominio: proyectar la gráfica sobre el eje X (ver qué valores de x están ocupados).
  • Rango: proyectar la gráfica sobre el eje Y (ver qué valores de y están ocupados).

7. Tipos de funciones según su comportamiento

Es importante reconocer estos tipos al determinar dominio y rango rápidamente:

Linealf(x) = mx + bRR
Cuadráticaf(x) = ax2 + bx + cR[k ; +∞) si a > 0  |  (−∞ ; k] si a < 0
Racionalf(x) = P(x)/Q(x)R − {ceros de Q}Depende de cada caso

⚠️ Truco para el rango de cuadráticas: el valor extremo (mínimo o máximo) ocurre en el vértice. Si f(x) = a(x − h)2 + k, entonces el vértice es (h, k) y el rango es [k ; +∞) si a > 0, o (−∞ ; k] si a < 0.
📐 f(x) = 2(x − 3)2 + 5: vértice en (3, 5), a = 2 > 0 → rango = [5 ; +∞)

Formulario

DEFINICIÓN Y NOTACIÓN

Función: relación donde a cada x le corresponde un único y
Notación: f : A → B  |  y = f(x)
Evaluación: sustituir x por el valor dado

DOMINIO — CÓMO ENCONTRARLO

Denominador: Q(x) ≠ 0
Raíz par: radicando ≥ 0
Raíz en denominador: radicando > 0
Polinomio sin restricciones: Dom = R

RANGO — CASOS FRECUENTES

f(x) = mx + b → Ran = R
f(x) = ax2 + … (a > 0) → Ran = [k ; +∞)  (k = valor en vértice)
f(x) = ax2 + … (a < 0) → Ran = (−∞ ; k]
f(x) = √g(x) → Ran ⊂ [0 ; +∞)

PRUEBA GRÁFICA

Línea vertical corta en 1 punto → es función
Línea vertical corta en 2+ puntos → no es función
Dom = proyección sobre eje X  |  Ran = proyección sobre eje Y

Problemas tipo admisión

NIVEL BÁSICO
1. Si f(x) = 4x − 3, ¿cuánto vale f(2)?
Ver resolución
Respuesta: a)
f(2) = 4(2) − 3 = 8 − 3 = 5.
2. ¿Cuál es el dominio de f(x) = 5x + 7?
Ver resolución
Respuesta: a)
Es una función lineal (polinomio de grado 1), sin restricciones. Dominio = R.
3. ¿Cuál de las siguientes relaciones NO es una función?
Ver resolución
Respuesta: b)
En b), el elemento x = 1 tiene dos imágenes distintas: y = 2 y y = 5. Eso viola la unicidad. No es función.
4. Si g(x) = x2 − 1, ¿cuánto vale g(0)?
Ver resolución
Respuesta: a)
g(0) = 02 − 1 = 0 − 1 = −1.
5. ¿Cuál es el rango de la función constante f(x) = 7?
Ver resolución
Respuesta: a)
Una función constante siempre entrega el mismo valor. El rango es el conjunto unitario {7}.
NIVEL INTERMEDIO
6. ¿Cuál es el dominio de f(x) = 1 / (x + 5)?
Ver resolución
Respuesta: a)
El denominador se anula cuando x + 5 = 0, es decir x = −5. Se excluye ese valor. Dominio = R − {−5}.
7. ¿Cuál es el dominio de f(x) = √(2x − 6)?
Ver resolución
Respuesta: a)
Condición: 2x − 6 ≥ 0 → 2x ≥ 6 → x ≥ 3. Dominio = [3 ; +∞).
8. ¿Cuál es el rango de f(x) = x2 + 4?
Ver resolución
Respuesta: a)
Como x2 ≥ 0 para todo x real, entonces x2 + 4 ≥ 4. El mínimo se alcanza en x = 0: f(0) = 4. Rango = [4 ; +∞).
9. Si f(x) = 2x − 1 y f(a) = 9, ¿cuánto vale a?
Ver resolución
Respuesta: a)
f(a) = 2a − 1 = 9 → 2a = 10 → a = 5. Verificación: f(5) = 10 − 1 = 9 ✓
10. La gráfica de x2 + y2 = 16, ¿representa una función?
Ver resolución
Respuesta: a)
La circunferencia, para ciertos valores de x (por ejemplo x = 0), produce dos valores de y: y = 4 e y = −4. No pasa la prueba de la línea vertical. No es función.
NIVEL AVANZADO
11. Halla el dominio de f(x) = √(x + 3) / (x − 1).
Ver resolución
Respuesta: a)
Condición 1 (raíz): x + 3 ≥ 0 → x ≥ −3, es decir x ∈ [−3 ; +∞)
Condición 2 (denominador): x − 1 ≠ 0 → x ≠ 1
Intersectando: [−3 ; +∞) − {1} = [−3 ; 1) ∪ (1 ; +∞)
12. Si f(x) = x2 − 6x + 5, ¿cuál es el valor mínimo del rango?
Ver resolución
Respuesta: a)
Completando cuadrados: f(x) = (x − 3)2 − 9 + 5 = (x − 3)2 − 4
El mínimo ocurre en x = 3: f(3) = 0 − 4 = −4. Rango = [−4 ; +∞)
13. ¿Cuál es el dominio de f(x) = 1 / √(9 − x2)?
Ver resolución
Respuesta: a)
La raíz está en el denominador, así que necesitamos 9 − x2 > 0 (estricto, no puede ser cero).
9 − x2 > 0 → x2 < 9 → −3 < x < 3. Dominio = (−3 ; 3)
14. Si f(x) = 3x + 1 y g(x) = x2, ¿cuánto vale f(g(2))?
Ver resolución
Respuesta: a)
Primero g(2) = 22 = 4.
Luego f(g(2)) = f(4) = 3(4) + 1 = 12 + 1 = 13.
(Esto se llama composición de funciones, un concepto del siguiente tema.)
15. ¿Cuál es el rango de f(x) = −2x2 + 8?
Ver resolución
Respuesta: a)
a = −2 < 0 → parábola con rama hacia abajo. El máximo está en el vértice: f(0) = 8.
La función decrece desde 8 hacia −∞. Rango = (−∞ ; 8]
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Si f(x) = (x2 − 9) / (x − 3), simplificando para x ≠ 3, ¿cuánto vale f(5)?
Ver resolución
Respuesta: a)
x2 − 9 = (x − 3)(x + 3) → f(x) = (x + 3) para x ≠ 3.
f(5) = 5 + 3 = 8.
17. Halla el dominio de f(x) = √(x − 2) + √(5 − x).
Ver resolución
Respuesta: a)
Condición 1: x − 2 ≥ 0 → x ≥ 2
Condición 2: 5 − x ≥ 0 → x ≤ 5
Ambas deben cumplirse simultáneamente: Dom = [2 ; 2] ∩ (−∞ ; 5] = [2 ; 5]
18. Se define f(x) = (2x + k) / (x − 3). Si f(5) = 7, ¿cuánto vale k?
Ver resolución
Respuesta: a)
f(5) = (2(5) + k) / (5 − 3) = (10 + k) / 2 = 7
10 + k = 14 → k = 4
19. ¿Cuántos elementos tiene el rango de la función f : {1, 2, 3, 4} → R definida por f(x) = x2 − x?
Ver resolución
Respuesta: a)
Evalúo: f(1) = 1 − 1 = 0 ; f(2) = 4 − 2 = 2 ; f(3) = 9 − 3 = 6 ; f(4) = 16 − 4 = 12
Rango = {0, 2, 6, 12} → 4 elementos (todos distintos).
20. Si f(x + 1) = 2x − 3, ¿cuánto vale f(4)?
Ver resolución
Respuesta: a)
Queremos f(4). Necesito que x + 1 = 4, es decir x = 3.
f(4) = f(3 + 1) = 2(3) − 3 = 6 − 3 = 3.
Método: se sustituye el valor de x que hace que el argumento sea igual al valor pedido.
