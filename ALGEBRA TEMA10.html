ÁLGEBRA · TEMA 10

MCD, MCM y Fracciones Algebraicas

MCD y MCM de polinomios · Fracción algebraica · Simplificación · Operaciones · Fracciones complejas

Explicación completa

1. Todo empieza por factorizar

Este tema es la continuación natural de la factorización (Tema 9). La idea central que debes interiorizar es:
⚠️ Antes de hallar un MCD, un MCM o simplificar una fracción, SIEMPRE factoriza primero el numerador y el denominador. Sin factorizar, casi nada se puede simplificar ni operar correctamente.

2. MCD y MCM de polinomios

Funcionan igual que con los números, pero usando factores en vez de números primos:
  • El MCD (Máximo Común Divisor) se forma con los factores comunes, elevados al menor exponente.
  • El MCM (Mínimo Común Múltiplo) se forma con los factores comunes y no comunes, elevados al mayor exponente.
📐 Ejemplo: sean P = (x − 1)2(x + 2) y Q = (x − 1)(x + 3).
• MCD = (x − 1)   (factor común, menor exponente)
• MCM = (x − 1)2(x + 2)(x + 3)   (todos los factores, mayor exponente)

3. Relación clave entre MCD y MCM

Para dos polinomios P y Q siempre se cumple:
MCD(P, Q) · MCM(P, Q) = P · Q
Esta fórmula resuelve muchos problemas de examen: si conoces tres de los cuatro valores, despejas el cuarto.

4. ¿Qué es una fracción algebraica?

Es un cociente de polinomios donde la variable aparece en el denominador, como (x + 1)/(x − 3).
⚠️ Valores prohibidos: una fracción algebraica está indefinida (no existe) cuando su denominador vale cero, porque dividir entre cero es imposible. Por ejemplo, 1/(x − 2) no existe cuando x = 2.

5. Simplificación de fracciones

Factorizamos numerador y denominador y cancelamos los factores idénticos.
📐 Ejemplo: (x2 − 9)/(x − 3) = [(x + 3)(x − 3)]/(x − 3) = x + 3. Se canceló el factor (x − 3).
⚠️ Error gravísimo: solo se cancelan factores (cosas multiplicando), nunca términos (cosas sumando). En (x + 2)/(x + 3) NO se puede cancelar la x, porque están sumando, no multiplicando.

6. Operaciones con fracciones

Suma y resta

Se necesita un denominador común (lo ideal: el MCM de los denominadores).
a/b ± c/d = (a·d ± c·b) / (b·d)

Multiplicación

Numerador por numerador, denominador por denominador (conviene simplificar antes).
(a/b) · (c/d) = (a·c) / (b·d)

División

Se multiplica por el inverso de la segunda fracción (se "voltea" y se multiplica).
(a/b) ÷ (c/d) = (a·d) / (b·c)
📐 Ejemplo de suma: 1/x + 1/y = (y + x)/(xy). El denominador común es xy.

7. Fracciones complejas

Son fracciones que tienen otras fracciones dentro (en el numerador, en el denominador o en ambos). Se resuelve cada parte por separado y luego se aplica la división.
📐 Ejemplo: (1 + 1/x) / (1 − 1/x2)
Numerador: (x + 1)/x. Denominador: (x2 − 1)/x2 = (x − 1)(x + 1)/x2.
Dividiendo: [(x + 1)/x] · [x2 / ((x − 1)(x + 1))] = x/(x − 1).

Formulario

MCD Y MCM

MCD: factores comunes, con el menor exponente
MCM: factores comunes y no comunes, con el mayor exponente
MCD(P,Q) · MCM(P,Q) = P · Q

FRACCIÓN ALGEBRAICA

Indefinida ⇔ denominador = 0
Solo se cancelan factores, nunca términos sumando

OPERACIONES

a/b ± c/d = (ad ± cb) / (bd)
(a/b) · (c/d) = ac / bd
(a/b) ÷ (c/d) = ad / bc

Problemas tipo admisión

NIVEL BÁSICO
1. Simplifica: (x2 − 9) / (x − 3)
Ver resolución
Respuesta: b)
Factorizo el numerador: x2 − 9 = (x + 3)(x − 3). Cancelo (x − 3) con el denominador y queda x + 3.
2. ¿Para qué valor de x la fracción 1/(x − 2) es indefinida?
Ver resolución
Respuesta: c)
La fracción no existe cuando el denominador es 0: x − 2 = 0 → x = 2.
3. Simplifica: 6x2 / (3x)
Ver resolución
Respuesta: a)
6/3 = 2 y x2/x = x. Resultado: 2x.
4. Halla el MCD de x2y   y   xy2.
Ver resolución
Respuesta: b)
El MCD toma los factores comunes con el menor exponente: de x (mín. exp. 1) y de y (mín. exp. 1) → xy.
5. Simplifica: (x2 − 1) / (x + 1)
Ver resolución
Respuesta: a)
x2 − 1 = (x + 1)(x − 1). Cancelo (x + 1) y queda x − 1.
NIVEL INTERMEDIO
6. Efectúa: 1/x + 1/y
Ver resolución
Respuesta: b)
Denominador común xy: 1/x = y/(xy), 1/y = x/(xy). Sumo: (y + x)/(xy) = (x + y)/(xy).
7. Simplifica: (x2 + 5x + 6) / (x2 − 4)
Ver resolución
Respuesta: a)
Numerador: x2 + 5x + 6 = (x + 2)(x + 3). Denominador: x2 − 4 = (x + 2)(x − 2). Cancelo (x + 2): (x + 3)/(x − 2).
8. Halla el MCM de (x − 1)   y   (x2 − 1).
Ver resolución
Respuesta: b)
x2 − 1 = (x − 1)(x + 1). El MCM toma todos los factores con mayor exponente: (x − 1)(x + 1) = x2 − 1.
9. Efectúa: [x/(x + 1)] · [(x + 1)/x2]
Ver resolución
Respuesta: a)
Multiplico: [x(x + 1)] / [(x + 1)x2]. Cancelo (x + 1) y una x: queda 1/x.
10. Halla el MCD de (x2 − 4)   y   (x2 + 4x + 4).
Ver resolución
Respuesta: b)
x2 − 4 = (x − 2)(x + 2); x2 + 4x + 4 = (x + 2)2. El factor común con menor exponente es (x + 2).
NIVEL AVANZADO
11. Efectúa: 1/(x − 1) + 1/(x + 1)
Ver resolución
Respuesta: a)
Denominador común (x − 1)(x + 1) = x2 − 1. Numerador: (x + 1) + (x − 1) = 2x. Resultado: 2x/(x2 − 1).
12. Simplifica: (x3 − 1) / (x2 − 1)
Ver resolución
Respuesta: a)
x3 − 1 = (x − 1)(x2 + x + 1); x2 − 1 = (x − 1)(x + 1). Cancelo (x − 1): (x2 + x + 1)/(x + 1).
13. Efectúa: [x/(x − 2)] − [2/(x − 2)]
Ver resolución
Respuesta: a)
Mismo denominador: (x − 2)/(x − 2) = 1 (siempre que x ≠ 2).
14. Si MCD(P, Q) = (x − 1) y MCM(P, Q) = (x − 1)(x + 2)(x − 3), y además P = (x − 1)(x + 2), halla Q.
Ver resolución
Respuesta: a)
Por la relación MCD · MCM = P · Q, despejamos Q = (MCD · MCM)/P.
Q = [(x − 1) · (x − 1)(x + 2)(x − 3)] / [(x − 1)(x + 2)] = (x − 1)(x − 3).
15. Simplifica la fracción compleja: (1 + 1/x) / (1 − 1/x2)
Ver resolución
Respuesta: a)
Numerador: 1 + 1/x = (x + 1)/x. Denominador: 1 − 1/x2 = (x2 − 1)/x2 = (x − 1)(x + 1)/x2.
Divido: [(x + 1)/x] · [x2/((x − 1)(x + 1))] = x/(x − 1).
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Si MCD(P, Q) = (x − 2) y MCM(P, Q) = (x − 2)(x + 2)(x + 3), halla el producto P · Q.
Ver resolución
Respuesta: a)
P · Q = MCD · MCM = (x − 2) · (x − 2)(x + 2)(x + 3) = (x − 2)2(x + 2)(x + 3).
17. Simplifica: (x4 − 1) / [(x2 + 1)(x − 1)]
Ver resolución
Respuesta: a)
x4 − 1 = (x2 + 1)(x2 − 1) = (x2 + 1)(x − 1)(x + 1). Cancelo (x2 + 1) y (x − 1): queda x + 1.
18. Efectúa: [1/(x − 2) + 1/(x + 2)] ÷ [1/(x2 − 4)]
Ver resolución
Respuesta: a)
Sumo dentro del corchete: [(x + 2) + (x − 2)] / (x2 − 4) = 2x/(x2 − 4).
Dividir entre 1/(x2 − 4) es multiplicar por (x2 − 4): [2x/(x2 − 4)] · (x2 − 4) = 2x.
