ÁLGEBRA · TEMA 16

Ecuaciones de Grado Superior

Teorema fundamental · Resolución por factorización · Raíces racionales y Ruffini · Bicuadradas · Cardano-Vieta

Explicación completa

1. ¿Qué es una ecuación de grado superior?

Es toda ecuación polinómica de grado 3 o mayor, de la forma:
anxn + an−1xn−1 + ... + a1x + a0 = 0
Por el Teorema Fundamental del Álgebra, una ecuación de grado n tiene exactamente n raíces (contando las repetidas y las complejas). Resolverla significa hallar todos los valores de x que la cumplen.
⚠️ Una ecuación de grado 3 tiene 3 raíces; una de grado 4 tiene 4; una de grado 5 tiene 5. Pueden ser reales o complejas, y algunas pueden repetirse (raíces múltiples).

2. Resolución por factorización

La estrategia base: llevar todo a un lado igualado a cero, factorizar y aplicar la regla del producto nulo (si un producto es 0, algún factor es 0).
📐 x3 − 4x = 0 → x(x2 − 4) = 0 → x(x − 2)(x + 2) = 0. Igualo cada factor: x = 0, x = 2, x = −2.
Aquí usas todo lo del Tema de Factorización: factor común, diferencia de cuadrados, suma y diferencia de cubos, aspa, etc.

3. Teorema de las raíces racionales

Cuando no factoriza a simple vista, este teorema te dice dónde buscar. Si una fracción p/q (irreducible) es raíz del polinomio, entonces:
  • p divide al término independiente a0.
  • q divide al coeficiente principal an.
📐 En x3 + 2x2 − 5x − 6 = 0, el coeficiente principal es 1, así que las posibles raíces enteras son los divisores de 6: ±1, ±2, ±3, ±6.

4. Divisores binómicos y Ruffini

Una vez que encuentras una raíz "a" (probando los candidatos), divides el polinomio entre (x − a) usando Ruffini. El cociente baja un grado y sigues resolviendo.
📐 Para x3 − 6x2 + 11x − 6 = 0: pruebo x = 1 → 1 − 6 + 11 − 6 = 0 ✓. Divido por (x − 1) y queda x2 − 5x + 6 = (x − 2)(x − 3). Raíces: 1, 2, 3.

5. Ecuaciones bicuadradas

Tienen la forma ax4 + bx2 + c = 0 (solo potencias pares). Se resuelven con el cambio de variable y = x2, que la convierte en una cuadrática.
📐 x4 − 5x2 + 4 = 0. Hago y = x2: y2 − 5y + 4 = 0 → (y − 1)(y − 4) = 0 → y = 1, y = 4. Vuelvo: x2 = 1 → x = ±1 ; x2 = 4 → x = ±2. Las cuatro raíces: ±1, ±2.

6. Relaciones de Cardano-Vieta

Generalizan las fórmulas de Vieta de la cuadrática. Para una ecuación de grado n con coeficiente principal an:
Suma de raíces = − an−1 / an
Producto de raíces = (−1)n · a0 / an
📐 En x3 − 6x2 + 11x − 6 = 0: suma de raíces = −(−6)/1 = 6 ; producto = (−1)3(−6)/1 = 6. (En efecto 1+2+3 = 6 y 1·2·3 = 6.)

Problemas tipo admisión

NIVEL BÁSICO
1. ¿Cuántas raíces tiene la ecuación x3 − x = 0?
Ver resolución
Respuesta: c)
Es de grado 3, así que tiene 3 raíces (de hecho x = 0, 1, −1).
2. ¿Cuál es la raíz real de x3 = 8?
Ver resolución
Respuesta: a)
x = ³√8 = 2 (porque 23 = 8).
3. Resuelve: x3 − 4x = 0
Ver resolución
Respuesta: a)
x(x2 − 4) = x(x − 2)(x + 2) = 0 → x = 0, 2, −2.
4. Resuelve en R: x4 = 16
Ver resolución
Respuesta: a)
x2 = ±4. Solo x2 = 4 da reales: x = ±2. (x2 = −4 no es real.) Raíces reales: x = 2 y x = −2.
5. ¿Cuántas raíces tiene como máximo una ecuación de grado 5?
Ver resolución
Respuesta: b)
El grado indica la cantidad de raíces: una de grado 5 tiene 5 raíces.
NIVEL INTERMEDIO
6. Resuelve: x3 − x2 − 6x = 0
Ver resolución
Respuesta: a)
x(x2 − x − 6) = x(x − 3)(x + 2) = 0 → x = 0, 3, −2.
7. Resuelve la bicuadrada: x4 − 5x2 + 4 = 0
Ver resolución
Respuesta: a)
y = x2: y2 − 5y + 4 = 0 → (y − 1)(y − 4) = 0 → y = 1, 4. Vuelvo: x = ±1 y x = ±2 → ±1, ±2.
8. Si x = 1 es raíz de x3 − 6x2 + 11x − 6 = 0, halla las otras dos raíces.
Ver resolución
Respuesta: a)
Divido por (x − 1): queda x2 − 5x + 6 = (x − 2)(x − 3). Las otras raíces son 2 y 3.
9. Halla la suma de las raíces de x3 − 6x2 + 11x − 6 = 0.
Ver resolución
Respuesta: a)
Suma = −an−1/an = −(−6)/1 = 6.
10. Halla el producto de las raíces de x3 − 6x2 + 11x − 6 = 0.
Ver resolución
Respuesta: a)
Producto = (−1)3·a0/an = (−1)(−6)/1 = 6. (En efecto 1·2·3 = 6.)
NIVEL AVANZADO
11. Resuelve: x3 − 7x + 6 = 0
Ver resolución
Respuesta: a)
Pruebo x = 1: 1 − 7 + 6 = 0 ✓. Divido por (x − 1): x2 + x − 6 = (x + 3)(x − 2). Raíces: 1, 2, −3.
12. Resuelve la bicuadrada: x4 − 13x2 + 36 = 0
Ver resolución
Respuesta: a)
y = x2: y2 − 13y + 36 = (y − 4)(y − 9) = 0 → y = 4, 9 → x = ±2 y x = ±3 → ±2, ±3.
13. Una raíz entera positiva de x3 + 2x2 − 5x − 6 = 0 es:
Ver resolución
Respuesta: a)
Las posibles raíces son divisores de 6. Pruebo x = 2: 8 + 8 − 10 − 6 = 0 ✓. La raíz entera positiva es 2. (Las otras: −1 y −3.)
14. Halla la suma de las raíces de 2x3 − 6x2 + x − 3 = 0.
Ver resolución
Respuesta: a)
Suma = −an−1/an = −(−6)/2 = 6/2 = 3.
15. Resuelve en R: x4 − 16 = 0
Ver resolución
Respuesta: a)
x4 − 16 = (x2 − 4)(x2 + 4) = 0. x2 = 4 → x = ±2; x2 = −4 no es real. Raíces reales: x = 2 y x = −2.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Halla la suma de las raíces de x3 − 3x2 − x + 3 = 0.
Ver resolución
Respuesta: a)
Por Cardano-Vieta, suma = −(−3)/1 = 3. (Factorizando: (x − 3)(x − 1)(x + 1), raíces 3, 1, −1, que suman 3.)
17. Las raíces de x3 − 6x2 + 11x − 6 = 0 son 1, 2 y 3. Halla 1/x1 + 1/x2 + 1/x3.
Ver resolución
Respuesta: a)
1/1 + 1/2 + 1/3 = (6 + 3 + 2)/6 = 11/6. (Equivale a "suma de productos de pares / producto de raíces" = 11/6.)
18. Halla "k" para que x = 2 sea raíz de x3 − kx + 2 = 0.
Ver resolución
Respuesta: a)
Reemplazo x = 2: 23 − 2k + 2 = 0 → 8 − 2k + 2 = 0 → 10 = 2k → k = 5.
19. Halla la suma de todas las raíces reales de x4 − 5x2 + 4 = 0.
Ver resolución
Respuesta: a)
Las raíces son 1, −1, 2, −2. Su suma: 1 − 1 + 2 − 2 = 0 (en toda bicuadrada las raíces vienen en pares opuestos).
20. En x3 − x2 − x + 1 = 0, una raíz se repite (raíz doble). Halla la raíz que NO se repite.
Ver resolución
Respuesta: a)
Agrupo: x2(x − 1) − (x − 1) = (x − 1)(x2 − 1) = (x − 1)(x − 1)(x + 1) = (x − 1)2(x + 1). La raíz doble es 1 y la que no se repite es −1.
