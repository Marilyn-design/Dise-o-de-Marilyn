ÁLGEBRA · TEMA 12

Ecuaciones de Segundo Grado

Forma general · Factorización · Fórmula general · Discriminante · Suma y producto de raíces

Explicación completa

1. ¿Qué es una ecuación cuadrática?

Es una ecuación en la que la incógnita aparece elevada al cuadrado como mayor potencia. Su forma general es:
ax2 + bx + c = 0   (con a ≠ 0)
Donde a es el coeficiente cuadrático, b el lineal y c el término independiente. A diferencia de las lineales, una cuadrática tiene hasta dos soluciones (llamadas raíces).
⚠️ El a nunca puede ser 0: si a = 0, desaparece el x2 y la ecuación deja de ser cuadrática (se vuelve lineal). Por eso siempre se exige a ≠ 0.

2. Método 1: por factorización

Es el más rápido cuando la ecuación factoriza fácil. Se basa en una propiedad clave: si un producto es cero, al menos uno de sus factores es cero.

Pasos:

  1. Lleva todo a un lado para que quede igualada a cero.
  2. Factoriza (con lo que aprendiste en el Tema 9, normalmente aspa simple).
  3. Iguala cada factor a cero y despeja.
📐 Ejemplo: x2 − 5x + 6 = 0
• Factorizo: (x − 2)(x − 3) = 0
• Igualo cada factor a 0: x − 2 = 0 → x = 2 ; x − 3 = 0 → x = 3
• Soluciones: x = 2 y x = 3

3. Método 2: la fórmula general

Funciona siempre, incluso cuando no se puede factorizar con números enteros. Es la fórmula más importante del tema:
x = [ −b ± √(b2 − 4ac) ] / (2a)
El ± indica que hay dos soluciones: una con + y otra con −. Solo necesitas identificar a, b y c, y reemplazar con cuidado de los signos.
📐 Ejemplo: 2x2 + 3x − 2 = 0. Aquí a = 2, b = 3, c = −2.
• Discriminante: b2 − 4ac = 9 − 4(2)(−2) = 9 + 16 = 25.
• x = (−3 ± √25)/(2·2) = (−3 ± 5)/4.
• x1 = (−3 + 5)/4 = 1/2 ; x2 = (−3 − 5)/4 = −2.

4. El discriminante: ¿cuántas soluciones hay?

La cantidad bajo la raíz se llama discriminante y se simboliza con la letra griega delta (Δ):
Δ = b2 − 4ac
Sin resolver toda la ecuación, su signo te dice qué tipo de raíces hay:
Si el discriminante Δ es...Las raíces son...
Δ > 0 (positivo)Dos raíces reales y diferentes
Δ = 0 (cero)Una raíz real doble (raíces iguales)
Δ < 0 (negativo)Dos raíces complejas (no reales)
⚠️ Pregunta clásica de admisión: "halla m para que la ecuación tenga raíces iguales" → significa imponer Δ = 0 y resolver para m.

5. Suma y producto de raíces (Teorema de Vieta)

Sin resolver la ecuación, puedes conocer la suma y el producto de sus raíces directamente desde los coeficientes:
Suma:   x1 + x2 = −b / a
Producto:   x1 · x2 = c / a
📐 Ejemplo: en x2 − 7x + 12 = 0 (a=1, b=−7, c=12): la suma de raíces es −(−7)/1 = 7 y el producto es 12/1 = 12. (En efecto las raíces son 3 y 4.)

6. Reconstruir una ecuación a partir de sus raíces

El proceso inverso: si conoces la suma (S) y el producto (P) de las raíces, la ecuación es:
x2 − S·x + P = 0
📐 Ejemplo: raíces 2 y 5 → S = 7, P = 10 → ecuación: x2 − 7x + 10 = 0.

Formulario

FORMA GENERAL Y FÓRMULA

ax2 + bx + c = 0   (a ≠ 0)
x = [ −b ± √(b2 − 4ac) ] / (2a)

DISCRIMINANTE (Δ = b2 − 4ac)

Δ > 0 → dos raíces reales distintas
Δ = 0 → una raíz real doble
Δ < 0 → dos raíces complejas

RAÍCES (TEOREMA DE VIETA)

Suma: x1 + x2 = −b/a
Producto: x1 · x2 = c/a
Reconstruir: x2 − S·x + P = 0

Problemas tipo admisión

NIVEL BÁSICO
1. Resuelve: x2 − 9 = 0
Ver resolución
Respuesta: b)
x2 = 9 → x = ±√9 → x = 3 y x = −3. (También sale factorizando: (x−3)(x+3)=0.)
2. Resuelve: (x − 4)(x + 1) = 0
Ver resolución
Respuesta: a)
Igualo cada factor a cero: x − 4 = 0 → x = 4; x + 1 = 0 → x = −1. Soluciones: x = 4 y x = −1.
3. En la ecuación 3x2 − 5x + 2 = 0, ¿cuánto vale el coeficiente b?
Ver resolución
Respuesta: b)
En ax2 + bx + c, el coeficiente b es el que acompaña a la x con su signo: b = −5.
4. Resuelve por factorización: x2 − 5x + 6 = 0
Ver resolución
Respuesta: a)
Factorizo: (x − 2)(x − 3) = 0 → x = 2 y x = 3.
5. ¿Cuál es la suma de las raíces de x2 − 7x + 12 = 0?
Ver resolución
Respuesta: b)
Suma = −b/a = −(−7)/1 = 7.
NIVEL INTERMEDIO
6. Resuelve: x2 + 2x − 8 = 0
Ver resolución
Respuesta: a)
Factorizo: dos números que multipliquen −8 y sumen +2 → +4 y −2. (x + 4)(x − 2) = 0 → x = −4 y x = 2.
7. Halla el producto de las raíces de 2x2 − 6x + 10 = 0.
Ver resolución
Respuesta: b)
Producto = c/a = 10/2 = 5. (No hace falta resolver la ecuación.)
8. Usa la fórmula general para resolver: x2 − 4x + 3 = 0
Ver resolución
Respuesta: a)
a=1, b=−4, c=3. Δ = 16 − 12 = 4. x = (4 ± √4)/2 = (4 ± 2)/2 → x = 3 o x = 1. Soluciones: x = 1 y x = 3.
9. ¿Qué tipo de raíces tiene x2 + 2x + 5 = 0?
Ver resolución
Respuesta: c)
Δ = b2 − 4ac = 4 − 4(1)(5) = 4 − 20 = −16 < 0. Como el discriminante es negativo, hay dos raíces complejas.
10. Reconstruye la ecuación cuadrática cuyas raíces son 2 y 5.
Ver resolución
Respuesta: a)
S = 2 + 5 = 7, P = 2·5 = 10. Ecuación: x2 − Sx + P = 0 → x2 − 7x + 10 = 0.
NIVEL AVANZADO
11. Halla el valor de "k" para que x2 − 6x + k = 0 tenga raíces iguales.
Ver resolución
Respuesta: a)
Raíces iguales → Δ = 0. b2 − 4ac = (−6)2 − 4(1)(k) = 36 − 4k = 0 → 4k = 36 → k = 9.
12. Las raíces de x2 − 8x + 12 = 0 son x1 y x2. Halla x12 + x22.
Ver resolución
Respuesta: a)
Suma S = 8, producto P = 12. Uso la identidad del Tema 6: x12 + x22 = S2 − 2P = 64 − 24 = 40.
13. "El cuadrado de un número menos su triple es igual a 10." Halla el número positivo.
Ver resolución
Respuesta: a)
Traduzco: x2 − 3x = 10 → x2 − 3x − 10 = 0 → (x − 5)(x + 2) = 0 → x = 5 o x = −2. El positivo es 5.
14. Si una raíz de x2 − 7x + c = 0 es 3, halla la otra raíz.
Ver resolución
Respuesta: a)
La suma de raíces es −b/a = 7. Si una raíz es 3, la otra es 7 − 3 = 4.
15. Resuelve: 2x2 − 7x + 3 = 0
Ver resolución
Respuesta: a)
a=2, b=−7, c=3. Δ = 49 − 24 = 25. x = (7 ± 5)/4 → x = 12/4 = 3 o x = 2/4 = 1/2. Soluciones: x = 3 y x = 1/2.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Las raíces de x2 − 5x + 6 = 0 son x1 y x2. Halla el valor de 1/x1 + 1/x2.
Ver resolución
Respuesta: a)
1/x1 + 1/x2 = (x1 + x2)/(x1·x2) = S/P. Aquí S = 5 y P = 6, así que el resultado es 5/6.
17. Halla "m" para que la suma de las raíces de x2 − (m + 1)x + 12 = 0 sea igual a su producto.
Ver resolución
Respuesta: a)
Suma = −b/a = (m + 1). Producto = c/a = 12. Igualo suma = producto: m + 1 = 12 → m = 11.
18. Halla el valor de "n" para que x2 + nx + 16 = 0 tenga una única solución (raíz doble), con n positivo.
Ver resolución
Respuesta: b)
Raíz doble → Δ = 0. n2 − 4(1)(16) = 0 → n2 = 64 → n = ±8. Como n debe ser positivo, n = 8.
