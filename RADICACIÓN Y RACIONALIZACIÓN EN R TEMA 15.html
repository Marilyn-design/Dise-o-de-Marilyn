ÁLGEBRA · TEMA 15

Radicación y Racionalización en R

Leyes de radicales · Simplificación · Operaciones · Racionalización de denominadores · Factor racionalizante

Explicación completa

1. Radicación en los reales

La raíz n-ésima de un número a es el número que, elevado al índice n, devuelve a. Lo más importante es recordar su forma como exponente fraccionario (Tema 2):
n√(am) = am/n
⚠️ En R hay una restricción: una raíz de índice par de un número negativo no existe en los reales (por ejemplo √(−4) no es real). Las de índice impar sí (³√(−8) = −2).

2. Leyes de los radicales

n√(ab) = n√a · n√b  ;  n√(a/b) = n√a / n√b
m√(n√a) = m·n√a

3. Simplificar radicales (extraer factores)

Para simplificar una raíz, se descompone el radicando y se sacan los factores que tengan exponente igual al índice.
📐 √8 = √(4 · 2) = √4 · √2 = 2√2.   √12 = √(4 · 3) = 2√3.

4. Suma, resta y producto de radicales

Solo se pueden sumar o restar radicales semejantes (mismo índice y mismo radicando). Para multiplicar o dividir, se opera dentro de la raíz.
📐 2√3 + 5√3 = 7√3.   √18 + √8 = 3√2 + 2√2 = 5√2.   √2 · √8 = √16 = 4.

5. Racionalización

Racionalizar es eliminar las raíces del denominador de una fracción, multiplicando arriba y abajo por un factor racionalizante (FR) conveniente. Hay tres casos clásicos:
CasoSe multiplica porEjemplo
Denominador √a√a1/√2 = √2/2
Denominador n√(am)n√(an−m)1/³√2 = ³√4 / 2
Denominador a + √b (o √a ± √b)la conjugada (a − √b)usa (a+√b)(a−√b)=a2−b
⚠️ La conjugada es la estrella del tema: para un denominador con dos términos, multiplicas por el mismo binomio con el signo del medio cambiado. Así aparece una diferencia de cuadrados y desaparecen las raíces.
📐 1/(√3 − 1) = (√3 + 1) / [(√3 − 1)(√3 + 1)] = (√3 + 1)/(3 − 1) = (√3 + 1)/2.

Formulario

LEYES DE RADICALES

n√(am) = am/n
n√(ab) = n√a · n√b
m√(n√a) = mn√a

OPERACIONES

Solo radicales semejantes se suman/restan
c·n√a ± d·n√a = (c ± d)n√a
n√a · n√b = n√(ab)

RACIONALIZACIÓN

1/√a → × √a → √a / a
1/n√(am) → × n√(an−m)
Binomio → × conjugada: (a+√b)(a−√b) = a2−b

3. Reduce: 2√3 + 5√3
Ver resolución
Respuesta: a)
Son radicales semejantes; sumo coeficientes: (2 + 5)√3 = 7√3.
4. Racionaliza: 1/√2
Ver resolución
Respuesta: a)
Multiplico arriba y abajo por √2: (1·√2)/(√2·√2) = √2/2.
5. Calcula: √2 · √8
Ver resolución
Respuesta: a)
√2 · √8 = √16 = 4.
NIVEL INTERMEDIO
6. Simplifica: √50
Ver resolución
Respuesta: a)
√50 = √(25·2) = 5√2 = 5√2.
7. Calcula: 3√2 · 2√3
Ver resolución
Respuesta: a)
Multiplico coeficientes y radicandos por separado: (3·2)·√(2·3) = 6√6.
8. Racionaliza: 3/√3
Ver resolución
Respuesta: a)
3/√3 = (3·√3)/(√3·√3) = 3√3/3 = √3.
9. Reduce: √18 + √8
Ver resolución
Respuesta: a)
√18 = 3√2 y √8 = 2√2. Suma: 3√2 + 2√2 = 5√2.
10. Calcula: (√5)2 + √9
Ver resolución
Respuesta: a)
(√5)2 = 5 y √9 = 3. Suma: 5 + 3 = 8.
NIVEL AVANZADO
11. Racionaliza: 1/(√3 − 1)
Ver resolución
Respuesta: a)
Multiplico por la conjugada (√3 + 1): (√3 + 1)/[(√3)2 − 12] = (√3 + 1)/(3 − 1) = (√3 + 1)/2.
12. Racionaliza: 2/(√5 + √3)
Ver resolución
Respuesta: a)
Multiplico por (√5 − √3): 2(√5 − √3)/(5 − 3) = 2(√5 − √3)/2 = √5 − √3.
13. Reduce: √75 − √27
Ver resolución
Respuesta: a)
√75 = 5√3 y √27 = 3√3. Resta: 5√3 − 3√3 = 2√3.
14. Calcula: (2 + √3)(2 − √3)
Ver resolución
Respuesta: a)
Diferencia de cuadrados: 22 − (√3)2 = 4 − 3 = 1.
15. Racionaliza: 1/³√2
Ver resolución
Respuesta: a)
Multiplico por ³√(22) = ³√4 para completar el cubo: (³√4)/(³√2 · ³√4) = ³√4/³√8 = ³√4/2.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Racionaliza: 1/(√x + √y)
Ver resolución
Respuesta: a)
Multiplico por la conjugada (√x − √y): (√x − √y)/[(√x)2 − (√y)2] = (√x − √y)/(x − y).
17. Racionaliza: 6/(√7 − 1)
Ver resolución
Respuesta: a)
Multiplico por (√7 + 1): 6(√7 + 1)/[(√7)2 − 1] = 6(√7 + 1)/(7 − 1) = 6(√7 + 1)/6 = √7 + 1.
18. ¿Cuál es el factor racionalizante (FR) de ³√a?
Ver resolución
Respuesta: a)
Se necesita completar el cubo: ³√a · ³√(a2) = ³√(a3) = a. Por eso el FR es ³√(a2).
19. Reduce: √8 + √18 − √2
Ver resolución
Respuesta: a)
√8 = 2√2, √18 = 3√2, √2 = 1√2. Entonces 2√2 + 3√2 − √2 = (2 + 3 − 1)√2 = 4√2.
20. Racionaliza: 1/(√5 + 2)
Ver resolución
Respuesta: a)
Multiplico por la conjugada (√5 − 2): (√5 − 2)/[(√5)2 − 22] = (√5 − 2)/(5 − 4) = (√5 − 2)/1 = √5 − 2.
