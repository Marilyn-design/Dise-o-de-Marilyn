ÁLGEBRA · TEMA 6

Productos Notables

Binomio al cuadrado y al cubo · Diferencia de cuadrados · Legendre · Suma y diferencia de cubos · Identidades

Explicación completa

1. ¿Qué son los productos notables?

Son multiplicaciones que aparecen tan seguido que conviene memorizar su resultado directo, sin tener que desarrollar paso a paso. Te ahorran muchísimo tiempo en el examen y son la base de la factorización (Tema 9).

2. Binomio al cuadrado (trinomio cuadrado perfecto)

(a + b)2 = a2 + 2ab + b2
(a − b)2 = a2 − 2ab + b2
📐 (x + 3)2 = x2 + 2(x)(3) + 32 = x2 + 6x + 9.
⚠️ Error más común: (a + b)2 NO es a2 + b2. Siempre aparece el doble producto 2ab en el medio. ¡No te lo comas!

3. Diferencia de cuadrados

La suma por la diferencia de los mismos términos da la diferencia de sus cuadrados.
(a + b)(a − b) = a2 − b2
📐 (x + 5)(x − 5) = x2 − 25.

4. Identidades de Legendre

Relacionan la suma y la resta de los binomios al cuadrado:
(a + b)2 + (a − b)2 = 2(a2 + b2)
(a + b)2 − (a − b)2 = 4ab

5. Binomio al cubo

(a + b)3 = a3 + 3a2b + 3ab2 + b3
(a − b)3 = a3 − 3a2b + 3ab2 − b3
📐 (x + 1)3 = x3 + 3x2 + 3x + 1.

6. Suma y diferencia de cubos

a3 + b3 = (a + b)(a2 − ab + b2)
a3 − b3 = (a − b)(a2 + ab + b2)

7. Producto de binomios con término común (Stevin) y Argand

(x + a)(x + b) = x2 + (a + b)x + ab
(x2 + x + 1)(x2 − x + 1) = x4 + x2 + 1   (identidad de Argand)
📐 (x + 4)(x + 2) = x2 + (4 + 2)x + (4)(2) = x2 + 6x + 8.

Formulario

CUADRADOS Y DIFERENCIA

(a ± b)2 = a2 ± 2ab + b2
(a + b)(a − b) = a2 − b2
(a+b)2 − (a−b)2 = 4ab

CUBOS

(a ± b)3 = a3 ± 3a2b + 3ab2 ± b3
a3 + b3 = (a + b)(a2 − ab + b2)
a3 − b3 = (a − b)(a2 + ab + b2)

TÉRMINO COMÚN E IDENTIDADES ÚTILES

(x + a)(x + b) = x2 + (a+b)x + ab
a2 + b2 = (a + b)2 − 2ab

Problemas tipo admisión

NIVEL BÁSICO
1. Desarrolla: (x + 3)2
Ver resolución
Respuesta: b)
(x + 3)2 = x2 + 2(x)(3) + 32 = x2 + 6x + 9.
2. Desarrolla: (x − 2)2
Ver resolución
Respuesta: a)
(x − 2)2 = x2 − 2(x)(2) + 22 = x2 − 4x + 4.
3. Efectúa: (x + 5)(x − 5)
Ver resolución
Respuesta: b)
Suma por diferencia: (x + 5)(x − 5) = x2 − 52 = x2 − 25.
4. Desarrolla: (x + 1)3
Ver resolución
Respuesta: b)
(x + 1)3 = x3 + 3x2(1) + 3x(1)2 + 13 = x3 + 3x2 + 3x + 1.
5. ¿Cuál es el desarrollo de (a + b)2?
Ver resolución
Respuesta: b)
Binomio al cuadrado: a2 + 2ab + b2 (no olvides el doble producto).
NIVEL INTERMEDIO
6. Desarrolla: (2x + 3)2
Ver resolución
Respuesta: a)
(2x + 3)2 = (2x)2 + 2(2x)(3) + 32 = 4x2 + 12x + 9.
7. Efectúa: (3x − 1)(3x + 1)
Ver resolución
Respuesta: a)
Diferencia de cuadrados: (3x)2 − 12 = 9x2 − 1.
8. Efectúa: (x + 4)(x + 2)
Ver resolución
Respuesta: a)
Término común: x2 + (4 + 2)x + (4)(2) = x2 + 6x + 8.
9. Desarrolla: (x − 3)3
Ver resolución
Respuesta: a)
(x − 3)3 = x3 − 3x2(3) + 3x(3)2 − 33 = x3 − 9x2 + 27x − 27.
10. Si a + b = 5 y ab = 6, halla a2 + b2.
Ver resolución
Respuesta: a)
a2 + b2 = (a + b)2 − 2ab = 52 − 2(6) = 25 − 12 = 13.
NIVEL AVANZADO
11. Simplifica: (x + 2)2 − (x − 2)2
Ver resolución
Respuesta: a)
Por Legendre, (a + b)2 − (a − b)2 = 4ab. Con a = x, b = 2: 4(x)(2) = 8x.
12. Si x + 1/x = 3, halla x2 + 1/x2.
Ver resolución
Respuesta: a)
Elevo al cuadrado: (x + 1/x)2 = x2 + 2 + 1/x2 = 9. Entonces x2 + 1/x2 = 9 − 2 = 7.
13. Efectúa: (x2 + x + 1)(x2 − x + 1)
Ver resolución
Respuesta: a)
Identidad de Argand: agrupo como [(x2 + 1) + x][(x2 + 1) − x] = (x2 + 1)2 − x2 = x4 + 2x2 + 1 − x2 = x4 + x2 + 1.
14. Simplifica: (a + b)2 − (a2 + b2)
Ver resolución
Respuesta: a)
(a + b)2 = a2 + 2ab + b2. Le resto (a2 + b2) y queda 2ab.
15. Simplifica: (x + 3)(x − 3) + 9
Ver resolución
Respuesta: a)
(x + 3)(x − 3) = x2 − 9. Sumo 9: x2 − 9 + 9 = x2.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Si a + b = 4 y a2 + b2 = 10, halla ab.
Ver resolución
Respuesta: a)
(a + b)2 = a2 + 2ab + b2 → 16 = 10 + 2ab → 2ab = 6 → ab = 3.
17. Simplifica: (x + 1)3 − (x − 1)3
Ver resolución
Respuesta: a)
(x + 1)3 = x3 + 3x2 + 3x + 1 ; (x − 1)3 = x3 − 3x2 + 3x − 1. Al restar se cancelan x3 y 3x: queda 6x2 + 2 = 6x2 + 2.
18. Si x − 1/x = 2, halla x2 + 1/x2.
Ver resolución
Respuesta: a)
Elevo al cuadrado: (x − 1/x)2 = x2 − 2 + 1/x2 = 4. Entonces x2 + 1/x2 = 4 + 2 = 6.
