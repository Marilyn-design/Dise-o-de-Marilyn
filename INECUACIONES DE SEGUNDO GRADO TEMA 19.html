ÁLGEBRA · TEMA 19

Inecuaciones de Segundo Grado

Puntos críticos · Regla de signos · Análisis con la parábola · Casos del discriminante

Explicación completa

1. ¿Qué es una inecuación cuadrática?

Es una desigualdad en la que la incógnita aparece al cuadrado como mayor potencia:
ax2 + bx + c > 0   (o con <, ≥, ≤)   con a ≠ 0
A diferencia de la ecuación, aquí no buscamos puntos sueltos sino intervalos donde el trinomio cumple la condición de signo.

2. Método de los puntos críticos

Es el método estrella. Tres pasos:
  1. Llevar todo a un lado para que quede comparado con cero y factorizar (o hallar las raíces con la fórmula del Tema de cuadráticas).
  2. Ubicar las raíces (puntos críticos) en la recta numérica; estas dividen la recta en zonas.
  3. Evaluar el signo en cada zona y elegir las que cumplen la desigualdad.
📐 x2 − 5x + 6 > 0 → (x − 2)(x − 3) > 0. Puntos críticos 2 y 3. El producto es positivo "por fuera" de las raíces: x < 2 o x > 3.

3. Regla práctica (cuando a > 0)

Si el coeficiente principal es positivo y ya está factorizado como (x − r1)(x − r2), con r1 < r2:
DesigualdadSoluciónFrase clave
> 0  (o ≥ 0)x < r1 o x > r2"por fuera" de las raíces
< 0  (o ≤ 0)r1 < x < r2"por dentro" de las raíces
⚠️ Esta regla supone a > 0. Si a < 0, primero multiplica toda la inecuación por (−1) — recordando invertir el signo — para dejar el coeficiente positivo.

4. Interpretación con la parábola

El trinomio ax2 + bx + c es una parábola. Si a > 0 abre hacia arriba: es positiva por fuera de las raíces y negativa entre ellas. Esto es justo lo que dice la tabla anterior.

5. Casos especiales del discriminante

Cuando el trinomio no tiene raíces reales (Δ = b2 − 4ac < 0), no cambia de signo nunca:
Si Δ < 0 y...El trinomio es...Ejemplo de solución
a > 0siempre positivo"> 0" → todo R ; "< 0" → vacío
a < 0siempre negativo"< 0" → todo R ; "> 0" → vacío
📐 x2 + x + 1 > 0: como Δ = 1 − 4 = −3 < 0 y a > 0, el trinomio es siempre positivo → la solución es todo R.

Problemas tipo admisión

NIVEL BÁSICO
1. Resuelve: (x − 2)(x − 3) > 0
Ver resolución
Respuesta: a)
Producto positivo → "por fuera" de las raíces 2 y 3: x < 2 o x > 3.
2. Resuelve: (x − 1)(x + 4) < 0
Ver resolución
Respuesta: a)
Producto negativo → "por dentro" de las raíces −4 y 1: −4 < x < 1.
3. ¿Cuáles son los puntos críticos de x2 − 5x + 6 = 0?
Ver resolución
Respuesta: a)
(x − 2)(x − 3) = 0 → puntos críticos 2 y 3.
4. Para a > 0, la inecuación (x − r1)(x − r2) < 0 tiene como solución...
Ver resolución
Respuesta: a)
Producto negativo (con a > 0) → solución "por dentro": entre las raíces.
5. Resuelve: x2 − 9 < 0
Ver resolución
Respuesta: a)
x2 − 9 = (x − 3)(x + 3) < 0 → "por dentro" de −3 y 3: −3 < x < 3.
NIVEL INTERMEDIO
6. Resuelve: x2 − 5x + 6 > 0
Ver resolución
Respuesta: a)
(x − 2)(x − 3) > 0 → por fuera: ⟨−∞ ; 2⟩ ∪ ⟨3 ; +∞⟩.
7. Resuelve: x2 − x − 6 ≤ 0
Ver resolución
Respuesta: a)
(x − 3)(x + 2) ≤ 0 → por dentro, con extremos incluidos: [−2 ; 3].
8. Resuelve: x2 + 2x − 8 > 0
Ver resolución
Respuesta: a)
(x + 4)(x − 2) > 0 → por fuera de −4 y 2: x < −4 o x > 2.
9. Resuelve: x2 ≤ 4
Ver resolución
Respuesta: a)
x2 − 4 ≤ 0 → (x − 2)(x + 2) ≤ 0 → −2 ≤ x ≤ 2.
10. Resuelve: x2 − 4x ≥ 0
Ver resolución
Respuesta: a)
x(x − 4) ≥ 0 → por fuera de 0 y 4 (incluidos): x ≤ 0 o x ≥ 4.
NIVEL AVANZADO
11. Resuelve: 2x2 − 7x + 3 < 0
Ver resolución
Respuesta: a)
2x2 − 7x + 3 = (2x − 1)(x − 3). Raíces 1/2 y 3. Como es "< 0" (a > 0), va por dentro: 1/2 < x < 3.
12. Resuelve: x2 − 6x + 9 > 0
Ver resolución
Respuesta: a)
Es (x − 3)2 > 0. Un cuadrado es positivo siempre, salvo cuando vale 0 (en x = 3). Solución: todo R excepto x = 3.
13. Resuelve: x2 + x + 1 > 0
Ver resolución
Respuesta: a)
Δ = 1 − 4 = −3 < 0 y a > 0 → el trinomio es siempre positivo. Solución: todo R.
14. Resuelve: −x2 + 4 > 0
Ver resolución
Respuesta: a)
Multiplico por −1 (invierto): x2 − 4 < 0 → (x − 2)(x + 2) < 0 → −2 < x < 2.
15. Resuelve: x2 − 2x − 3 ≥ 0
Ver resolución
Respuesta: a)
(x − 3)(x + 1) ≥ 0 → por fuera de −1 y 3 (incluidos): x ≤ −1 o x ≥ 3.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Resuelve: (x − 1)2 ≤ 0
Ver resolución
Respuesta: a)
Un cuadrado nunca es negativo; solo puede ser ≤ 0 cuando vale exactamente 0, es decir en x = 1.
17. ¿Para qué valores de k la ecuación x2 + kx + 9 = 0 NO tiene raíces reales?
Ver resolución
Respuesta: a)
Sin raíces reales → Δ < 0: k2 − 4(1)(9) < 0 → k2 − 36 < 0 → (k − 6)(k + 6) < 0 → −6 < k < 6.
18. Resuelve: x2 + 4 > 0
Ver resolución
Respuesta: a)
x2 ≥ 0 siempre, así que x2 + 4 ≥ 4 > 0 para todo valor. Solución: todo R.
19. ¿Cuántos enteros satisfacen x2 − 5x + 4 < 0?
Ver resolución
Respuesta: a)
(x − 1)(x − 4) < 0 → 1 < x < 4. Los enteros en ese intervalo son 2 y 3 → 2 enteros.
20. Resuelve: x2 − 7x + 10 ≥ 0
Ver resolución
Respuesta: a)
(x − 2)(x − 5) ≥ 0 → por fuera de 2 y 5 (incluidos): ⟨−∞ ; 2] ∪ [5 ; +∞⟩.
