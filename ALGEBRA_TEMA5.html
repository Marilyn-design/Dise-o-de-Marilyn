ÁLGEBRA · TEMA 5

Expresiones Algebraicas III: Notación Funcional y Polinómica

Notación P(x) · Valor numérico · Sustitución y composición · Cambio de variable · Valores notables

Explicación completa

1. La notación P(x)

La notación P(x) ("P de x") es una forma de ponerle nombre a un polinomio e indicar cuál es su variable. La letra de afuera es el nombre del polinomio y la de adentro es la variable.
📐 Si escribimos P(x) = 2x + 1, estamos diciendo: "el polinomio P, cuya variable es x, vale 2x + 1".
⚠️ Lo que está entre paréntesis indica qué se reemplaza en el lugar de la variable. P(3) significa "reemplaza la x por 3 en todo el polinomio".

2. Valor numérico

Calcular un valor numérico es simplemente reemplazar la variable por un número y operar.
📐 Si P(x) = x2 − 2x, entonces P(3) = (3)2 − 2(3) = 9 − 6 = 3.

3. Sustitución: P(x + a) y expresiones

A veces no se reemplaza por un número, sino por otra expresión. La regla es la misma: poner esa expresión en cada lugar donde aparezca la variable.
📐 Si P(x) = 2x − 1, entonces P(x + 1) = 2(x + 1) − 1 = 2x + 2 − 1 = 2x + 1.

4. Composición: P(P(x))

La composición consiste en aplicar el polinomio dos veces: primero se calcula lo de adentro y ese resultado se vuelve a meter en P.
📐 Si P(x) = x + 3, para hallar P(P(1)): primero P(1) = 1 + 3 = 4; luego P(4) = 4 + 3 = 7.

5. Cambio de variable

Cuando te dan P de una expresión (por ejemplo P(2x + 1)) y te piden hallar P(x), se hace un cambio de variable: igualas la expresión de adentro a una nueva letra, despejas y reemplazas.
📐 Si P(2x + 1) = 4x + 5, hago u = 2x + 1 → x = (u − 1)/2. Reemplazo: P(u) = 4·(u−1)/2 + 5 = 2(u − 1) + 5 = 2u + 3. Por tanto P(x) = 2x + 3.

6. Valores notables (repaso útil)

Igual que en el tema anterior, estos dos atajos siguen valiendo:
Suma de coeficientes = P(1)     Término independiente = P(0)
📐 Si P(x) = x2 + x, la suma de sus coeficientes es P(1) = 1 + 1 = 2.

Formulario

EVALUACIÓN Y SUSTITUCIÓN

P(a): reemplazar la variable por "a"
P(x + a): reemplazar la variable por (x + a)
P(P(x)): primero lo de adentro, luego de nuevo en P

CAMBIO DE VARIABLE

Si dan P(expresión), iguala: u = expresión
Despeja x en función de u y reemplaza
Renombra u como x al final

VALORES NOTABLES

Suma de coeficientes = P(1)
Término independiente = P(0)

Problemas tipo admisión

NIVEL BÁSICO
1. Si P(x) = 2x + 1, halla P(3).
Ver resolución
Respuesta: b)
Reemplazo x = 3: P(3) = 2(3) + 1 = 6 + 1 = 7.
2. Si P(x) = x2 − 4, halla P(0).
Ver resolución
Respuesta: c)
P(0) = (0)2 − 4 = −4 (que es además el término independiente).
3. Si P(x) = x + 5, halla P(−2).
Ver resolución
Respuesta: a)
P(−2) = (−2) + 5 = 3.
4. Si f(x) = 3x, halla f(4).
Ver resolución
Respuesta: b)
f(4) = 3(4) = 12.
5. Si P(x) = x2 + 1, halla P(1).
Ver resolución
Respuesta: b)
P(1) = (1)2 + 1 = 2 (que es también la suma de sus coeficientes).
NIVEL INTERMEDIO
6. Si P(x) = x2 − 2x, halla P(3).
Ver resolución
Respuesta: a)
P(3) = (3)2 − 2(3) = 9 − 6 = 3.
7. Si P(x) = 2x − 1, halla P(x + 1).
Ver resolución
Respuesta: a)
Reemplazo x por (x + 1): P(x + 1) = 2(x + 1) − 1 = 2x + 2 − 1 = 2x + 1.
8. Si P(x) = x2, halla P(a + 1).
Ver resolución
Respuesta: b)
P(a + 1) = (a + 1)2 = a2 + 2a + 1 (binomio al cuadrado).
9. Si P(x) = 3x + 2 y P(a) = 11, halla "a".
Ver resolución
Respuesta: a)
P(a) = 3a + 2 = 11 → 3a = 9 → a = 3.
10. Si P(x) = x + 3, halla P(P(1)).
Ver resolución
Respuesta: b)
Primero P(1) = 1 + 3 = 4. Luego P(4) = 4 + 3 = 7.
NIVEL AVANZADO
11. Si P(x) = x2 + 1, halla P(x − 1).
Ver resolución
Respuesta: a)
P(x − 1) = (x − 1)2 + 1 = (x2 − 2x + 1) + 1 = x2 − 2x + 2.
12. Si P(2x + 1) = 4x + 5, halla P(x).
Ver resolución
Respuesta: a)
Hago u = 2x + 1 → x = (u − 1)/2. Reemplazo: P(u) = 4·(u − 1)/2 + 5 = 2(u − 1) + 5 = 2u + 3. Por tanto P(x) = 2x + 3.
13. Si P(x) = 2x − 3, halla P(P(x)).
Ver resolución
Respuesta: a)
P(P(x)) = 2·P(x) − 3 = 2(2x − 3) − 3 = 4x − 6 − 3 = 4x − 9.
14. Si P(x − 1) = x2 + 1, halla P(2).
Ver resolución
Respuesta: a)
Quiero que x − 1 = 2, entonces x = 3. Reemplazo: P(2) = (3)2 + 1 = 9 + 1 = 10.
15. Si P(x) = x2 + x, halla la suma de sus coeficientes.
Ver resolución
Respuesta: a)
Suma de coeficientes = P(1) = (1)2 + 1 = 2.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Si P(x + 2) = 3x + 1, halla P(5).
Ver resolución
Respuesta: a)
Quiero x + 2 = 5 → x = 3. Reemplazo: P(5) = 3(3) + 1 = 9 + 1 = 10.
17. Si P(x) = ax + b, con P(1) = 5 y P(2) = 8, halla P(0).
Ver resolución
Respuesta: a)
P(1) = a + b = 5 ; P(2) = 2a + b = 8. Resto: a = 3, luego b = 2. P(0) = b = 2.
18. Si P(2x) = 8x3, halla P(x).
Ver resolución
Respuesta: a)
Hago u = 2x → x = u/2. Reemplazo: P(u) = 8(u/2)3 = 8 · u3/8 = u3. Por tanto P(x) = x3.
