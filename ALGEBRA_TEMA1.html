ÁLGEBRA · TEMA 1

Leyes de Exponentes I: Potenciación

Producto y cociente de bases iguales · Potencia de potencia · Producto y cociente · Exponente cero y negativo

Explicación completa

1. ¿Qué es la potenciación?

La potenciación es una multiplicación abreviada de factores iguales. En la expresión an, la base (a) es el factor que se repite y el exponente (n) indica cuántas veces se multiplica:
an = a · a · a · ... · a   (n veces)
📐 Ejemplo: 24 = 2 · 2 · 2 · 2 = 16. La base es 2 y el exponente es 4.

2. Producto de potencias de igual base

Cuando multiplicas potencias de la misma base, se conserva la base y se suman los exponentes.
am · an = am+n
📐 x3 · x2 = x3+2 = x5.

3. Cociente de potencias de igual base

Al dividir potencias de igual base, se conserva la base y se restan los exponentes.
am ÷ an = am−n   (a ≠ 0)
📐 x5 ÷ x2 = x5−2 = x3.

4. Potencia de una potencia

Cuando una potencia está elevada a otro exponente, los exponentes se multiplican.
(am)n = am·n
⚠️ No confundir: (am)n = am·n (se multiplican), pero am · an = am+n (se suman). Es el error más común del tema.

5. Potencia de un producto y de un cociente

El exponente se reparte a cada factor:
(a · b)n = an · bn    (a / b)n = an / bn
📐 (2x)3 = 23 · x3 = 8x3.

6. Exponente cero y exponente negativo

Dos reglas clave que siempre caen en el examen:
a0 = 1   (a ≠ 0)     a−n = 1 / an
📐 50 = 1 ;   3−2 = 1/32 = 1/9.
⚠️ El exponente negativo no hace negativa la potencia: 2−3 no es −8, sino 1/8. El signo negativo "invierte" (manda al denominador), no cambia el signo del resultado.

Problemas tipo admisión

NIVEL BÁSICO
1. Efectúa: 23 · 22
Ver resolución
Respuesta: b)
Misma base, los exponentes se suman: 23+2 = 25 = 32.
2. Simplifica: x5 ÷ x2
Ver resolución
Respuesta: a)
Cociente de igual base: se restan los exponentes: x5−2 = x3.
3. Simplifica: (x2)3
Ver resolución
Respuesta: b)
Potencia de potencia: se multiplican: x2·3 = x6.
4. ¿Cuánto vale 50?
Ver resolución
Respuesta: c)
Todo número distinto de cero elevado a la 0 vale 1.
5. Desarrolla: (2x)3
Ver resolución
Respuesta: c)
El exponente afecta a cada factor: (2x)3 = 23x3 = 8x3.
NIVEL INTERMEDIO
6. Simplifica: (x3 · x4) ÷ x2
Ver resolución
Respuesta: a)
Arriba: x3+4 = x7. Luego x7 ÷ x2 = x7−2 = x5.
7. Desarrolla: (2x2)3
Ver resolución
Respuesta: c)
(2x2)3 = 23(x2)3 = 8 · x6 = 8x6.
8. ¿Cuánto vale 3−2?
Ver resolución
Respuesta: c)
El exponente negativo invierte: 3−2 = 1/32 = 1/9.
9. Desarrolla: (x2y3)2
Ver resolución
Respuesta: a)
(x2y3)2 = (x2)2(y3)2 = x4y6.
10. Simplifica: (a5 ÷ a2) · a
Ver resolución
Respuesta: a)
a5−2 = a3; luego a3 · a = a3+1 = a4.
NIVEL AVANZADO
11. Simplifica: (x4 · x6) ÷ (x2)3
Ver resolución
Respuesta: a)
Arriba: x4+6 = x10. Abajo: (x2)3 = x6. Cociente: x10−6 = x4.
12. Calcula: (23 · 24) ÷ 25
Ver resolución
Respuesta: a)
Arriba: 23+4 = 27. Cociente: 27−5 = 22 = 4.
13. ¿Cuánto vale (1/2)−3?
Ver resolución
Respuesta: b)
Un exponente negativo invierte la fracción: (1/2)−3 = 23 = 8.
14. Simplifica: (a−2 · b3) ÷ (a−4 · b)
Ver resolución
Respuesta: a)
Resto exponentes de igual base: a−2−(−4) = a2 ; b3−1 = b2. Resultado: a2b2.
15. Simplifica: [(x2)3 · x] ÷ x5
Ver resolución
Respuesta: a)
(x2)3 = x6; x6 · x = x7; x7 ÷ x5 = x7−5 = x2.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Simplifica: (xn+2 · xn−1) ÷ x2n
Ver resolución
Respuesta: a)
Arriba: x(n+2)+(n−1) = x2n+1. Cociente: x(2n+1)−2n = x.
17. Calcula: (4 · 106) ÷ (2 · 102)
Ver resolución
Respuesta: a)
Separo: (4/2) · (106/102) = 2 · 106−2 = 2 · 104 (es decir 20 000).
18. Si ax = 5, halla el valor de a3x.
Ver resolución
Respuesta: b)
a3x = (ax)3 = 53 = 125.
