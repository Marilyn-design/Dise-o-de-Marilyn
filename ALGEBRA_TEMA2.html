ÁLGEBRA · TEMA 2

Leyes de Exponentes II: Radicación

Exponente fraccionario · Raíz de productos y de raíces · Ecuaciones exponenciales · Exponentes trascendentes

Explicación completa

1. La radicación como exponente fraccionario

La radicación es la operación inversa de la potenciación. Lo más importante de este tema es entender que toda raíz se puede escribir como una potencia de exponente fraccionario: el índice de la raíz va al denominador.
n√(am) = am/n
📐 Ejemplos: √a = a1/2 ;   3√a = a1/3 ;   3√(a2) = a2/3.
⚠️ Truco de oro del tema: casi todos los ejercicios de radicación se resuelven mejor si primero conviertes las raíces en exponentes fraccionarios. Así puedes usar todas las leyes del Tema 1.

2. Propiedades de la radicación

n√(ab) = n√a · n√b
m√(n√a) = m·n√a   (raíz de raíz: se multiplican los índices)
📐 2√(3√64) = 6√64 = 2 (porque 26 = 64).

3. Combinando con exponentes fraccionarios

Una vez en forma de potencia, aplicas las mismas leyes de antes:
📐 82/3 = (3√8)2 = 22 = 4. Conviene sacar primero la raíz y luego elevar.

4. Ecuaciones exponenciales

Son ecuaciones donde la incógnita está en el exponente. La estrategia principal: igualar las bases. Si las bases son iguales, los exponentes también lo son.
Si ax = ay   entonces   x = y
📐 Ejemplo: 2x = 32. Como 32 = 25, escribo 2x = 25 → x = 5.

5. Ecuaciones con cambio de base

Cuando las bases no son iguales pero una es potencia de la otra, las igualamos.
📐 Ejemplo: 9x = 27. Llevo todo a base 3: (32)x = 33 → 32x = 33 → 2x = 3 → x = 3/2.

6. Exponentes trascendentes

Son ecuaciones donde la incógnita aparece en la base y en el exponente a la vez, como xx. Se resuelven dándole a ambos lados la misma "forma".
📐 Ejemplo: xx = 4. Busco un número que, elevado a sí mismo, dé 4: como 22 = 4, entonces x = 2.

Formulario

RADICACIÓN

n√(am) = am/n
n√(ab) = n√a · n√b
m√(n√a) = m·n√a

ECUACIONES EXPONENCIALES

Si ax = ay → x = y
Estrategia: igualar las bases
Trascendente: xx = k → buscar la base que cumpla

Problemas tipo admisión

NIVEL BÁSICO
1. Calcula: √16
Ver resolución
Respuesta: b)
√16 es el número que al cuadrado da 16: como 42 = 16, √16 = 4.
2. Calcula: 3√27
Ver resolución
Respuesta: a)
3√27 es el número que al cubo da 27: 33 = 27, así que es 3.
3. Escribe como potencia: 3√(a2)
Ver resolución
Respuesta: b)
El exponente del radicando va arriba y el índice abajo: 3√(a2) = a2/3.
4. Simplifica: √(x4)
Ver resolución
Respuesta: a)
√(x4) = x4/2 = x2.
5. Calcula: 41/2
Ver resolución
Respuesta: a)
41/2 = √4 = 2.
NIVEL INTERMEDIO
6. Simplifica: √(x2 · x4)
Ver resolución
Respuesta: a)
Dentro: x2+4 = x6. Luego √(x6) = x6/2 = x3.
7. Calcula: 82/3
Ver resolución
Respuesta: a)
82/3 = (3√8)2 = 22 = 4.
8. Calcula: 4√16
Ver resolución
Respuesta: b)
Busco el número que elevado a la 4 da 16: 24 = 16, así que 4√16 = 2.
9. Resuelve: 2x = 32
Ver resolución
Respuesta: b)
32 = 25, así que 2x = 25 → x = 5.
10. Calcula: (√2)4
Ver resolución
Respuesta: b)
(√2)4 = (21/2)4 = 24/2 = 22 = 4.
NIVEL AVANZADO
11. Resuelve: 3x+1 = 81
Ver resolución
Respuesta: a)
81 = 34, así que 3x+1 = 34 → x + 1 = 4 → x = 3.
12. Resuelve: 9x = 27
Ver resolución
Respuesta: a)
Llevo a base 3: (32)x = 33 → 32x = 33 → 2x = 3 → x = 3/2.
13. Simplifica: √(x3) · √x
Ver resolución
Respuesta: a)
x3/2 · x1/2 = x(3/2 + 1/2) = x2. (O bien: √(x3·x) = √(x4) = x2.) Resultado: x2.
14. Calcula: √(3√64)
Ver resolución
Respuesta: a)
Raíz de raíz: los índices se multiplican: √(3√64) = 6√64. Como 26 = 64, el resultado es 2.
15. Resuelve: 4x = 8
Ver resolución
Respuesta: a)
Base 2: (22)x = 23 → 22x = 23 → 2x = 3 → x = 3/2.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Resuelve: 2x+2 = 8x
Ver resolución
Respuesta: a)
Llevo todo a base 2: 8x = (23)x = 23x. Entonces 2x+2 = 23x → x + 2 = 3x → 2 = 2x → x = 1.
17. Resuelve (trascendente): xx = 4   (x > 0)
Ver resolución
Respuesta: b)
Busco la base que elevada a sí misma da 4. Como 22 = 4, se cumple con x = 2.
18. Resuelve: 25x = 125
Ver resolución
Respuesta: a)
Base 5: (52)x = 53 → 52x = 53 → 2x = 3 → x = 3/2.
