ÁLGEBRA · TEMA 18

Desigualdades e Inecuaciones de Primer Grado

Propiedades de las desigualdades · Intervalos · Resolución de inecuaciones · Inecuaciones dobles

Explicación completa

1. ¿Qué es una desigualdad?

Una desigualdad compara dos expresiones que no son iguales, usando los símbolos:
< (menor que)   > (mayor que)   ≤ (menor o igual)   ≥ (mayor o igual)
Una inecuación es una desigualdad con una incógnita; su solución no es un solo número, sino un conjunto de valores (normalmente un intervalo).

2. Propiedades (¡la regla más importante!)

  • Si sumas o restas lo mismo a ambos lados, la desigualdad no cambia.
  • Si multiplicas o divides por un número positivo, no cambia.
  • Si multiplicas o divides por un número negativo, ¡el sentido se invierte!
⚠️ El error N.°1 del tema: al dividir entre un número negativo hay que voltear el signo. Por ejemplo: −2x > 6 → x < −3 (el > se vuelve <).

3. Intervalos

La solución se expresa como intervalo. El corchete [ ] incluye el extremo; el paréntesis ( ) lo excluye. El infinito siempre va con paréntesis.
InecuaciónIntervaloSignificado
x > 2⟨2 ; +∞⟩mayores que 2, sin incluir el 2
x ≥ 2[2 ; +∞⟩desde 2 (incluido) en adelante
x < 5⟨−∞ ; 5⟩menores que 5
2 < x ≤ 5⟨2 ; 5]entre 2 y 5, incluyendo el 5

4. Resolver una inecuación de primer grado

Se despeja la incógnita igual que en una ecuación, pero cuidando el signo al multiplicar o dividir por negativos.
📐 3x − 6 > 0 → 3x > 6 → x > 2. Solución: ⟨2 ; +∞⟩.
📐 5 − x < 2 → −x < −3 → (divido por −1 y volteo) x > 3.

5. Inecuaciones dobles

Tienen la forma a < expresión < b. Se opera en las tres partes a la vez para dejar sola a la x.
📐 −3 < 2x − 1 < 5. Sumo 1 a todo: −2 < 2x < 6. Divido entre 2: −1 < x < 3. Solución: ⟨−1 ; 3⟩.

Formulario

PROPIEDADES

Sumar/restar → el signo no cambia
× o ÷ por (+) → no cambia
× o ÷ por (−) → se invierte el sentido

INTERVALOS

[ ] incluye el extremo
⟨ ⟩ lo excluye
∞ siempre con paréntesis

RESOLUCIÓN

Despejar x como en una ecuación
Ojo al dividir por negativos
Doble: operar las tres partes a la vez

Problemas tipo admisión

NIVEL BÁSICO
1. Resuelve: x + 3 > 5
Ver resolución
Respuesta: a)
El 3 pasa restando: x > 5 − 3 → x > 2.
2. Resuelve: 2x < 8
Ver resolución
Respuesta: a)
Divido entre 2 (positivo, no cambia): x < 8/2 → x < 4.
3. Resuelve: x − 1 ≥ 0
Ver resolución
Respuesta: a)
El 1 pasa sumando: x ≥ 1 → x ≥ 1.
4. Al multiplicar una desigualdad por (−1), el sentido del signo...
Ver resolución
Respuesta: b)
Multiplicar o dividir por un número negativo invierte el sentido de la desigualdad.
5. ¿El intervalo ⟨2 ; 5⟩ incluye a los números 2 y 5?
Ver resolución
Respuesta: b)
Los paréntesis ⟨ ⟩ indican intervalo abierto: no incluye los extremos 2 ni 5.
NIVEL INTERMEDIO
6. Resuelve: 3x − 6 > 0
Ver resolución
Respuesta: a)
3x > 6 → x > 2 → x > 2.
7. Resuelve: −2x > 4
Ver resolución
Respuesta: a)
Divido entre −2 (negativo) y volteo el signo: x < 4/(−2) → x < −2.
8. Resuelve: 2x + 1 ≤ 7
Ver resolución
Respuesta: a)
2x ≤ 6 → x ≤ 3 → x ≤ 3. (Intervalo ⟨−∞ ; 3].)
9. Resuelve: x/2 > 3
Ver resolución
Respuesta: a)
Multiplico por 2 (positivo): x > 6 → x > 6.
10. Resuelve: 5 − x < 2
Ver resolución
Respuesta: a)
−x < 2 − 5 → −x < −3 → (volteo) x > 3 → x > 3.
NIVEL AVANZADO
11. Resuelve: 3(x − 1) < x + 5
Ver resolución
Respuesta: a)
3x − 3 < x + 5 → 2x < 8 → x < 4 → x < 4.
12. Resuelve: (x + 1)/2 ≥ (x − 1)/3
Ver resolución
Respuesta: a)
Multiplico por 6: 3(x + 1) ≥ 2(x − 1) → 3x + 3 ≥ 2x − 2 → x ≥ −5 → x ≥ −5.
13. Resuelve la inecuación doble: −3 < 2x − 1 < 5
Ver resolución
Respuesta: a)
Sumo 1 a las tres partes: −2 < 2x < 6. Divido entre 2: −1 < x < 3.
14. Resuelve: 4 − 2x ≥ 0
Ver resolución
Respuesta: a)
−2x ≥ −4 → (divido entre −2 y volteo) x ≤ 2 → x ≤ 2.
15. Halla el menor entero que cumple 2x + 3 > 7.
Ver resolución
Respuesta: a)
2x > 4 → x > 2. El menor entero mayor que 2 es 3.
NIVEL EXPERTO · INGRESO UNIVERSITARIO
16. Resuelve: −2 ≤ (3x − 1)/2 < 4
Ver resolución
Respuesta: a)
Multiplico por 2: −4 ≤ 3x − 1 < 8. Sumo 1: −3 ≤ 3x < 9. Divido entre 3: −1 ≤ x < 3.
17. Halla el conjunto solución de 5x − 2 ≥ 3x + 6.
Ver resolución
Respuesta: a)
5x − 3x ≥ 6 + 2 → 2x ≥ 8 → x ≥ 4. Como es "≥", el 4 se incluye: [4 ; +∞⟩.
18. "El doble de un número aumentado en 5 es menor que 17." ¿Qué valores cumple el número?
Ver resolución
Respuesta: a)
Traduzco: 2x + 5 < 17 → 2x < 12 → x < 6.
19. Resuelve el sistema de inecuaciones: x > 1 y x ≤ 4.
Ver resolución
Respuesta: a)
La solución es la intersección: mayores que 1 (1 excluido) y hasta 4 (incluido): ⟨1 ; 4].
20. ¿Cuántos números enteros satisfacen −2 < x ≤ 3?
Ver resolución
Respuesta: a)
Los enteros mayores que −2 (sin incluirlo) y hasta 3 (incluido) son: −1, 0, 1, 2, 3 → 5 enteros.
